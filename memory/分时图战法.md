## 🎯 形态定义

### 核心特征

| 特征 | 说明 | 重要性 |
|------|------|--------|
| **成交价在分时均线上方** | 股价始终不跌破分时均线 | ⭐⭐⭐⭐⭐ |
| **分时均线稳步向上** | 均线斜率>0，持续上行 | ⭐⭐⭐⭐⭐ |
| **回调不破均线** | 每次回踩均线都反弹 | ⭐⭐⭐⭐ |
| **成交量配合** | 上涨放量，回调缩量 | ⭐⭐⭐⭐ |

### 形态含义

```
成交价在分时均线上方 = 多头强势
分时均线向上 = 趋势向上
回调不破 = 主力护盘
成交量配合 = 资金认可

综合含义：主力控盘良好，当天大概率涨停或大涨
```

## 模块 1: 分时图基础

### 分时图组成

| 组成 | 含义 | 颜色 |
|------|------|------|
| **白色线** | 实时成交价 | 白/黄 |
| **黄色线** | 分时均线 (均价线) | 黄/蓝 |
| **成交量柱** | 每分钟成交量 | 红绿 |
| **昨收线** | 昨日收盘价 | 虚线 |

### 分时均线计算

```
分时均线 = 从开盘到当前的成交总金额 / 成交总股数

特点:
  - 每分钟更新一次
  - 反映当日平均持仓成本
  - 是最强的日内支撑/压力位
```

## 模块 2: 核心形态识别

### 形态 1: 强势控盘型 🟢🟢🟢

**特征**:
```
✅ 开盘后股价一直在分时均线上方
✅ 分时均线持续向上 (斜率>30 度)
✅ 每次回调到均线都反弹
✅ 上涨放量，回调缩量
✅ 股价与均线距离适中 (3-5%)
```

**含义**: 主力控盘良好，资金持续流入

**操作**:
```
买入点:
  1. 开盘后站稳均线 (9:35-9:40)
  2. 第一次回调到均线 (10:00 左右)
  3. 突破早盘高点 (10:30 左右)

成功率：75%+
目标：涨停或 +7% 以上
```

### 形态 2: 震荡整理型 🟡

**特征**:
```
🟡 股价围绕分时均线震荡
🟡 分时均线走平或微向上
🟡 多次穿越均线
🟡 成交量不规则
```

**含义**: 多空分歧，方向不明

**操作**:
```
观望为主
等待方向选择
突破均线 + 放量 = 跟进
跌破均线 + 放量 = 卖出
```

### 形态 3: 弱势下跌型 🔴🔴🔴

**特征**:
```
🔴 股价一直在分时均线下方
🔴 分时均线持续向下
🔴 每次反弹到均线都受阻
🔴 下跌放量，反弹缩量
```

**含义**: 主力出货，资金流出

**操作**:
```
坚决不买入
持仓者反弹到均线卖出
跌破重要支撑止损
```

## 模块 3: 量化判断标准

### 强势控盘评分 (100 分)

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **股价位置** | 30 分 | 在均线上方时间占比 |
| **均线斜率** | 25 分 | 向上角度 |
| **回调支撑** | 20 分 | 回踩不破次数 |
| **量价配合** | 15 分 | 上涨放量/回调缩量 |
| **涨幅** | 10 分 | 当前涨跌幅 |

### 评分细则

#### 1. 股价位置 (30 分)
```
股价在均线上方时间占比:
  >90% = 30 分 (极强)
  70-90% = 25 分 (强势)
  50-70% = 15 分 (震荡)
  <50% = 0 分 (弱势)
```

#### 2. 均线斜率 (25 分)
```
均线角度:
  >45 度 = 25 分 (强势上涨)
  30-45 度 = 20 分 (稳健上涨)
  15-30 度 = 15 分 (缓慢上涨)
  0-15 度 = 5 分 (走平)
  <0 度 = 0 分 (向下)
```

#### 3. 回调支撑 (20 分)
```
回踩均线不破次数:
  3 次以上都反弹 = 20 分 (强支撑)
  2 次反弹 = 15 分
  1 次反弹 = 10 分
  0 次或跌破 = 0 分
```

#### 4. 量价配合 (15 分)
```
上涨放量 + 回调缩量 = 15 分 (完美)
上涨放量 + 回调持平 = 10 分
上涨缩量 = 5 分 (危险)
下跌放量 = 0 分 (出货)
```

#### 5. 涨幅 (10 分)
```
当前涨幅:
  >7% = 10 分 (强势)
  5-7% = 8 分
  3-5% = 6 分
  0-3% = 4 分
  <0% = 0 分
```

### 综合评级

| 总分 | 评级 | 含义 | 操作 |
|------|------|------|------|
| ≥85 | 🟢🟢🟢 极强 | 主力控盘，大概率涨停 | 积极买入 |
| 70-84 | 🟢🟢 强势 | 趋势良好，可能大涨 | 持有/加仓 |
| 55-69 | 🟡 中性 | 震荡整理 | 观望 |
| <55 | 🔴 弱势 | 主力出货 | 卖出/空仓 |

## 模块 4: 盘中监控要点

### 9:30-9:35 (开盘 5 分钟)

**关键观察**:
1. 开盘位置 (高开/低开/平开)
2. 是否站稳分时均线
3. 第一根成交量柱

**强势信号**:
```
✅ 高开 2-5%
✅ 开盘后快速拉升
✅ 站稳分时均线
✅ 成交量放大
```

**警惕信号**:
```
🔴 低开 -3% 以下
🔴 开盘后快速跌破均线
🔴 反弹无法站上均线
```

### 9:35-10:00 (确认期)

**关键观察**:
1. 分时均线方向
2. 第一次回调是否破均线
3. 量价配合

**买入信号**:
```
✅ 分时均线向上 (>30 度)
✅ 股价在均线上方
✅ 第一次回调到均线不破
✅ 回调缩量，反弹放量
```

### 10:00-11:30 (持仓期)

**关键观察**:
1. 股价是否持续在均线上方
2. 均线是否持续向上
3. 是否有大单护盘

**持有信号**:
```
✅ 股价一直在均线上方
✅ 均线持续向上
✅ 多次回踩都不破
✅ 成交量温和
```

**卖出信号**:
```
🔴 跌破分时均线
🔴 均线开始走平或向下
🔴 反弹无法站上均线
🔴 下跌放量
```

### 13:00-14:00 (午后确认)

**关键观察**:
1. 午后是否延续上午趋势
2. 是否有资金继续流入
3. 是否准备冲击涨停

**强势延续**:
```
✅ 午后继续上涨
✅ 均线斜率不变
✅ 成交量配合
```

**警惕信号**:
```
🔴 午后跳水
🔴 跌破均线
🔴 成交量异常
```

### 14:00-15:00 (尾盘决策)

**关键观察**:
1. 是否封涨停
2. 封单金额
3. 是否持股过夜

**涨停确认**:
```
✅ 14:30 前封涨停
✅ 封单>2 亿
✅ 不炸板
```

## 模块 6: 自动化监控

### 实时监控指标

```python
def check_intraday_pattern(stock_data):
    """检查分时图形态"""
    
    # 数据准备
    prices = stock_data['price']  # 每分钟价格
    vwap = stock_data['vwap']     # 分时均线 (成交均价)
    volumes = stock_data['volume'] # 每分钟成交量
    
    score = 0
    
    # 1. 股价位置 (30 分)
    above_ratio = sum(p > v for p, v in zip(prices, vwap)) / len(prices)
    if above_ratio > 0.9:
        score += 30
    elif above_ratio > 0.7:
        score += 25
    elif above_ratio > 0.5:
        score += 15
    
    # 2. 均线斜率 (25 分)
    slope = (vwap[-1] - vwap[0]) / vwap[0] * 100
    if slope > 3:  # >3% 涨幅，约 45 度
        score += 25
    elif slope > 2:
        score += 20
    elif slope > 1:
        score += 15
    elif slope > 0:
        score += 5
    
    # 3. 回调支撑 (20 分)
    support_count = 0
    for i in range(5, len(prices)):
        if prices[i] < vwap[i] * 1.01 and prices[i] > vwap[i]:
            # 回踩均线
            if prices[i+1] > prices[i]:
                # 反弹
                support_count += 1
    
    if support_count >= 3:
        score += 20
    elif support_count >= 2:
        score += 15
    elif support_count >= 1:
        score += 10
    
    # 4. 量价配合 (15 分)
    # 简化：上涨分钟成交量 vs 下跌分钟成交量
    up_vol = sum(v for p, v in zip(prices[1:], volumes[1:]) if p > prices[0])
    down_vol = sum(v for p, v in zip(prices[1:], volumes[1:]) if p < prices[0])
    
    if up_vol > down_vol * 1.5:
        score += 15
    elif up_vol > down_vol:
        score += 10
    
    # 5. 涨幅 (10 分)
    change = (prices[-1] - prices[0]) / prices[0] * 100
    if change > 7:
        score += 10
    elif change > 5:
        score += 8
    elif change > 3:
        score += 6
    elif change > 0:
        score += 4
    
    return {
        "score": score,
        "above_ratio": round(above_ratio * 100, 1),
        "slope": round(slope, 2),
        "support_count": support_count,
        "change": round(change, 2),
    }
```

## 核心心法

1. **分时均线是日内生命线** — 破了就跑
2. **均线向上才持有** — 走平或向下卖出
3. **回调不破是买点** — 第一次最可靠
4. **量价配合是关键** — 上涨必须放量
5. **不抱幻想** — 跌破均线立即止损
6. **早盘定强弱** — 9:30-10:00 最关键
7. **午后防跳水** — 13:30-14:00 警惕
