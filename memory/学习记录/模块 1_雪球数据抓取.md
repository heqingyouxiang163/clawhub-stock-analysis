## ⚠️ **遇到的问题**

### 问题 1: 雪球 API 需要登录

**尝试方法**:
```python
# 方法 1: 直接 API 调用
url = "https://stock.xueqiu.com/v5/stock/quote.json"
params = {'symbol': 'SZ002828', 'extend': 'detail'}

结果：400 Error - 需要登录认证
```

**原因分析**:
- 雪球 API 需要 cookie 认证
- cookie 有有效期，需要定期刷新
- 浏览器自动化需要登录状态

### 问题 2: 浏览器服务不可用

**错误信息**:
```
Can't reach the OpenClaw browser control service
(timed out after 20000ms)
```

**原因**: OpenClaw 浏览器控制服务未运行

## ✅ **替代方案**

### 方案 1: MCP API + 严格校验 (当前使用)

**优势**:
- ✅ 无需登录
- ✅ 快速稳定 (1-2 秒)
- ✅ 数据准确

**劣势**:
- ⚠️ 数据延迟 15 分钟
- ⚠️ 不是真正的实时数据

**校验规则**:
```python
# A 股硬约束
- 涨跌幅：-10% to +10%
- 换手率：0% to 50%
- 最高价 >= 现价
- 最低价 <= 现价
- 成交量 > 0
```

### 方案 2: 浏览器自动化 (需要修复)

**前提条件**:
1. 启动 OpenClaw 浏览器服务
2. 登录雪球账号
3. 保持登录状态

**实现方式**:
```python
# 使用 browser 工具
browser(action=navigate, targetUrl="https://xueqiu.com/SZ002828")
browser(action=snapshot, targetId="<会话 ID>")
```
