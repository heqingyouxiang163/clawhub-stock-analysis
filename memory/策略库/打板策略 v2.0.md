# 打板策略 v2.0

**版本**: v2.0 (2026-03-17 升级)
**类型**: 超短线 (1-3 日)
**风险等级**: 高
**适用市场**: A 股 (沪深主板、创业板、科创板)

---

## 策略升级说明

### v1.0 → v2.0 核心改进

| 改进点 | v1.0 | v2.0 | 优化理由 |
|--------|------|------|----------|
| 封板时间窗口 | 全天 | 分时段差异化 | 早盘板质量更高 |
| 换手率要求 | 固定 10%+ | 分板块/价位差异化 | 更精准 |
| 炸板次数限制 | ≤3 次 | ≤2 次 (早盘)/≤4 次 (午后) | 更严格 |
| 板块效应 | 可选 | 必选 | 提高胜率 |
| 仓位管理 | 固定 20% | 动态分级 (10%-30%) | 风险匹配 |
| 止损策略 | 固定 -5% | 分场景 (-3%/-5%/-8%) | 更灵活 |

---

## 核心逻辑

### 策略理念
**打板 = 打确定性**

- 涨停是资金一致性最强的表现
- 打板追求的是次日溢价和连板概率
- 宁可错过，不可做错 (质量>数量)

### 盈利来源
1. **次日溢价**: 涨停股次日高开概率 ~70%
2. **连板溢价**: 连板股超额收益
3. **板块轮动**: 热点板块持续性强

### 板块效应要求

**必须满足以下任一条件**:

1. **板块涨停≥3 只**: 强板块效应
2. **板块涨幅≥3%**: 板块整体强势
3. **板块内已有连板股**: 有龙头带队

**板块优先级**:
| 优先级 | 板块特征 | 说明 |
|--------|----------|------|
| ⭐⭐⭐ | 主线热点 (连续 3 日强势) | 如 AI、新能源 |
| ⭐⭐ | 轮动热点 (当日爆发) | 如突发利好 |
| ⭐ | 冷门板块 (首只涨停) | 谨慎 |

## 排除条件 (一票否决)

### 硬性排除
- ❌ 股价 < 3 元
- ❌ 流通市值 < 20 亿 或 > 300 亿
- ❌ ST 股
- ❌ 尾盘板 (14:30 后)
- ❌ 炸板次数超标
- ❌ 无板块效应

### 风险排除
- ❌ 昨日涨停今日跌停 (核按钮)
- ❌ 连续 3 日大涨 (>30%)
- ❌ 监管函/立案调查
- ❌ 大股东减持公告
- ❌ 业绩大幅下滑

## 打板类型分级

### S 级：一字板/秒板

**特征**:
- 9:25-9:30 封板
- 换手率 < 10%
- 板块效应强
- 龙头股

**操作**:
- 排队买入 (需要快速通道)
- 仓位：20%-30%
- 止损：-3%

**风险**: 买不进或开板即坑

---

### A 级：早盘强势板

**特征**:
- 9:30-10:00 封板
- 换手率 5%-15%
- 炸板≤1 次
- 板块效应强

**操作**:
- 封板瞬间买入
- 仓位：20%
- 止损：-5%

**风险**: 中等，质量较高

---

### B 级：上午板/午后板

**特征**:
- 10:00-14:00 封板
- 换手率 8%-20%
- 炸板≤3 次
- 板块效应中等

**操作**:
- 封板瞬间买入
- 仓位：15%
- 止损：-5%

**风险**: 中高，需要精选

---

### C 级：尾盘板

**特征**:
- 14:00-14:30 封板
- 换手率 10%-25%
- 炸板≤4 次
- 板块效应弱

**操作**:
- 谨慎参与
- 仓位：10%
- 止损：-3%

**风险**: 高，次日溢价差

## 止损止盈

### 止损策略

| 场景 | 止损位 | 操作 |
|------|--------|------|
| S 级 (秒板) | -3% | 开盘不及预期即走 |
| A 级 (早盘板) | -5% | 标准止损 |
| B/C 级 | -5% | 标准止损 |
| 连板股断板 | 次日开盘 | 不及预期即走 |
| 核按钮 | 开盘 | 不抱幻想 |

### 止盈策略

| 涨幅 | 操作 |
|------|------|
| +5% | 减仓 30% |
| +10% | 减仓 50% |
| +15% | 减仓 70% |
| 涨停 | 持有 |
| 断板 | 清仓 |

## 操作流程

### 盘前 (9:00-9:25)
1. 查看隔夜消息
2. 观察集合竞价
3. 筛选目标股 (高开 3%-7%，量能配合)
4. 确认板块效应

### 盘中 (9:30-15:00)
1. 监控目标股
2. 封板瞬间执行买入
3. 记录交易数据
4. 监控持仓股

### 盘后 (15:00-22:00)
1. 复盘当日交易
2. 分析涨停股
3. 更新策略胜率
4. 制定明日计划

## 风险提示

⚠️ **打板是高风险策略**，不适合以下投资者：
- 风险承受能力低
- 无法及时盯盘
- 新手投资者
- 资金量过小 (<10 万)

⚠️ **严格执行止损**，不抱侥幸心理

⚠️ **仓位管理**是生存关键，不要满仓一只

⚠️ **情绪周期**决定仓位，退潮期空仓
