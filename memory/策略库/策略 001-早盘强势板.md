# 策略 001: 早盘强势板策略

**版本**: v2.1 (2026-03-17 优化)
**类型**: 超短线 (1-3 日)
**风险等级**: 中高

---

## 策略逻辑

### 核心思路
捕捉 9:30-10:00 之间快速封板的强势股，利用早盘资金攻击意愿强的特点，博取次日溢价。

### 入选条件
1. **时间窗口**: 9:30-10:00 封板
2. **封板质量**: 炸板次数 ≤ 2 次
3. **换手率**: 5%-20% (过低没换手，过高分歧大)
4. **成交量**: 较前 5 日平均放大 ≥ 50%
5. **股价位置**: 突破近期平台或前高
6. **板块效应**: 所属板块当日涨幅前 5

### 排除条件
- 一字板 (买不进)
- 尾盘板 (太弱)
- 炸板次数 > 3 次 (分歧太大)
- 换手率 > 30% (过度博弈)

## 操作规范

### 入场时机
- **最佳**: 封板瞬间排队 (需要快速通道)
- **次选**: 次日开盘观察，弱转强时介入

### 仓位管理
- 单只股票 ≤ 20% 仓位
- 同时持有 ≤ 3 只

### 止损止盈
| 条件 | 操作 |
|------|------|
| 次日低开 > -3% | 开盘即止损 |
| 次日高开 > 3% | 持有观察 |
| 盘中涨停 | 持有 |
| 盘中跌破分时均线 | 减仓 50% |
| 持有 3 日未涨 | 清仓 |

### 止盈位
- +5%: 减仓 30%
- +10%: 减仓 50%
- 涨停: 持有
