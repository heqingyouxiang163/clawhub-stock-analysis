## 模块 1: 均线系统 (MA) - 趋势之王

### 常用均线周期

| 均线 | 周期 | 含义 | 用途 |
|------|------|------|------|
| **MA5** | 5 日 | 周线 | 短线生命线 |
| **MA10** | 10 日 | 半月线 | 短线趋势 |
| **MA20** | 20 日 | 月线 | 中线生命线 |
| **MA60** | 60 日 | 季线 | 中线趋势 |
| **MA120** | 120 日 | 半年线 | 长线支撑 |
| **MA250** | 250 日 | 年线 | 牛熊分界 |

### 核心信号

#### 1. 多头排列 (最强买入信号) 🟢🟢🟢
```
条件: MA5 > MA10 > MA20 > MA60
      所有均线向上
含义: 趋势强势，各周期一致看涨
操作: 持有/加仓，止损设 MA20
成功率: 70%+
```

#### 2. 空头排列 (最强卖出信号) 🔴🔴🔴
```
条件: MA5 < MA10 < MA20 < MA60
      所有均线向下
含义: 趋势弱势，各周期一致看跌
操作: 空仓观望，不抄底
成功率: 70%+
```

#### 3. 金叉 (买入信号) 🟢
```
条件: 短周期均线上穿长周期均线
      MA5 上穿 MA10 = 短线金叉
      MA10 上穿 MA20 = 中线金叉
含义: 趋势转强
操作: 金叉 + 放量 = 买入
成功率: 50-60%
```

#### 4. 死叉 (卖出信号) 🔴
```
条件: 短周期均线下穿长周期均线
      MA5 下穿 MA10 = 短线死叉
      MA10 下穿 MA20 = 中线死叉
含义: 趋势转弱
操作: 死叉 + 放量 = 卖出
成功率: 50-60%
```

#### 5. 均线支撑/压力
```
支撑: 股价回调到均线附近企稳
      MA5/MA10/MA20 都可作为支撑
操作: 触及支撑 + 缩量 = 买入
      跌破支撑 + 放量 = 止损

压力: 股价反弹到均线附近受阻
操作: 触及压力 + 缩量 = 卖出
      突破压力 + 放量 = 跟进
```

### 实战口诀
```
均线多头排列，持股待涨
均线空头排列，空仓观望
金叉放量买入，死叉放量卖出
五日线是短线生命线
二十日线是中线生命线
```

## 模块 2: MACD - 指标之王

### MACD 组成

| 组成 | 含义 | 计算 |
|------|------|------|
| **DIF (快线)** | 12 日 EMA - 26 日 EMA | 短期趋势 |
| **DEA (慢线)** | DIF 的 9 日 EMA | 长期趋势 |
| **MACD 柱** | DIF - DEA | 动能强弱 |
| **0 轴** | 多空分界 | 上方多头，下方空头 |

### 核心信号

#### 1. 金叉 (买入信号) 🟢
```
条件: DIF 上穿 DEA
位置: 0 轴下方 = 低位金叉 (更强)
      0 轴上方 = 高位金叉 (一般)
配合: 金叉 + 放量 = 强烈买入
      金叉 + 缩量 = 谨慎买入
成功率: 55-65%
```

#### 2. 死叉 (卖出信号) 🔴
```
条件: DIF 下穿 DEA
位置: 0 轴上方 = 高位死叉 (更强)
      0 轴下方 = 低位死叉 (一般)
配合: 死叉 + 放量 = 强烈卖出
      死叉 + 缩量 = 谨慎卖出
成功率: 55-65%
```

#### 3. 0 轴突破
```
上穿 0 轴:
  含义：由空转多，趋势反转
  操作：上穿 + 放量 = 买入
  成功率：60%+

下穿 0 轴:
  含义：由多转空，趋势反转
  操作：下穿 = 卖出/空仓
  成功率：60%+
```

#### 4. 顶背离 (强烈卖出信号) 🔴🔴🔴
```
条件: 股价创新高
      MACD 未创新高 (反而下降)
含义: 上涨动能衰竭，即将回调
位置: 高位顶背离 = 极危险
操作: 顶背离 + 死叉 = 坚决卖出
成功率: 70%+
```

#### 5. 底背离 (强烈买入信号) 🟢🟢🟢
```
条件: 股价创新低
      MACD 未创新低 (反而上升)
含义: 下跌动能衰竭，即将反弹
位置: 低位底背离 = 极强势
操作: 底背离 + 金叉 = 坚决买入
成功率: 70%+
```

#### 6. 红绿柱变化
```
红柱 (DIF>DEA):
  红柱变长 = 多头动能增强 ✅
  红柱变短 = 多头动能减弱 ⚠️

绿柱 (DIF<DEA):
  绿柱变长 = 空头动能增强 🔴
  绿柱变短 = 空头动能减弱 🟢
```

### 实战口诀
```
MACD 金叉，低位更可靠
MACD 死叉，高位要逃跑
顶背离出现，坚决要止盈
底背离出现，大胆可抄底
红柱变长持有，红柱变短减仓
```

## 模块 3: RSI - 超买超卖指标

### RSI 计算
```
RSI = 100 - (100 / (1 + RS))
RS = N 日内涨幅平均值 / N 日内跌幅平均值

常用周期: RSI(6) 短线, RSI(14) 中线, RSI(24) 长线
```

### 核心信号

#### 1. 超买区 (卖出区域) 🔴
```
RSI > 70: 超买区，警惕回调
RSI > 80: 严重超买，坚决卖出
RSI > 90: 极度超买，立刻清仓

操作:
  RSI>70 + 股价滞涨 = 减仓
  RSI>80 + 顶背离 = 清仓
  RSI>90 = 无条件卖出
```

#### 2. 超卖区 (买入区域) 🟢
```
RSI < 30: 超卖区，可能反弹
RSI < 20: 严重超卖，坚决买入
RSI < 10: 极度超卖，大胆抄底

操作:
  RSI<30 + 股价企稳 = 试错
  RSI<20 + 底背离 = 重仓
  RSI<10 = 无条件买入
```

#### 3. 50 中轴线
```
RSI > 50: 多头市场，持股为主
RSI < 50: 空头市场，空仓为主

操作:
  RSI 上穿 50 = 买入信号
  RSI 下穿 50 = 卖出信号
```

#### 4. RSI 背离
```
顶背离:
  股价新高，RSI 未新高
  含义：上涨动能衰竭
  操作：坚决卖出

底背离:
  股价新低，RSI 未新低
  含义：下跌动能衰竭
  操作：坚决买入
```

#### 5. RSI 形态
```
W 底:
  RSI 形成 W 底形态
  含义：底部确认
  操作：突破颈线买入

M 顶:
  RSI 形成 M 顶形态
  含义：顶部确认
  操作：跌破颈线卖出
```

### 实战口诀
```
RSI 上 80，超买要警惕
RSI 下 20，超卖可抄底
50 是多空线，上穿持有下穿跑
顶背离坚决卖，底背离大胆买
```

## 模块 4: KDJ - 随机指标

### KDJ 组成

| 组成 | 含义 | 特点 |
|------|------|------|
| **K 线** | 快速线 | 敏感 |
| **D 线** | 慢速线 | 稳定 |
| **J 线** | 方向线 | 最敏感 |

### 核心信号

#### 1. 金叉 (买入信号) 🟢
```
条件: K 线上穿 D 线
位置: 20 以下金叉 = 低位金叉 (更强)
      80 以上金叉 = 高位金叉 (风险)
配合: 金叉 + J 线向上 = 强烈买入
成功率: 50-60%
```

#### 2. 死叉 (卖出信号) 🔴
```
条件: K 线下穿 D 线
位置: 80 以上死叉 = 高位死叉 (更强)
      20 以下死叉 = 低位死叉 (风险)
配合: 死叉 + J 线向下 = 强烈卖出
成功率: 50-60%
```

#### 3. 超买超卖
```
超买区:
  KDJ > 80: 超买，警惕回调
  KDJ > 90: 严重超买，卖出
  
超卖区:
  KDJ < 20: 超卖，可能反弹
  KDJ < 10: 严重超卖，买入
```

#### 4. J 线极值
```
J > 100: 极度超买，强烈卖出信号
J < 0: 极度超卖，强烈买入信号

注意: J 线极值通常只维持 1-2 天
```

#### 5. KDJ 背离
```
顶背离:
  股价新高，KDJ 未新高
  操作：坚决卖出

底背离:
  股价新低，KDJ 未新低
  操作：坚决买入
```

### KDJ vs RSI

| 特点 | KDJ | RSI |
|------|-----|-----|
| 敏感度 | 更高 | 较低 |
| 适用 | 震荡市 | 趋势市 |
| 假信号 | 较多 | 较少 |
| 配合使用 | KDJ+RSI 双确认 | 提高准确率 |

### 实战口诀
```
KDJ 上 80，超买要减仓
KDJ 下 20，超卖可建仓
金叉低位买，死叉高位卖
J 线破百卖，J 线破零买
```

## 模块 5: 布林带 (BOLL) - 波动率指标

### 布林带组成

| 组成 | 计算 | 含义 |
|------|------|------|
| **上轨** | 中轨 + 2 倍标准差 | 压力位 |
| **中轨** | 20 日均线 | 趋势线 |
| **下轨** | 中轨 - 2 倍标准差 | 支撑位 |

### 核心信号

#### 1. 股价位置
```
上轨上方:
  含义：严重超买
  操作：卖出/减仓
  成功率：70%+

上轨附近:
  含义：强势区域
  操作：持有，警惕回调

中轨附近:
  含义：震荡区域
  操作：观望，等方向

下轨附近:
  含义：弱势区域
  操作：观望，可能反弹

下轨下方:
  含义：严重超卖
  操作：买入/抄底
  成功率：70%+
```

#### 2. 收口/开口
```
收口 (带宽变窄):
  含义：波动率降低，即将变盘
  操作：观望，等方向选择
  注意：收口后通常有大行情

开口 (带宽变宽):
  含义：波动率增加，趋势形成
  操作：顺势操作
  向上开口 = 买入
  向下开口 = 卖出
```

#### 3. 股价突破
```
突破上轨:
  含义：强势突破
  配合：突破 + 放量 = 跟进
        突破 + 缩量 = 警惕假突破

跌破下轨:
  含义：弱势破位
  配合：跌破 + 放量 = 止损
        跌破 + 缩量 = 可能反弹
```

#### 4. 中轨支撑/压力
```
中轨支撑:
  条件：股价回调到中轨企稳
  操作：中轨 + 缩量 = 买入
  止损：跌破中轨

中轨压力:
  条件：股价反弹到中轨受阻
  操作：中轨 + 缩量 = 卖出
  突破：突破中轨 + 放量 = 跟进
```

### 实战口诀
```
股价触上轨，超买要减仓
股价触下轨，超卖可建仓
带宽收窄时，等待变盘方向
带宽开口时，顺势可跟进
中轨是生命线，跌破要止损
```

## 模块 6: 成交量指标 (VOL)

### 成交量均线

| 均线 | 周期 | 用途 |
|------|------|------|
| **VOL_MA5** | 5 日 | 短期资金动向 |
| **VOL_MA20** | 20 日 | 中期资金动向 |
| **VOL_MA60** | 60 日 | 长期资金动向 |

### 核心信号

#### 1. 量比
```
量比 = 今日成交量 / 5 日平均成交量

量比 > 3: 巨量，警惕变盘
量比 2-3: 明显放量，趋势加强
量比 1.5-2: 温和放量，健康
量比 0.8-1.5: 正常
量比 < 0.5: 极度缩量，可能变盘
```

#### 2. 量价配合
```
量增价涨: ✅ 健康上涨，持有
量缩价涨: ⚠️ 上涨乏力，减仓
量增价跌: 🔴 资金出逃，止损
量缩价跌: 🟡 下跌放缓，观望
```

#### 3. 成交量突破
```
成交量上穿 VOL_MA20:
  含义：资金开始活跃
  操作：跟进

成交量下穿 VOL_MA20:
  含义：资金开始撤离
  操作：减仓
```

## 模块 7: ATR - 平均真实波幅

### ATR 计算
```
ATR = N 日内真实波幅的平均值
真实波幅 = max(最高 - 最低，|最高 - 昨收 |，|最低 - 昨收|)

常用周期: ATR(14)
```

### 核心用途

#### 1. 止损位设置
```
止损位 = 入场价 - 2×ATR
例：入场价 10 元，ATR=0.5 元
    止损位 = 10 - 2×0.5 = 9 元
```

#### 2. 止盈位设置
```
止盈位 = 入场价 + 3×ATR
例：入场价 10 元，ATR=0.5 元
    止盈位 = 10 + 3×0.5 = 11.5 元
```

#### 3. 波动率判断
```
ATR 上升:
  含义：波动率增加，趋势形成
  操作：顺势操作

ATR 下降:
  含义：波动率降低，震荡整理
  操作：观望
```

## 模块 8: 指标组合实战

### 最佳组合

#### 1. 趋势组合 (MA+MACD) 🟢🟢
```
买入条件:
  MA 多头排列
  MACD 金叉

```

#### 2. 超买超卖组合 (RSI+KDJ) 🟢🟢
```
买入条件:
  RSI < 20 (超卖)
  KDJ < 20 (超卖)

```

#### 3. 波动率组合 (BOLL+VOL) 🟢🟢
```
买入条件:
  股价触及下轨
  成交量缩量

成功率: 65%+
```

#### 4. 全能组合 (MA+MACD+RSI+VOL) 🟢🟢🟢
```
买入条件:
  MA 多头排列
  MACD 金叉
  RSI 50-70 (强势但不超买)
  成交量温和放量
  
成功率: 75%+
```

### 指标权重

| 指标 | 权重 | 说明 |
|------|------|------|
| **MA** | 30% | 趋势判断 |
| **MACD** | 25% | 动能判断 |

## 核心心法

1. **趋势第一** — MA 和 MACD 定方向
2. **量价配合** — 上涨必须放量
3. **超买超卖** — RSI 和 KDJ 找买卖点
4. **指标共振** — 多个指标同信号更可靠
5. **不迷信指标** — 指标是辅助，不是圣杯
6. **止损第一** — 指标失灵时止损保命
7. **简单有效** — 精通 2-3 个指标胜过浅尝 10 个
