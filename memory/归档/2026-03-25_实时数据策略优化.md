- "实时涨幅榜里面肯定还有高确定性的" → 需要真实全市场数据

### 2. 高确定性推荐策略重构

**文件**: `tools/高确定性推荐 - 定时任务.py`

#### 数据源优化

**之前**: 腾讯财经固定股票池 (500 只)
```python
# ❌ 问题：硬编码列表，不包含实时热点
from tencent_stock_rank import get_realtime_rank
```

**现在**: 新浪财经实时涨幅榜 (全市场)
```python
# ✅ 优势：真实全市场涨幅排名
def fetch_top_gainers(limit=100):
    stocks = _fetch_sina(limit)  # 新浪财经
    if not stocks:
        stocks = _fetch_eastmoney(limit)  # 东方财富备用
    if not stocks:
        stocks = _fetch_tencent(limit)  # 腾讯财经最后备用
    return stocks
```

#### 评分标准优化

| 涨幅区间 | 之前评分 | 现在评分 | 策略意图 |

## 🎯 新策略核心

### 潜伏主升浪策略

**核心思想**: 涨停买不进，要在涨停前介入

**买点选择**:
1. **最佳**: 7-9% 主升浪加速段 (70 分)
2. **次佳**: 5-7% 强势上涨 (55 分)
3. **潜伏**: 3-5% 温和上涨 (40 分)

**卖点选择**:
- 涨停 (9-11%) 出货
- 目标盈利：3-8%
- 止损位：-5%
