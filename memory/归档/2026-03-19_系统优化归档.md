## 📝 重要经验总结

### 技术经验

1. **路径规范**: 必须使用绝对路径，避免 `~`
2. **缓存管理**: 自动清理比手动更可靠
3. **错误告警**: 连续 3 次失败立即告警
4. **实时更新**: 重大变更立即记录，不要等
5. **执行时间**: 记录趋势，提前发现劣化

### 系统优化

1. **频率平衡**: 及时性和资源消耗要平衡
2. **自动化**: 能自动的不手动
3. **备份机制**: 多重备份，防止丢失
4. **监控机制**: 主动监控，被动响应
5. **文档记录**: 每次优化都要记录

### 交易相关

1. **形态学习**: 历史胜率很重要
2. **持仓优先**: 优先关注持仓股
3. **止损纪律**: 严格执行 -5% 止损
4. **推荐谨慎**: ≥75 分才推荐
5. **空仓等待**: 无机会时宁可空仓
