## 🔍 修复方法

**使用 sed 批量替换**:
```bash

```

**参数说明**:
- `-i`: 直接修改文件
- `s|旧|新|g`: 全局替换
- 使用 `|` 分隔符避免与路径中的 `/` 冲突

## 📚 路径规范

### ❌ 错误写法
```python

```

### ✅ 正确写法
```python
# 方法 1: 绝对路径 (推荐)
path = "/home/admin/openclaw/workspace/temp/file.txt"

# 方法 2: 展开用户目录
import os
path = os.path.expanduser("~/openclaw/workspace/temp/file.txt")

# 方法 3: pathlib
from pathlib import Path
path = Path.home() / "openclaw" / "workspace" / "temp" / "file.txt"
```
