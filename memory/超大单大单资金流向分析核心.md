# 超大单/大单资金流向分析核心

**学习时间**: 2026-03-15
**核心理念**: 资金流向是短线交易的灵魂，超大单和大单是主力动向的最直接体现

---

## 🎯 为什么超大单/大单最重要？

### 资金分级与代表群体

| 订单类型 | 金额范围 | 代表群体 | 信号可靠性 | 权重 |
|----------|----------|----------|------------|------|
| **超大单** | > 100 万元 | 机构/主力/国家队 | ⭐⭐⭐⭐⭐ | 25 分 |
| **大单** | 20-100 万元 | 大户/游资/私募 | ⭐⭐⭐⭐ | 15 分 |
| **中单** | 4-20 万元 | 中户/大户 | ⭐⭐⭐ | - |
| **小单** | < 4 万元 | 散户 | ⭐⭐ (反向) | - |

### 核心逻辑

```
超大单 + 大单 = 主力资金动向 (最可靠)
中单 = 跟随者 (参考)
小单 = 散户情绪 (反向指标)
```

**为什么可靠？**
1. **信息优势**: 机构/主力有信息渠道优势
2. **研究优势**: 专业团队研究，不是拍脑袋
3. **资金优势**: 大资金进出需要时间，形成趋势
4. **真金白银**: 可以用钱投票，不是嘴炮

## 📊 资金流向组合信号

### 最强势信号 🟢🟢🟢
```
超大单：大幅净流入 (>5000 万)
大单：大幅净流入 (>3000 万)
中单：小幅流入/流出
小单：大幅净流出

含义：主力建仓，散户出逃
操作：坚决跟进
```

### 最危险信号 🔴🔴🔴
```
超大单：大幅净流出 (<-5000 万)
大单：大幅净流出 (<-3000 万)
中单：小幅流入
小单：大幅净流入

含义：主力出货，散户接盘
操作：坚决止损/清仓
```

### 洗盘信号 🟡
```
超大单：小幅流出/持平
大单：小幅流出
中单：流入
小单：流入

含义：主力洗散户，不是真出货
操作：持有观察
```

### 分歧信号 ⚪
```
超大单：流入
大单：流出
或
超大单：流出
大单：流入

含义：主力内部有分歧
操作：观望，等方向明确
```

## 🔍 实战应用场景

### 场景 1: 涨停板 + 超大单流入

**数据**:
- 价格：涨停 (+10%)
- 超大单：+8000 万
- 大单：+5000 万
- 小单：-3000 万

**解读**: 主力真金白银封板，散户出逃
**次日策略**: 高开概率大，持有/加仓

### 场景 2: 涨停板 + 超大单流出

**数据**:
- 价格：涨停 (+10%)
- 超大单：-5000 万
- 大单：-2000 万
- 小单：+8000 万

**解读**: 主力借涨停出货，散户接盘
**次日策略**: 低开概率大，坚决卖出

### 场景 3: 大跌 + 超大单流入

**数据**:
- 价格：大跌 (-8%)
- 超大单：+6000 万
- 大单：+3000 万
- 小单：-1 亿

**解读**: 主力借大跌吸筹，散户恐慌出逃
**次日策略**: 可能反弹，关注抄底机会

### 场景 4: 大涨 + 超大单流出

**数据**:
- 价格：大涨 (+8%)
- 超大单：-4000 万
- 大单：-2000 万
- 小单：+7000 万

**解读**: 主力借大涨出货，散户追高
**次日策略**: 警惕回调，减仓/止盈

## 💡 核心心法

1. **超大单是最强信号** — 机构/主力不会轻易暴露动向，一旦暴露就是真动作
2. **大单确认趋势** — 游资/大户跟进，说明趋势形成
3. **小单是反向指标** — 散户集体看好时，往往是顶部
4. **量价 + 资金双验证** — 价格上涨 + 超大单流入 = 健康上涨
5. **连续观察** — 单日数据可能偶然，连续 3 日趋势更可靠

---

## ⚠️ 注意事项

1. **数据延迟**: 资金流向数据可能有 15 分钟延迟
2. **对倒操作**: 主力可能对倒制造假象，需要结合其他指标
3. **板块联动**: 个股资金流向要结合板块整体看
4. **大盘环境**: 大盘暴跌时，主力也可能被动减仓
5. **消息面**: 突发消息可能改变资金流向
