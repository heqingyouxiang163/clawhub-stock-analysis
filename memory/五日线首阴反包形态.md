# 五日均线首阴反包形态

**学习时间**: 2026-03-15
**核心目标**: 掌握强势股首次回调 5 日线，次日反包涨停的操盘手法

---

## 🎯 形态定义

**五日均线首阴**:
- 股票处于上升趋势 (5 日线向上)
- 连续上涨后首次收阴线
- 股价回调到 5 日均线附近
- 成交量缩量 (主力未出逃)

**次日反包涨停**:
- 次日高开或平开
- 盘中拉升涨停
- 成交量放大 (资金回流)
- 形成"阴包阳"或"阳包阴"

---

## 模块 1: 形态特征

### 前置条件 (必须满足)

| 条件 | 要求 | 说明 |
|------|------|------|
| **趋势向上** | 5 日线、10 日线、20 日线多头排列 | 确保是强势股 |
| **连续上涨** | 之前至少 3-5 根阳线 | 确立强势地位 |
| **首次回调** | 第一次收阴线 | 不是连续下跌 |
| **缩量回调** | 阴线成交量 < 前日阳线 | 主力未出逃 |
| **支撑有效** | 股价在 5 日线附近企稳 | 确认支撑 |

### K 线形态

```
D1: 大阳线 (+5% 以上)
    成交量：放量
    收盘价：接近最高价
    
D2: 首阴 (-1% 至 -5%)
    成交量：缩量 (关键!)
    最低价：触及 5 日线
    收盘价：在 5 日线上方
    
D3: 反包涨停 (+10%)
    开盘：高开/平开
    盘中：拉升涨停
    成交量：放大
```

## 模块 2: 选股条件

### 量化选股公式

```python
# 条件 1: 趋势向上
MA5 > MA10 > MA20
MA5 斜率 > 0 (向上)

# 条件 2: 连续上涨
昨日收盘价 > 前日收盘价 > 大前日收盘价 (至少 3 连阳)

# 条件 3: 首阴
今日阴线 (收盘价 < 开盘价)
今日涨跌幅：-1% 至 -5%
今日是首次阴线 (之前至少 3 连阳)

# 条件 4: 缩量回调
今日成交量 < 昨日成交量 * 0.8
今日成交量 > 5 日均量 * 0.5 (不是极度缩量)

# 条件 5: 5 日线支撑
今日最低价 <= MA5 * 1.02 (触及或接近 5 日线)
今日收盘价 >= MA5 * 0.98 (收盘在 5 日线附近)

# 条件 6: 资金流向
今日超大单流入 > 0 或 小幅流出 (<1000 万)
```

### 板块要求

| 板块状态 | 成功率 | 说明 |
|----------|--------|------|
| 板块龙头 | 70%+ | 优先选择 |
| 板块强势 | 60%+ | 板块指数向上 |
| 板块跟风 | 40%+ | 谨慎参与 |
| 板块退潮 | <20% | 放弃 |

## 模块 3: 买入时机

### 集合竞价阶段 (9:15-9:25)

**观察要点**:
1. 开盘幅度
2. 封单变化
3. 竞价成交量

**理想开盘**:
- 高开 2-5%：最佳 (强势信号)
- 平开 ±1%：可以接受
- 低开 -1% 至 -3%：观察反抽

**警惕信号**:
- 低开 -5% 以下：放弃
- 竞价巨量：警惕出货

### 开盘后买入点

#### 买点 1: 高开快速拉升 (最强) 🟢🟢🟢

```
条件:
  开盘：高开 2-5%
  开盘后：快速拉升 (>3%/分钟)
  成交量：放大
  板块：集体上涨

操作:
  9:30-9:35: 买入 30-50%
  突破昨日高点：加仓 20-30%
  涨停：持有
```

**成功率**: 70%+

#### 买点 2: 平开后拉升 (稳健) 🟢🟢

```
条件:
  开盘：平开 ±1%
  开盘后：站稳开盘价
  5 分钟内：突破开盘价
  成交量：温和放大

操作:
  突破开盘价：买入 30%
  突破昨日高点：加仓 30%
  涨停：持有
```

**成功率**: 60%+

#### 买点 3: 低开后反抽 (激进) 🟢

```
条件:
  开盘：低开 -1% 至 -3%
  开盘后：快速反抽翻红
  成交量：放大
  5 日线支撑有效

操作:
  翻红时：买入 20-30%
  突破昨日高点：加仓 20%
  不突破：观望
```

**成功率**: 50%+

## 模块 4: 盘中监控要点

### 9:30-9:35 (开盘 5 分钟)

**关键观察**:
1. 开盘是否符合预期
2. 拉升速度
3. 成交量匹配
4. 板块联动

**买入信号**:
```
✅ 高开 2%+ 快速拉升 → 买入
✅ 平开站稳后拉升 → 买入
✅ 低开快速反抽翻红 → 轻仓试错
```

**放弃信号**:
```
🔴 低开 -5% 以下 → 放弃
🔴 开盘后快速下跌 → 放弃
🔴 板块集体走弱 → 放弃
```

### 9:35-10:00 (确认期)

**关键观察**:
1. 是否突破昨日高点
2. 涨停封单情况
3. 炸板率
4. 情绪周期

**加仓信号**:
```
IF 突破昨日高点 + 放量 → 加仓
IF 即将涨停 + 封单大 → 持有
IF 板块多股涨停 → 确认
```

## 模块 8: 自动化监控

### 选股公式 (伪代码)

```python
def select_ma5_rebound_stocks():
    candidates = []
    
    for stock in all_stocks:
        # 条件 1: 趋势向上
        if not (MA5 > MA10 > MA20):
            continue
        
        # 条件 2: 连续上涨至少 3 天
        if not consecutive_gains(stock, 3):
            continue
        
        # 条件 3: 今日首阴
        if not is_first_negative(stock):
            continue
        
        # 条件 4: 缩量回调
        if volume_ratio(stock) > 0.8:
            continue
        
        # 条件 5: 5 日线支撑
        if not near_ma5_support(stock):
            continue
        
        # 条件 6: 资金未大幅流出
        if super_large_outflow(stock) > 1000:
            continue
        
        # 计算评分
        score = calculate_rebound_score(stock)
        
        candidates.append({
            "code": stock.code,
            "name": stock.name,
            "score": score,
        })
    
    # 按评分排序
    return sorted(candidates, key=lambda x: x["score"], reverse=True)
```

## 核心心法

1. **只做首次回调** — 第二次、第三次成功率大降
2. **必须缩量** — 放量说明主力出逃
3. **5 日线是生命线** — 跌破坚决止损
4. **板块联动** — 板块强成功率高
5. **集合竞价定强弱** — 高开最佳，低开谨慎
6. **快进快出** — 不恋战，次日不涨停就卖
