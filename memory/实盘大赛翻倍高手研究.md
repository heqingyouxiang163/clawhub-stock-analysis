## 模块 3: 翻倍高手的核心心法

### 选股心法

```
1. 只做龙头，不碰跟风
   - 板块涨幅第 1-3
   - 涨停时间最早
   - 封单金额最大

2. 打板确认，不猜顶
   - 涨停再买，不追高
   - 首板最安全
   - 二板确认加仓

3. 集合竞价定强弱
   - 高开 2-5% 最佳
   - 封单增加确认
   - 竞价量放大

4. 情绪周期选股
   - 冰点试错首板
   - 启动参与龙头
   - 高潮持股待涨
   - 退潮空仓观望
```

### 买卖心法

```
买入:
  1. 集合竞价高开 5%+ → 跟进
  2. 分时均线向上 → 持有
  3. 涨停确认 → 打板
  4. 五日线首阴 → 反包买入

卖出:
  1. 不涨停 → 次日卖出
  2. 跌破分时均线 → 止损
  3. 跌破 5 日线 → 清仓
  4. 高开低走 → 反抽卖出
  5. 盈利 20%+ → 分批止盈
```

### 仓位心法

```
单只股票仓位:
  首板试错：20-30%
  二板确认：30-40%
  龙头主升：40-50%
  高位风险：20-30%

总仓位控制:
  情绪冰点：0-20%
  情绪启动：30-50%
  情绪强势：70-90%
  情绪高潮：50-70%(止盈)
  情绪退潮：0%
```

### 止损心法

```
止损铁律:
  1. 亏损 -5% → 无条件止损
  2. 跌破分时均线 → 立即卖出
  3. 跌破 5 日线 → 清仓
  4. 高开低走翻绿 → 跑
  5. 炸板不回封 → 跑

止损心态:
  - 止损是成本，不是失败
  - 连续止损 3 次 → 暂停交易
  - 保住本金，不怕没机会
  - 一次大亏需要 3 次盈利弥补
```

## 模块 5: 翻倍高手的选股公式

### 首板选股公式

```
条件:
  ✅ 集合竞价高开 2-5%
  ✅ 竞价量 > 昨日 10%
  ✅ 板块集体高开
  ✅ 昨日有异动 (涨 3-7%)
  ✅ 流通市值 50-200 亿
  ✅ 股价 5-50 元
  ✅ 无利空消息

成功率：65%
```

### 连板选股公式

```
条件:
  ✅ 昨日涨停
  ✅ 涨停质量 A 级 (早盘封板)
  ✅ 封单金额 > 2 亿
  ✅ 量比 1.5-3 倍
  ✅ 超大单流入 > 3000 万
  ✅ 板块强势
  ✅ 连板数≤4

成功率：60%
```

### 五日线首阴公式

```
条件:
  ✅ 趋势向上 (MA5>MA10>MA20)
  ✅ 之前 3-5 连阳
  ✅ 今日首阴 (-1% 至 -5%)
  ✅ 缩量回调 (<0.8 倍)
  ✅ 触及 5 日线
  ✅ 超大单未大幅流出

成功率：70%
```

### 突破平台公式

```
条件:
  ✅ 盘整至少 5 天
  ✅ 今日突破平台高点
  ✅ 成交量 > 突破前 2 倍
  ✅ 涨幅 5-8%
  ✅ 板块配合
  ✅ 超大单流入

成功率：65%
```

## 模块 6: 翻倍高手的风控体系

### 单笔风险控制

| 风险指标 | 阈值 | 操作 |
|----------|------|------|
| 单笔亏损 | -5% | 无条件止损 |
| 单日亏损 | -10% | 停止交易 |
| 连续亏损 | 3 次 | 暂停 1 天 |
| 单只仓位 | 50% | 不超过 |
| 总仓位 | 100% | 不满仓 |

### 回撤控制

```
最大回撤控制:
  总资金回撤 -10% → 仓位减半
  总资金回撤 -15% → 仓位 30%
  总资金回撤 -20% → 空仓 1 周

复利保护:
  盈利 50% 后 → 出金 20%
  盈利 100% 后 → 出金 30%
  盈利 200% 后 → 出金 50%
```

### 黑天鹅应对

```
突发利空:
  1. 立即卖出，不抱幻想
  2. 跌停排队，能出就出
  3. 次日反抽，坚决清仓

板块退潮:
  1. 龙头跌停 → 清仓跟风
  2. 多股跌停 → 全部清仓
  3. 空仓观望，等情绪企稳

大盘暴跌:
  1. 大盘跌 -3% → 仓位 30%
  2. 大盘跌 -5% → 空仓
  3. 不逆势操作
```

## 模块 7: 翻倍高手的心理素质

### 心态管理

```
成功心态:
  ✅ 接受亏损 (止损是成本)
  ✅ 不抱幻想 (破位就跑)
  ✅ 不后悔 (错过不追)
  ✅ 不贪婪 (止盈果断)
  ✅ 不恐惧 (信号到就买)
  ✅ 有耐心 (等最佳机会)
  ✅ 有纪律 (严格执行)

失败心态:
  ❌ 侥幸 (再等等看)
  ❌ 贪婪 (再涨一点卖)
  ❌ 恐惧 (不敢买)
  ❌ 愤怒 (报复性交易)
  ❌ 自负 (不按规则)
  ❌ 急躁 (频繁交易)
```

### 情绪控制

```
交易前:
  - 深呼吸，平静心态
  - 回顾交易规则
  - 确认今日计划

交易中:
  - 严格执行计划
  - 不临时改变决策
  - 不被盘面左右

交易后:
  - 盈利不骄傲
  - 亏损不气馁
  - 复盘总结
```
