## 什么是超短线

**定义**: 持有时间 1-3 天的交易方式

**核心**:
- 快进快出
- 追求胜率 + 盈亏比
- 不恋战，不格局

---

## 超短线核心要素

### 1. 情绪周期

**四个阶段**:
1. **冰点期**: 涨停少，跌停多 → 观望
2. **回暖期**: 涨停增加，连板出现 → 试错
3. **上升期**: 连板高度提升，赚钱效应好 → 重仓
4. **高潮期**: 百股涨停，情绪亢奋 → 减仓
5. **退潮期**: 高位股核按钮，跌停增加 → 空仓

### 2. 题材炒作

**题材生命周期**:
1. 消息刺激 → 2. 龙头崛起 → 3. 板块发酵 → 4. 高潮分化 → 5. 退潮

**操作要点**:
- 做龙头，不做跟风
- 做主流，不做冷门
- 做早期，不做末期

### 3. 量价关系

**健康量价**:
- 上涨放量，下跌缩量
- 换手率 10%-30% (充分换手)
- 量比>2 (资金活跃)

**危险信号**:
- 高位缩量加速
- 放巨量滞涨
- 尾盘突然放量

## 常用术语

| 术语 | 含义 |
|------|------|
| 打板 | 涨停价买入 |
| 半路 | 涨幅 5%-8% 时买入 |
| 低吸 | 绿盘或低位买入 |
| 核按钮 | 跌停价卖出 |
| 弱转强 | 昨日弱，今日强 |
| 连板 | 连续涨停 |
| 炸板 | 涨停打开 |
| 回封 | 炸板后重新封板 |
| 换手 | 成交量/流通股本 |
| 量比 | 今日成交量/过去 5 日平均 |

## 超短线纪律

1. **止损第一**: -7% 硬止损，不抱幻想
2. **不恋战**: 不达预期就走，不格局
3. **空仓也是操作**: 行情不好就休息
4. **复盘必须做**: 每天总结得失
5. **仓位管理**: 单股不超过 4 成

---

## 新手常见错误

1. ❌ 追高杀跌 (情绪化交易)
2. ❌ 不止损 (越套越深)
3. ❌ 满仓干 (没有仓位管理)
4. ❌ 做冷门 (没有流动性)
5. ❌ 频繁交易 (没有选择性)
