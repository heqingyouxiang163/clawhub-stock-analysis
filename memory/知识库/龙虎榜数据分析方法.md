**用途**: 解读龙虎榜数据

---

## 什么是龙虎榜

**定义**: 交易所公布的当日异动个股买卖前五席位

**上榜条件**:
- 日涨跌幅±7%
- 日换手率>20%
- 日振幅>15%
- 连续 3 日涨跌幅偏离值累计±20%

---

## 席位分类

### 机构席位
- **特征**: "机构专用"
- **含义**: 公募基金、保险、社保等
- **解读**: 机构买入=利好，机构卖出=利空

### 游资席位
- **特征**: 券商营业部名称
- **知名游资**:
  - 章盟主：中信证券杭州延安路
  - 作手新一：国泰君安南京太平南路
  - 赵老哥：中国银河绍兴
  - 炒股养家：华鑫证券上海宛平南路
  - 小鳄鱼：南京证券南京大钟亭

### 北向资金
- **特征**: "深股通专用"/"沪股通专用"
- **含义**: 外资通过港股通买入
- **解读**: 持续流入=看好

---

## 龙虎榜解读技巧

### 1. 看净买入

**公式**: 净买入 = 买入总额 - 卖出总额

**解读**:
- 净买入>5000 万：强势
- 净买入>1 亿：非常强势
- 净卖出>5000 万：弱势
- 净卖出>1 亿：非常弱势

### 2. 看买卖集中度

**买入集中度** = 前五买入 / 总成交

**解读**:
- >30%：资金集中，主力明显
- 20%-30%：正常
- <20%：资金分散，无明显主力

### 3. 看席位联动

**一字型**: 买一独大，其他少 → 主力控盘
**均匀型**: 买卖均匀 → 游资接力
**机构型**: 机构主导 → 趋势行情

### 4. 看席位历史

**知名游资入驻**:
- 历史战绩好 = 加分
- 历史砸盘多 = 减分

## 龙虎榜实战应用

### 买入信号
1. 机构大笔买入 (>5000 万)
2. 知名游资入驻 (战绩好)
3. 买一独大，净买入高
4. 北向资金持续流入

### 卖出信号
1. 机构大笔卖出
2. 知名砸盘游资入驻
3. 卖一独大，净卖出高
4. 高位股龙虎榜 (容易核按钮)

### 注意事项
1. 龙虎榜是 T-1 数据 (滞后)
2. 可能是对倒 (自买自卖)
3. 结合K 线位置判断 (低位/高位)
4. 结合情绪周期判断
