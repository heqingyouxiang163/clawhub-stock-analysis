## 模块 2: 集合竞价操盘手法

### 9:15-9:20 (可撤单阶段)

**高手观察点**:
1. 昨日涨停股封单变化
2. 是否有大单撤单
3. 板块集体高开情况
4. 情绪指标 (涨跌比)

**警惕信号**:
```
IF 昨日涨停今日跌停封单 → 情绪极差，空仓
IF 板块龙头封单快速减少 → 主力撤单，观望
IF 高开股集体回落 → 情绪转弱，谨慎
```

### 9:20-9:25 (不可撤单阶段)

**高手决策点**:
1. 最终高开幅度
2. 真实封单金额
3. 竞价成交量
4. 板块联动

**买入信号**:
```
✅ 龙头股高开 5-8% + 封单>1 亿 + 竞价放量 → 跟进
✅ 板块集体高开 + 多股涨停预期 → 打板龙头
✅ 昨日 A 级涨停 + 今日高开 7%+ → 持有/加仓
```

**卖出信号**:
```
🔴 昨日涨停今日低开 -5% → 反抽卖出
🔴 封单<5000 万 + 高开回落 → 减仓
🔴 板块退潮 + 龙头跌停预期 → 清仓
```

## 模块 3: 开盘后操盘手法

### 9:30-9:35 (开盘 5 分钟)

**高手观察**:
1. 是否符合集合竞价预判
2. 快速拉升/下跌速度
3. 成交量是否匹配
4. 板块联动情况

**打板信号**:
```
IF 高开 5%+ 快速拉升 → 准备打板
IF 突破关键价位 + 放量 → 跟进
IF 板块多股拉升 → 打龙头
```

### 9:35-10:00 (确认期)

**高手操作**:
1. 确认涨停股封单
2. 观察炸板率
3. 判断当日情绪
4. 决定是否加仓

**情绪判断**:
```
涨停家数 > 50 + 炸板率 < 20% → 情绪好，积极
涨停家数 20-50 + 炸板率 20-40% → 中性，谨慎
涨停家数 < 20 + 炸板率 > 40% → 情绪差，空仓
```

## 模块 5: 盘中预警系统

### 一级预警 (极度危险) 🔴🔴🔴

**触发条件**:
- 持仓股跌停封单 > 10 万手
- 持仓股低开 -8% 以下
- 板块集体跌停
- 市场跌停家数 > 50

**应对**:
```
立即执行止损，不要犹豫
能卖出就卖出，不抱幻想
```

### 二级预警 (高风险) 🔴🔴

**触发条件**:
- 持仓股低开 -5% 至 -8%
- 持仓股快速杀向跌停
- 昨日涨停股集体低开
- 炸板率 > 40%

**应对**:
```
准备止损，等待反抽
反抽无力坚决卖出
```

### 三级预警 (注意) 🔴

**触发条件**:
- 持仓股跌破开盘价
- 持仓股无法翻红
- 板块走弱
- 涨停封单减少

**应对**:
```
减仓，设置止损位
观察是否企稳
```

### 机会预警 (买入信号) 🟢

**触发条件**:
- 持仓股高开 5%+ 快速拉升
- 板块集体启动
- 龙头股即将涨停
- 情绪冰点后的首板

**应对**:
```
确认信号后跟进
打板确认最安全
仓位 30-50%
```

### 9:15-9:25 (集合竞价)

```
1. 查看昨日涨停股开盘情况
2. 确认封单金额变化
3. 判断市场情绪
4. 制定当日预案
5. 9:24 分做出最终决策
```

### 9:30-9:35 (开盘)

```
1. 确认开盘是否符合预期
2. 观察快速拉升/下跌
3. 执行集合竞价决策
4. 准备打板/止损
```

### 9:35-10:00 (确认)

```
1. 确认涨停股封单
2. 观察炸板率
3. 判断情绪周期
4. 决定是否加仓/减仓
```

### 10:00-11:30 (持仓)

```
1. 监控持仓股走势
2. 观察板块轮动
3. 等待新机会

```

### 13:00-14:00 (午后)

```
1. 确认上午趋势
2. 加仓强势股
3. 减仓弱势股

```

### 14:00-15:00 (尾盘)

```
1. 判断是否持股过夜
2. 止盈/止损决策
3. 尾盘买入法选股
4. 总结当日操作
```

## 模块 7: 情绪周期判断

### 情绪周期四阶段

| 阶段 | 特征 | 操作 | 仓位 |
|------|------|------|------|
| **冰点期** | 涨停<20，跌停>30 | 试错首板 | 0-20% |
| **启动期** | 涨停增加，出现新题材 | 参与龙头 | 30-50% |
| **高潮期** | 涨停>80，连板高度高 | 持仓/止盈 | 70-100% |
| **退潮期** | 涨停减少，跌停增加 | 空仓观望 | 0% |

### 情绪指标

```
情绪得分 = 
  涨停家数得分 × 30% +
  跌停家数得分 × 25% +
  炸板率得分 × 20% +
  连板高度得分 × 15% +
  北向资金得分 × 10%
```

**操作指导**:
- 情绪得分 > 80: 高潮期，止盈为主
- 情绪得分 60-80: 强势期，积极操作
- 情绪得分 40-60: 震荡期，谨慎操作
- 情绪得分 < 40: 冰点期，空仓/试错

## 模块 8: 实战交割单分析

### 典型案例 1: 龙头股连板

```
D1: 首板 (早盘快速封板)
    操作：打板买入
    仓位：30%
    
D2: 二板 (高开 7% 快速涨停)
    操作：持有
    仓位：30%
    
D3: 三板 (高开 8% 一字板)
    操作：持有
    仓位：30%
    
D4: 四板 (高开 5% 炸板回落)
    操作：止盈卖出
    收益：+40%
```

**关键点**:
- 只做龙头，不碰跟风
- 涨停确认后再买
- 炸板立即止盈

### 典型案例 2: 止损案例

```
D1: 打板买入
    仓位：30%
    
D2: 低开 -6%
    操作：反抽 -3% 卖出
    亏损：-3%
```

**关键点**:
- 低开坚决止损
- 反抽就是逃命机会
- 不抱幻想

## 核心心法

1. **集合竞价定生死** — 9:15-9:25 最重要
2. **只做龙头** — 跟风股不碰
3. **打板确认** — 不猜顶，涨停再买
4. **止损果断** — 触及止损位立即执行
5. **情绪周期** — 冰点试错，高潮止盈
6. **仓位集中** — 看好就重仓，不看好就空仓
7. **不频繁操作** — 一天最多 2-3 笔
8. **尾盘不追高** — 14:30 后不新开仓
