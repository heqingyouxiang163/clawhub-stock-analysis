## 模块 1: 量化基础概念 ✅

### 什么是量化交易？

**定义**: 量化交易 (Quantitative Trading) 是利用数学模型、统计分析和计算机算法来进行交易决策的方法。

**核心特点**:
- 📐 **数据驱动**: 基于历史数据和实时数据做决策，不靠"感觉"
- 🤖 **规则明确**: 买卖条件事先定义好，严格执行
- ⚡ **快速执行**: 计算机自动监控和交易
- 📊 **可回测**: 策略可以在历史数据上验证效果

### 量化 vs 主观交易

| 维度 | 量化交易 | 主观交易 |
|------|----------|----------|
| 决策依据 | 数据 + 模型 | 经验 + 直觉 |
| 执行方式 | 自动化/半自动 | 人工判断 |
| 情绪影响 | 几乎为零 | 影响较大 |
| 可复制性 | 高 | 低 |
| 适合人群 | 理工科背景 | 经验丰富者 |

### 常见量化策略类型

1. **趋势跟踪策略** - 追涨杀跌，顺势而为
2. **均值回归策略** - 高抛低吸，赌回归
3. **多因子选股** - 综合多个指标打分
4. **统计套利** - 利用价格差异赚钱
5. **高频交易** - 毫秒级快速交易

## 模块 2: 技术指标体系 ✅

### 趋势指标

#### MA (移动平均线)
- **MA5/MA10/MA20**: 短期趋势
- **MA60/MA120/MA250**: 中长期趋势
- **金叉**: 短周期上穿长周期 → 看涨
- **死叉**: 短周期下穿长周期 → 看跌
- **多头排列**: MA5>MA10>MA20 → 强势上涨
- **空头排列**: MA5<MA10<MA20 → 弱势下跌

#### MACD (平滑异同移动平均线)
- **DIF**: 快线 (12 日 EMA - 26 日 EMA)
- **DEA**: 慢线 (DIF 的 9 日 EMA)
- **柱状图**: DIF - DEA
- **金叉**: DIF 上穿 DEA → 买入信号
- **死叉**: DIF 下穿 DEA → 卖出信号
- **顶背离**: 价格新高但 MACD 不新高 → 警惕回调
- **底背离**: 价格新低但 MACD 不新低 → 可能反弹

### 动量指标

#### RSI (相对强弱指标)
- **范围**: 0-100
- **>70**: 超买区，警惕回调
- **<30**: 超卖区，可能反弹
- **50 以上**: 多头市场
- **50 以下**: 空头市场

#### KDJ (随机指标)
- **K 线**: 快速线
- **D 线**: 慢速线
- **J 线**: 方向敏感线
- **>80**: 超买
- **<20**: 超卖
- **K 上穿 D**: 金叉买入
- **K 下穿 D**: 死叉卖出

### 成交量指标

#### 量比
- **公式**: 当日成交量 / 过去 5 日平均成交量
- **>2.5**: 放量明显，趋势可能延续
- **<0.5**: 缩量明显，可能变盘
- **1 左右**: 正常水平

#### OBV (能量潮)
- 成交量与价格的关系
- **OBV 上升 + 价格上升**: 健康上涨
- **OBV 上升 + 价格下跌**: 主力吸筹
- **OBV 下降 + 价格上升**: 主力出货

### 波动率指标

#### 布林带 (Bollinger Bands)
- **上轨**: 中轨 + 2 倍标准差
- **中轨**: 20 日均线
- **下轨**: 中轨 - 2 倍标准差
- **突破上轨**: 超买，可能回调
- **跌破下轨**: 超卖，可能反弹
- **收口**: 波动率降低，即将变盘
- **开口**: 波动率增加，趋势形成

#### ATR (平均真实波幅)
- 衡量价格波动幅度
- 用于设置止损位
- **止损位** = 入场价 ± 2×ATR

## 模块 3: 量化选股方法 ✅

### 多因子模型框架

**基本思路**: 给每只股票多个维度打分，综合排序选股

#### 基本面因子
| 因子 | 含义 | 偏好 |
|------|------|------|
| PE (市盈率) | 估值水平 | 低 PE 通常更好 |
| PB (市净率) | 市价/净资产 | 低 PB 更安全 |
| ROE (净资产收益率) | 盈利能力 | 高 ROE 更优 |
| 营收增长率 | 成长性 | 高增长更好 |
| 净利润增长率 | 盈利成长 | 稳定增长最佳 |
| 资产负债率 | 财务风险 | <60% 较安全 |

#### 技术面因子
| 因子 | 含义 | 信号 |
|------|------|------|
| 动量因子 | 过去 N 日涨幅 | 强者恒强 |
| 反转因子 | 超跌反弹 | 跌多必涨 |
| 波动率因子 | 价格波动程度 | 低波动更稳 |
| 均线因子 | 价格与均线关系 | 站上均线好 |

#### 资金面因子
| 因子 | 含义 | 信号 |
|------|------|------|
| 北向资金 | 外资流向 | 净流入好 |
| 主力资金 | 大单流向 | 净流入好 |
| 融资余额 | 杠杆资金 | 增加看好 |
| 龙虎榜 | 游资动向 | 机构买入好 |

### 因子打分示例

```
股票 A 综合得分 = 
  基本面得分 × 40% + 
  技术面得分 × 35% + 
  资金面得分 × 25%
```

## 模块 4: 回测与评估 ✅

### 回测框架

**步骤**:
1. 定义策略规则 (买入/卖出条件)
2. 获取历史数据
3. 模拟交易 (考虑手续费、滑点)
4. 统计收益和风险指标

### 核心评估指标

| 指标 | 含义 | 优秀标准 |
|------|------|----------|
| **年化收益率** | 年均回报 | >15% |
| **夏普比率** | 风险调整后收益 | >1 良好，>2 优秀 |
| **最大回撤** | 最大亏损幅度 | <20% 较好 |
| **胜率** | 盈利交易占比 | >50% |
| **盈亏比** | 平均盈利/平均亏损 | >1.5 |
| **交易次数** | 总交易数 | 足够统计意义 |

### 夏普比率 (Sharpe Ratio)
```
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
```
- **>1**: 策略有效
- **>2**: 策略优秀
- **<0**: 策略不如存银行

### 最大回撤 (Max Drawdown)
```
最大回撤 = (峰值 - 谷值) / 峰值 × 100%
```
- 衡量最坏情况下的亏损
- 心理承受能力参考

### 过拟合问题

**表现**: 回测收益很高，实盘一塌糊涂

**原因**: 策略参数过度优化，只适应历史数据

**解决方法**:
- 样本外测试 (用没见过的数据验证)
- 简化策略 (参数越少越好)
- 逻辑驱动 (有经济学逻辑支撑)

## 模块 5: 实战应用 ✅

### A 股量化策略案例

#### 策略 1: 双均线趋势策略
```
买入条件:
- MA5 上穿 MA20 (金叉)
- 成交量放大 (>1.5 倍均量)
- RSI > 50 (多头区域)

卖出条件:
- MA5 下穿 MA20 (死叉)
- 或者亏损达到 8% 止损
- 或者盈利达到 20% 止盈
```

#### 策略 2: 低位反转策略
```
选股条件:
- RSI < 30 (超卖)
- 价格跌破布林带下轨
- 成交量萎缩后开始放大

买入: 次日开盘
卖出: RSI > 70 或 盈利 15%
```

#### 策略 3: 突破策略
```
买入条件:
- 价格突破 20 日新高
- 成交量 > 2 倍均量
- MACD 金叉

止损: 跌破突破 K 线最低价
止盈: 追踪止损 (最高点回撤 10%)
```

### 如何用量化方法分析持仓股

**分析框架**:

1. **趋势判断**
   - 价格在 MA20 上方还是下方？
   - MA 排列是多头还是空头？
   - MACD 是金叉还是死叉？

2. **动量评估**
   - RSI 在什么位置？
   - 近期涨幅是否过大？
   - 是否有背离信号？

3. **支撑压力**
   - 布林带上下轨位置
   - 前期高低点
   - 整数关口

4. **成交量验证**
   - 上涨是否放量？
   - 下跌是否缩量？
   - 量比是否异常？

5. **风险评估**
   - 止损位设在哪里？
   - 仓位应该多少？
   - 盈亏比是否合理？

### 构建简单交易系统

**入场三要素**:
1. ✅ 趋势方向正确
2. ✅ 信号触发 (技术指标)
3. ✅ 成交量确认

**出场三要素**:
1. ✅ 止损位 (保命)
2. ✅ 止盈位 (落袋)
3. ✅ 信号反转 (离场)

**仓位管理**:
- 单只股票不超过总资金 30%
- 总仓位根据市场趋势调整
- 盈利后加仓，亏损后减仓

## 关键收获

1. **量化不是预测，是概率** - 不追求 100% 准确，追求长期正期望
2. **纪律比技术重要** - 严格执行规则，不被情绪左右
3. **风险管理第一** - 先考虑亏多少，再考虑赚多少
4. **简单往往有效** - 复杂策略容易过拟合
5. **持续学习迭代** - 市场在变，策略也要进化
