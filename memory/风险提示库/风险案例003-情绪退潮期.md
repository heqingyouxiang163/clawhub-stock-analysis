# 风险案例 003: 情绪退潮期接飞刀

**风险等级**: ⚠️⚠️⚠️ 高危  
**发生频率**: 周期性  
**杀伤力**: 极大

---

## 案例描述

**典型特征**:
- 高位股集体跌停
- 连板股晋级率<20%
- 昨日涨停今日跌停
- 市场恐慌情绪蔓延

**风险逻辑**:
- 情绪退潮是系统性风险
- 个股再强也难挡大势
- 抄底=接飞刀
- 容易连续亏损

---

## 真实案例

### 案例 A: 202X 年 X 月退潮期
- **背景**: 前期龙头 7 连板后跌停
- **退潮特征**:
  - 高位股集体核按钮
  - 连板晋级率<10%
  - 跌停家数>50 家
- **抄底者**: 连续 3 次抄底，每次 -10%
- **总亏损**: -30%+

### 案例 B: 202X 年 X 月退潮期
- **背景**: 主流题材退潮
- **退潮特征**:
  - 板块龙头跌停
  - 跟风股集体大跌
  - 资金出逃明显
- **抄底者**: 以为低吸机会，结果 -15%

---

## 识别方法

**退潮信号**:
1. ✅ 高位股集体跌停 (≥3 只)
2. ✅ 连板晋级率<20%
3. ✅ 昨日涨停指数收跌
4. ✅ 跌停家数>涨停家数
5. ✅ 成交量萎缩 (资金观望)
6. ✅ 龙头股 A 杀 (连续跌停)

---

## 应对策略

### 最佳策略：空仓
- **退潮期第一原则**: 空仓观望
- **不要抄底**: 飞刀不能接
- **不要试错**: 胜率极低
- **等待冰点**: 冰点后才有机会

### 如果非要操作
- **仓位**: <1 成 (娱乐仓)
- **标的**: 低位首板 (避开高位)
- **止损**: -5% 就走 (不能格局)
- **预期**: 赚了就跑，不贪

---

## 教训总结

> **退潮期，空仓是最高的智慧！**

1. 情绪退潮是系统性风险，个股难挡
2. 抄底=接飞刀，越抄越亏
3. 空仓不丢人，亏钱才丢人
4. 等待冰点，冰点后才有新周期

**口诀**: 退潮空仓保本金，冰点之后新周期
