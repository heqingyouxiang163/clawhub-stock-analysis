# 任务执行报告模板

**用途**: 每次 cron 任务执行后自动输出简洁报告

---

## 报告格式 (超简版)

```
🦞 [任务名]
✅/❌ 状态 | ⏱️ 耗时 | 📝 关键结果
---
详情：1 句话总结
```

---

## 示例

### 示例 1: 成功

```
🦞 涨停形态每日学习
✅ 成功 | ⏱️ 248 秒 | 📊 44 只涨停，3 只连板
---
详情：已保存到 memory/涨停形态库/2026-03-17.md
```

### 示例 2: 失败

```
🦞 盘中监控 -10 点
❌ 失败 | ⏱️ 43 秒 | ⚠️ 数据 API 超时
---
原因：东方财富 API 响应超时，已重试 3 次
```

### 示例 3: 部分成功

```
🦞 自我学习升级
⚠️ 部分 | ⏱️ 187 秒 | 📝 3/5 任务完成
---
详情：策略优化完成，胜率统计待数据
```

---

## 状态图标

| 图标 | 含义 |
|------|------|
| ✅ | 完全成功 |
| ❌ | 失败 |
| ⚠️ | 部分成功/有警告 |
| ⏳ | 执行中 |

---

## 必填字段

1. **任务名** - 简洁明了
2. **状态** - ✅/❌/⚠️
3. **耗时** - 秒数
4. **关键结果** - 1 句话
5. **详情/原因** - 1 句话说明

---

## 使用场景

**所有 cron 任务执行后自动输出**:
- 每小时心跳学习
- 每日学习任务 (9 个)
- 每周学习任务 (2 个)
- 临时任务
