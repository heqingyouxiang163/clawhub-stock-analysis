```

  操作：买入 40%
  
D2: 二板 (早盘封板)
  竞价：高开 6%
  封单：2 亿
  操作：持有
  
D3: 三板 (一字板)
  竞价：高开 9%
  封单：3 亿
  操作：持有
  
D4: 断板
  竞价：高开 3%
  操作：止盈
  
总收益：+35%
```

## 2️⃣ 失败案例 (10 个)

### 失败案例 1: 追高 7 连板
```
买入：7 连板后追高
价格：50 元
卖出：35 元
亏损：-30%

教训:
  - 不追 5 板以上
  - 高潮期止盈
  - 严格止损
```

### 失败案例 2: 不止损
```
买入：20 元
止损位：19 元 (-5%)
实际卖出：15 元
亏损：-25%

教训:
  - 触及止损立即执行
  - 不抱幻想
  - -5% 是底线
```

### 失败案例 3: 抄底接飞刀
```
买入：连续跌停后抄底
价格：20 元
卖出：12 元
亏损：-40%

教训:
  - 不抄底跌停股
  - 等企稳信号
  - 不接飞刀
```

## 3️⃣ 龙虎榜案例 (10 个)

### 案例 1: 机构 + 游资共振
```
买入:
  - 机构专用：+5000 万
  - 作手新一：+3000 万
  - 赵老哥：+2000 万

次日：涨停
收益：+10%

教训:
  - 机构 + 游资共振是最强信号
  - 积极跟进
```

### 案例 2: 机构连续卖出
```
卖出:
  - 机构专用：-3000 万
  - 机构专用：-2000 万

次日：跌停
亏损：-10%

教训:
  - 机构连续卖出坚决卖出
  - 不抱幻想
```

## 4️⃣ 经验教训总结

### 成功经验
```
1. 只做龙头，不碰跟风
2. 打板确认，不猜顶
3. 集合竞价定强弱
4. 止损果断，不抱幻想
5. 仓位集中，不满仓
6. 情绪周期，冰点试错
7. 机构 + 游资共振最可靠
```

### 失败教训
```
1. 不追 5 板以上
2. 严格止损 -5%
3. 不抄底跌停股
4. 分批止盈
5. 高潮期不追高
6. 退潮期空仓
```
