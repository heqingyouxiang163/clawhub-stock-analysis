## 1️⃣ 失败案例分类

### 追高被套 (30%)

#### 典型案例
```
案例 1:
  股票：XXX
  买入：7 连板后追高
  价格：50 元
  卖出：35 元
  亏损：-30%
  
原因:
  - 高位接盘
  - 情绪高潮期追高
  - 不止损
  
教训:
  - 不追 5 板以上
  - 高潮期止盈
  - 严格止损
```

#### 共同特征
```
🔴 连板数≥5 板
🔴 情绪高潮期
🔴 媒体疯狂报道
🔴 缩量加速
🔴 不止损

避免方法:
  1. 不追 5 板以上
  2. 高潮期止盈
  3. 严格止损 -5%
```

---

### 不止损 (25%)

#### 典型案例
```
案例 2:
  股票：XXX
  买入：20 元
  止损位：19 元 (-5%)
  实际卖出：15 元
  亏损：-25%
  
原因:
  - 止损犹豫
  - 抱幻想
  - 想等反弹
  
教训:
  - 触及止损立即执行
  - 不抱幻想
  - -5% 是底线
```

#### 共同特征
```
🔴 止损位明确但不执行
🔴 抱幻想等反弹
🔴 亏损扩大
🔴 心态崩溃

避免方法:
  1. 设置自动止损
  2. 无条件执行
  3. 接受亏损是成本
```

---

### 抄底接飞刀 (20%)

#### 典型案例
```
案例 3:
  股票：XXX
  买入：连续跌停后抄底
  价格：20 元
  卖出：12 元
  亏损：-40%
  
原因:
  - 连续跌停后抄底
  - 以为是底部
  - 实际是下跌中继
  
教训:
  - 不抄底跌停股
  - 等企稳信号
  - 不接飞刀
```

#### 共同特征
```
🔴 连续跌停
🔴 以为是底部
🔴 实际下跌中继
🔴 不止损

避免方法:
  1. 不抄底跌停股
  2. 等企稳信号
  3. 右侧交易
```

---

### 贪婪不止盈 (15%)

#### 典型案例
```
案例 4:
  股票：XXX
  买入：10 元
  最高：20 元 (+100%)
  卖出：12 元
  最终盈利：+20%
  
原因:
  - 盈利 100% 不止盈
  - 想赚更多
  - 坐过山车
  
教训:
  - 分批止盈
  - 不板即卖
  - 落袋为安
```

#### 共同特征
```
🔴 盈利丰厚不止盈
🔴 想赚更多
🔴 坐过山车
🔴 最终盈利少

避免方法:
  1. 分批止盈
  2. 不板即卖
  3. 落袋为安
```
