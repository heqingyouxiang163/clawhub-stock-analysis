## 1️⃣ 龙虎榜数据核心

### 关键指标
```
1. 买入金额前 5 席位
2. 卖出金额前 5 席位
3. 净买入额
4. 机构席位数量
5. 游资席位识别
```

### 数据解读
```
买入金额>卖出金额：主力净流入
机构席位多：机构看好
游资席位多：短线机会
买卖均衡：分歧大
```

## 3️⃣ 机构席位动向

### 机构买入信号
```
✅ 机构专用席位净买入
✅ 多机构同时买入
✅ 机构买入>游资买入
✅ 连续多日机构买入

含义:
  - 价值投资
  - 中长期看好
  - 基本面支撑
```

### 机构卖出信号
```
🔴 机构专用席位净卖出
🔴 多机构同时卖出
🔴 机构卖出>游资买入
🔴 连续多日机构卖出

含义:
  - 价值看空
  - 基本面转差
  - 坚决卖出
```

## 4️⃣ 席位联动分析

### 多席位共振
```
现象：多个知名席位同时买入
含义:
  - 一致看好
  - 大行情
  - 连板预期

操作:
  - 积极跟进
  - 仓位 40-50%
  - 目标连板
```

### 机构 + 游资共振
```
现象：机构席位 + 游资席位同时买入
含义:
  - 价值 + 情绪共振
  - 最强买入信号
  - 持续性好

操作:
  - 重仓跟进
  - 仓位 50-70%
  - 目标 30%+
```
