## 1️⃣ 买卖五档解读

### 买盘信号

#### 买一托单
```
现象：买一挂大单 (1000 手+)
含义:
  - 主力护盘
  - 支撑位确认
  - 短期不跌

操作:
  - 持有观望
  - 不破买一可加仓
  - 破买一止损
```

#### 买盘递增
```
现象：买一<买二<买三
含义:
  - 主力吸筹
  - 买盘积极
  - 上涨预期

操作:
  - 积极跟进
  - 仓位 30-50%
```

#### 买盘集中
```
现象：买一集中大单，买二 - 买五稀疏
含义:
  - 主力控盘
  - 短期方向明确
  - 可能拉升

操作:
  - 持有待涨
  - 破买一减仓
```

### 卖盘信号

#### 卖一压单
```
现象：卖一挂大单 (1000 手+)
含义:
  - 主力压盘
  - 压力位确认
  - 短期不涨

操作:
  - 减仓观望
  - 突破卖一加仓
  - 不突破减仓
```

#### 卖盘递增
```
现象：卖一<卖二<卖三
含义:
  - 抛压沉重
  - 主力出货
  - 下跌预期

操作:
  - 坚决卖出
  - 空仓观望
```

#### 卖盘集中
```
现象：卖一集中大单，卖二 - 卖五稀疏
含义:
  - 主力试盘
  - 压力测试
  - 可能突破

操作:
  - 观察突破
  - 突破加仓
  - 不突破减仓
```

## 2️⃣ 大单识别

### 大单买入信号

#### 连续性大单
```
现象：连续多笔大单买入 (100 手+)
含义:
  - 主力建仓
  - 资金流入
  - 上涨信号

操作:
  - 积极跟进
  - 仓位 40-50%
```

#### 间歇性大单
```
现象：间隔出现大单买入
含义:
  - 主力控盘
  - 缓慢吸筹
  - 中长期看好

操作:
  - 逢低吸纳
  - 仓位 30-40%
```

#### 尾盘大单
```
现象：14:30 后出现大单买入
含义:
  - 主力抢筹
  - 次日预期好
  - 可能高开

操作:
  - 持有过夜
  - 次日高开止盈
```

### 大单卖出信号

#### 连续性大单卖出
```
现象：连续多笔大单卖出
含义:
  - 主力出货
  - 资金流出
  - 下跌信号

操作:
  - 坚决卖出
  - 空仓观望
```

#### 大单对倒
```
现象：大单买入同时大单卖出
含义:
  - 主力对倒
  - 制造成交量
  - 诱多/诱空

操作:
  - 谨慎观望
  - 不轻易跟进
```

## 3️⃣ 分时图盘口

### 强势盘口

#### 阶梯式上涨
```
形态：价格阶梯式上涨，每阶站稳
含义:
  - 主力控盘良好
  - 资金持续流入
  - 涨停预期

操作:
  - 持有待涨停
  - 破阶梯减仓
```

#### 45 度角上涨
```
形态：分时线呈 45 度角上涨
含义:
  - 主力稳步拉升
  - 不急于封板
  - 可能连板

操作:
  - 持有
  - 破均线减仓
```

#### 尾盘抢筹
```
形态：14:30 后快速拉升
含义:
  - 主力抢筹
  - 次日高开预期
  - 可能有利好

操作:
  - 持有过夜
  - 次日高开止盈
```

### 弱势盘口

#### 阶梯式下跌
```
形态：价格阶梯式下跌，每阶受阻
含义:
  - 主力出货
  - 资金持续流出
  - 下跌预期

操作:
  - 坚决卖出
  - 不抄底
```

#### 高位横盘
```
形态：高位横盘不涨
含义:
  - 主力出货
  - 诱多嫌疑
  - 可能跳水

操作:
  - 减仓观望
  - 破位清仓
```

#### 尾盘跳水
```
形态：14:30 后快速下跌
含义:
  - 主力出货
  - 次日低开预期
  - 可能有利空

操作:
  - 坚决卖出
  - 不过夜
```

## 5️⃣ 主力操盘手法识别

### 吸筹手法

#### 压盘吸筹
```
手法：卖一压大单，下方小单吸筹
识别:
  - 卖一大单不成交
  - 下方持续小单买入
  - 价格不跌

操作:
  - 逢低吸纳
  - 主力成本附近买入
```

#### 震荡吸筹
```
手法：区间震荡，高抛低吸
识别:
  - 价格区间明确
  - 成交量温和
  - 时间较长 (5-10 天)

操作:
  - 区间下沿买入
  - 区间上沿卖出
```

### 出货手法

#### 拉高出货
```
手法：快速拉升，吸引跟风，然后出货
识别:
  - 快速拉升 5-10%
  - 成交量放大
  - 高位横盘

操作:
  - 不追高
  - 高位坚决卖出
```

#### 对倒出货
```
手法：大单对倒，制造成交量
识别:
  - 大单买入同时大单卖出
  - 成交量异常
  - 价格不涨

操作:
  - 坚决卖出
  - 不跟进
```
