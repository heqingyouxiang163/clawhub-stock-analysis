## 1️⃣ 假突破常见形态

### 突破平台假突破

#### 真突破特征
```
✅ 突破时放量 (>2 倍)
✅ 突破后站稳 3 天
✅ 突破后继续上涨
✅ 资金持续流入
✅ 板块配合

成功率：70%
```

#### 假突破特征
```
🔴 突破时缩量
🔴 突破后立刻回落
🔴 无法站稳平台
🔴 资金流出
🔴 板块不配合

失败率：80%
```

#### 识别要点
```
1. 看成交量：放量真，缩量假
2. 看站稳时间：3 天真，1 天假
3. 看资金：流入真，流出假
4. 看板块：配合真，不配合假
```

## 2️⃣ 假突破识别技巧

### 成交量识别

```
真突破:
  - 突破时成交量>前日 150%
  - 突破后成交量温和
  - 量价配合完美

假突破:
  - 突破时成交量<前日 100%
  - 突破后成交量异常
  - 量价背离
```

### 时间识别

```
真突破:
  - 突破后站稳 3 天以上
  - 第 4 天继续上涨
  - 趋势确立

假突破:
  - 突破后 1-2 天回落
  - 无法站稳
  - 重回震荡
```

### 资金识别

```
真突破:
  - 超大单净流入>3000 万
  - 大单净流入>1000 万
  - 资金持续流入

假突破:
  - 超大单净流出
  - 大单净卖出
  - 资金流出
```

### 盘口识别

```
真突破:
  - 买盘积极
  - 封单金额大
  - 卖盘压力小

假突破:
  - 买盘虚假
  - 封单快速减少
  - 卖盘压力大
```

## 3️⃣ 假突破陷阱

### 诱多陷阱

```
手法:
  1. 快速突破关键位
  2. 吸引跟风盘
  3. 立刻回落
  4. 套牢追高资金

特征:
  - 突破时无量
  - 突破后快速回落
  - 长上影线
  - 大单卖出

操作：不追高，等确认
```

### 对倒陷阱

```
手法:
  1. 大单对倒制造成交量
  2. 突破关键位
  3. 吸引技术派跟进
  4. 出货

特征:
  - 成交量异常
  - 大单买入同时大单卖出
  - 价格不涨
  - 资金净流出

操作：不跟进，观望
```

## 4️⃣ 真突破确认

### 确认条件

```
条件 1: 成交量确认
  - 突破时放量>150%
  - 突破后温和放量

条件 2: 时间确认
  - 站稳 3 天以上
  - 第 4 天继续上涨

条件 3: 资金确认
  - 超大单净流入
  - 资金持续流入

条件 4: 板块确认
  - 板块集体上涨
  - 多股涨停

条件 5: 趋势确认
  - 均线多头排列
  - 趋势向上
```

### 确认策略

```
保守策略:
  - 等 3 天站稳后跟进
  - 成功率：80%
  - 错过部分涨幅

激进策略:
  - 突破当日跟进
  - 成功率：60%
  - 抓住全部涨幅

平衡策略:
  - 突破日买入 30%
  - 站稳 3 天加仓 30%
  - 成功率：70%
```

## 5️⃣ 避险策略

### 不追高原则
```
1. 突破日不追高 (>5%)
2. 等回踩确认
3. 缩量突破不跟进
4. 板块不配合不跟进
```

### 止损策略
```
1. 突破失败立刻止损
2. 跌破突破位止损
3. -5% 坚决止损
4. 不抱幻想
```

### 仓位管理
```
突破日：20-30%
确认后：40-50%
失败：0%
```
