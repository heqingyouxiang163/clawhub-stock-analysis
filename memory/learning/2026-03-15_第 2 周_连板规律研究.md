## 1️⃣ 连板股统计规律

### 连板高度分布

```
统计周期：近 1 年 A 股连板股
样本数量：500+ 只连板股

连板数分布:
  2 连板：45% (最常见)
  3 连板：25%
  4 连板：15%
  5 连板：8%
  6 连板：4%
  7 连板以上：3% (妖股)

结论:
  - 2-3 连板是主流 (70%)
  - 5 连板是分水岭
  - 7 连板以上是妖股
```

### 连板成功率

```
首板→二板：60%
二板→三板：50%
三板→四板：40%
四板→五板：30%
五板→六板：20%
六板→七板：10%

结论:
  - 连板高度越高，成功率越低
  - 3 板是重要关口
  - 5 板以上风险极大
```

## 3️⃣ 连板周期规律

### 连板时间规律

```
首板时间:
  - 9:30-10:00:45% (早盘板)
  - 10:00-11:30:30%
  - 13:00-14:00:15%
  - 14:00-15:00:10% (尾盘板)

结论:
  - 早盘板质量最高
  - 尾盘板质量最差
  - 10 点前涨停易连板
```

### 连板日规律

```
周一连板：25%
周二连板：22%
周三连板：20%
周四连板：18%
周五连板：15%

结论:
  - 周初连板概率高
  - 周末前连板概率低
  - 资金周末避险
```

## 4️⃣ 连板中断信号

### 常见中断信号

#### 集合竞价
```
🔴 低开 -5% 以下
🔴 竞价量异常放大 (>30%)
🔴 封单大幅减少

操作：坚决卖出
```

#### 开盘后
```
🔴 快速跌破开盘价
🔴 无法翻红
🔴 成交量异常

操作：反抽卖出
```

#### 盘中
```
🔴 炸板不回封
🔴 封单快速减少
🔴 板块集体退潮

操作：清仓
```
