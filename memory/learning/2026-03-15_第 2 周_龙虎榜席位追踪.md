## 2️⃣ 机构席位识别

### 机构席位特征
```
席位名称：机构专用
特点:
  - 价值投资
  - 持仓周期长
  - 资金量大
  - 偏好白马股

操作风格:
  - 底部建仓
  - 分批买入
  - 长期持有
  - 基本面驱动

跟随策略:
  - 机构净买入跟进
  - 中长期持有
  - 目标 30-50%
```

### 机构 vs 游资博弈
```
情景 1: 机构买入 + 游资买入
  含义：共振看好
  操作：积极跟进
  目标：30%+

情景 2: 机构买入 + 游资卖出
  含义：分歧
  操作：观望
  等待：方向明确

情景 3: 机构卖出 + 游资买入
  含义：游资接盘
  操作：谨慎
  止损：严格

情景 4: 机构卖出 + 游资卖出
  含义：一致看空
  操作：坚决卖出
  空仓：观望
```
