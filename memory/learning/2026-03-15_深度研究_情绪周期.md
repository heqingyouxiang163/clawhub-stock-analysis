## 1️⃣ 情绪周期四阶段

### 冰点期

#### 特征
```
涨停家数：<20 家
跌停家数：>30 家
连板高度：≤2 板
炸板率：>40%
涨跌比：<0.5

市场情绪：极度悲观
```

#### 操作策略
```
仓位：0-20%
策略：试错首板
选股:
  - 率先涨停股
  - 独立逻辑股
  - 超跌反弹股

止损：-5% 坚决止损
止盈：10-15% 分批止盈
```

#### 识别信号
```
✅ 昨日涨停今日集体低开
✅ 跌停家数激增
✅ 连板股全部断板
✅ 市场无主线
✅ 成交量萎缩
```

## 2️⃣ 情绪周期量化

### 情绪得分计算

```python
情绪得分 = 
  涨停家数得分 × 30% +
  跌停家数得分 × 25% +
  连板高度得分 × 20% +
  炸板率得分 × 15% +
  涨跌比得分 × 10%
```

### 评分标准

#### 涨停家数得分 (30 分)
```
>100 家：30 分 (高潮)
80-100 家：25 分
50-80 家：20 分 (强势)
20-50 家：15 分 (启动)
<20 家：5 分 (冰点)
```

#### 跌停家数得分 (25 分)
```
<5 家：25 分 (好)
5-10 家：20 分
10-30 家：10 分 (差)
>30 家：0 分 (冰点)
```

#### 连板高度得分 (20 分)
```
≥7 板：20 分 (高潮，警惕)
5-6 板：18 分
3-4 板：15 分 (启动)
2 板：10 分
1 板：5 分 (冰点)
```

#### 炸板率得分 (15 分)
```
<20%: 15 分 (好)
20-30%: 12 分
30-40%: 8 分 (差)
>40%: 0 分 (冰点)
```

#### 涨跌比得分 (10 分)
```
>3: 10 分 (高潮)
2-3: 8 分
1-2: 5 分 (启动)
0.5-1: 3 分
<0.5: 0 分 (冰点)
```
