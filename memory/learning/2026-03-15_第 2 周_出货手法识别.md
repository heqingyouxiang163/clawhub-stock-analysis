## 1️⃣ 常见出货手法

### 拉高出货

#### 手法特征
```
步骤:
  1. 快速拉升 5-10%
  2. 吸引跟风盘
  3. 高位横盘
  4. 分批出货
  5. 尾盘跳水

盘口特征:
  - 大单拉升，小单出货
  - 卖盘持续增加
  - 买盘虚假托单
  - 成交量异常放大

K 线特征:
  - 长上影线
  - 高位十字星
  - 乌云盖顶
  - 高位阴线
```

#### 识别信号
```
🔴 快速拉升后横盘不涨
🔴 成交量放大但价格不涨
🔴 大单买入小单卖出
🔴 卖盘持续增加
🔴 尾盘跳水

操作：坚决卖出
```

---

### 震荡出货

#### 手法特征
```
步骤:
  1. 高位震荡
  2. 高抛低吸
  3. 逐步出货
  4. 跌破支撑

盘口特征:
  - 区间内大单对倒
  - 上方压单，下方托单
  - 成交量温和放大
  - 价格区间明确

K 线特征:
  - 高位箱体震荡
  - 阴阳交错
  - 成交量放大
  - 最终跌破箱体
```

#### 识别信号
```
🔴 高位横盘超过 5 天
🔴 成交量放大但价格不涨
🔴 区间内大单对倒
🔴 跌破箱体下沿

操作：破位坚决卖出
```

---

### 跌停板出货

#### 手法特征
```
步骤:
  1. 直接跌停开盘
  2. 吸引抄底资金
  3. 开板出货
  4. 继续跌停

盘口特征:
  - 跌停板大单
  - 开板时巨量成交
  - 开板后快速回封
  - 反复开板

K 线特征:
  - 一字跌停
  - T 字跌停
  - 巨量阴线
```

#### 识别信号
```
🔴 跌停板开板
🔴 巨量成交
🔴 快速回封
🔴 反复开板

操作：不抄底，坚决卖出
```

---

### 利好出货

#### 手法特征
```
步骤:
  1. 发布利好消息
  2. 高开吸引跟风
  3. 高开低走
  4. 出货完成

盘口特征:
  - 高开 5%+
  - 竞价量大
  - 开盘后快速下跌
  - 大单卖出

K 线特征:
  - 高开低走大阴线
  - 乌云盖顶
  - 吞没形态
```

#### 识别信号
```
🔴 利好消息公布
🔴 高开 5%+
🔴 高开低走
🔴 成交量放大
🔴 大单卖出

操作：高开坚决卖出
```

## 3️⃣ 出货 vs 洗盘

### 区别要点

| 特征 | 出货 | 洗盘 |
|------|------|------|
| **位置** | 高位 | 中低位 |
| **成交量** | 放大 | 缩量 |
| **价格** | 不涨 | 横盘 |
| **时间** | 短 (3-5 天) | 长 (5-10 天) |
| **支撑** | 跌破 | 守住 |
| **资金** | 流出 | 流入 |
| **后续** | 下跌 | 上涨 |

### 判断技巧
```
1. 看位置：高位出货，低位洗盘
2. 看成交量：放量出货，缩量洗盘
3. 看支撑：跌破出货，守住洗盘
4. 看资金：流出出货，流入洗盘
5. 看时间：短出货，长洗盘
```

### 止盈策略
