## 1️⃣ 涨停基因特征

### 基本面基因

#### 小市值
```
特征：流通市值 50-200 亿
原因:
  - 便于拉升
  - 资金需求小
  - 弹性大

统计:
  - 70% 涨停股流通市值<200 亿
  - 成功率：65%
```

#### 低股价
```
特征：股价 5-50 元
原因:
  - 散户友好
  - 便于接力
  - 心理门槛低

统计:
  - 80% 涨停股股价<50 元
  - 成功率：60%
```

#### 业绩预增
```
特征：业绩增长>50%
原因:
  - 基本面支撑
  - 资金认可
  - 持续性好

统计:
  - 40% 涨停股业绩预增
  - 成功率：70%
```

### 技术面基因

#### 均线多头
```
特征：MA5>MA10>MA20
原因:
  - 趋势向上
  - 阻力小
  - 资金认可

统计:
  - 85% 涨停股均线多头
  - 成功率：75%
```

#### 突破形态
```
特征：突破平台/前高
原因:
  - 压力释放
  - 空间打开
  - 资金跟进

统计:
  - 60% 涨停股突破形态
  - 成功率：70%
```

#### 缩量回调
```
特征：回调缩量<0.8 倍
原因:
  - 主力未出逃
  - 洗盘整理
  - 再次拉升

统计:
  - 50% 涨停股缩量回调后涨停
  - 成功率：70%
```

  - 成功率：65%
```

### 题材基因

```
特征：属于当前热点
原因:

  - 板块联动
  - 持续性好

统计:

  - 成功率：70%
```

#### 政策利好
```
