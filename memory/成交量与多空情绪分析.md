## 模块 1: 成交量深度分析

### 成交量的本质

**定义**: 成交量 = 买卖双方达成交易的数量
**核心逻辑**: 
- 价格上涨 + 成交量放大 = 多头真金白银买入 (健康)
- 价格上涨 + 成交量缩小 = 多头力量衰竭 (危险)
- 价格下跌 + 成交量放大 = 空头恐慌出逃 (危险)
- 价格下跌 + 成交量缩小 = 空头力量衰竭 (可能反弹)

### 成交量分析框架

#### 1. 量比 (Volume Ratio)

**公式**: 
```
量比 = 当日成交量 / 过去 5 日平均成交量
```

**解读**:
| 量比 | 含义 | 操作 |
|------|------|------|
| < 0.5 | 极度缩量 | 观望，可能变盘 |
| 0.5-0.8 | 缩量 | 趋势可能延续 |
| 0.8-1.5 | 正常 | 无特殊信号 |
| 1.5-2.5 | 温和放量 | 趋势加强 |
| 2.5-5 | 明显放量 | 重点关注 |
| > 5 | 巨量 | 警惕变盘 |

**实战应用**:
```
上涨 + 量比 2-3 = 健康上涨，持有
上涨 + 量比 > 5 = 警惕，可能见顶
下跌 + 量比 2-3 = 恐慌出逃，止损
下跌 + 量比 < 0.5 = 缩量下跌，可能反弹
```

#### 2. 成交量均线

**常用周期**:
- VOL_MA5: 短期资金动向
- VOL_MA20: 中期资金动向
- VOL_MA60: 长期资金动向

**信号**:
- 成交量上穿 VOL_MA20 = 资金开始活跃
- 成交量下穿 VOL_MA20 = 资金开始撤离
- VOL_MA5 > VOL_MA20 = 短期资金活跃

#### 3. 量价关系 8 种形态

| 形态 | 价格 | 成交量 | 含义 | 操作 |
|------|------|--------|------|------|
| 量增价涨 | ↑ | ↑↑ | 健康上涨 | 持有/加仓 |
| 量缩价涨 | ↑ | ↓ | 上涨乏力 | 减仓/观望 |
| 量平价涨 | ↑ | → | 温和上涨 | 持有 |
| 量增价跌 | ↓ | ↑↑ | 恐慌出逃 | 止损 |
| 量缩价跌 | ↓ | ↓ | 下跌放缓 | 观望/抄底 |
| 量平价跌 | ↓ | → | 阴跌不止 | 观望 |
| 量增价平 | → | ↑↑ | 主力吸筹/出货 | 观察方向 |
| 量缩价平 | → | ↓ | 盘整等待 | 观望 |

#### 4. 天量与地量

**天量 (巨量)**:
- 定义：成交量创近期新高 (如 60 日最高)
- 位置决定含义:
  - **低位天量**: 主力建仓，看涨
  - **高位天量**: 主力出货，看跌
  - **上涨中途天量**: 加速赶顶，警惕

**地量 (极度缩量)**:
- 定义：成交量创近期新低 (如 60 日最低)
- 含义：
  - **下跌后地量**: 抛压耗尽，可能反弹
  - **上涨后地量**: 买盘不足，可能回调
  - **盘整期地量**: 等待方向选择

### 成交量实战案例

#### 案例 1: 涨停板成交量分析

**健康涨停**:
- 量比：1.5-3 倍
- 封单：大单封死，不开板
- 含义：资金一致看好，次日大概率高开

**问题涨停**:
- 量比：> 5 倍 (放量过大)
- 封单：反复开板
- 含义：分歧大，次日可能低开

#### 案例 2: 突破平台成交量

**真突破**:
- 突破日成交量 > 2 倍均量
- 收盘价站稳平台上方
- 次日继续放量
- 操作：跟进

**假突破**:
- 突破日成交量不足
- 次日立刻回落
- 操作：止损

#### 案例 3: 回调缩量买入法

**条件**:
1. 上升趋势中 (MA20 向上)
2. 回调到支撑位 (MA20/MA60)
3. 成交量明显缩量 (< 0.8 倍均量)
4. K 线出现止跌信号 (十字星/小阳线)

**操作**: 买入，止损设在支撑位下方

## 模块 2: 多空情绪量化

### 🎯 核心指标：大单/超大单资金流向

**为什么最重要？**
- **超大单** = 机构/主力真金白银 (最可靠信号)
- **大单** = 大户/游资动向 (短线风向标)
- **中单** = 中户跟随
- **小单** = 散户情绪 (反向指标)

**资金流向分级**:
| 订单类型 | 金额范围 | 代表群体 | 信号可靠性 |
|----------|----------|----------|------------|
| **超大单** | > 100 万元 | 机构/主力 | ⭐⭐⭐⭐⭐ |
| **大单** | 20-100 万元 | 大户/游资 | ⭐⭐⭐⭐ |
| **中单** | 4-20 万元 | 中户 | ⭐⭐⭐ |
| **小单** | < 4 万元 | 散户 | ⭐⭐ (反向) |

**核心逻辑**:
```
超大单净流入 + 大单净流入 = 主力建仓 → 看涨 🟢
超大单净流出 + 大单净流出 = 主力出货 → 看跌 🔴
超大单流入 + 大单流出 = 主力分歧 → 观望 🟡
```

**实战应用**:
| 情景 | 超大单 | 大单 | 中单 | 小单 | 含义 | 操作 |
|------|--------|------|------|------|------|------|
| 主力建仓 | ++ | + | - | -- | 主力吸筹 | 跟进 🟢 |
| 主力出货 | -- | - | + | ++ | 主力出逃 | 止损 🔴 |
| 主力洗盘 | - | - | + | ++ | 洗散户 | 持有 🟡 |
| 散户接盘 | -- | -- | + | ++ | 高位派发 | 清仓 🔴 |

**量化评分权重**:
- 超大单净流入：+20 分 (最高权重)
- 大单净流入：+15 分
- 大单 - 小单差值：+10 分 (主力 vs 散户)

### 情绪指标体系

#### 1. 涨跌比指标

**公式**:
```
涨跌比 = 上涨家数 / 下跌家数
```

**解读**:
| 涨跌比 | 市场情绪 | 操作 |
|--------|----------|------|
| > 3 | 极度乐观 | 警惕回调 |
| 2-3 | 乐观 | 持有 |
| 1-2 | 中性 | 观望 |
| 0.5-1 | 悲观 | 谨慎 |
| < 0.5 | 极度悲观 | 可能反弹 |

#### 2. 涨停板情绪

**指标**:
- 涨停家数：> 50 家 = 情绪高涨
- 跌停家数：> 20 家 = 情绪恐慌
- 连板高度：最高连板数 (情绪风向标)
- 炸板率：涨停后开板比例 (> 40% = 情绪差)

**解读**:
```
涨停 80 家 + 跌停 5 家 + 连板 7 层 = 情绪高潮，警惕
涨停 20 家 + 跌停 30 家 + 连板 2 层 = 情绪冰点，可能反弹
```

#### 3. 龙虎榜情绪

**机构动向**:
- 机构净买入 > 5 亿 = 机构看好
- 机构净卖出 > 5 亿 = 机构看空

**游资动向**:
- 知名游资集中买入 = 短线情绪好
- 游资集体出货 = 短线情绪差

#### 4. 北向资金情绪

**公式**:
```
北向资金情绪 = 当日净流入 / 总成交额
```

**解读**:
| 净流入 | 情绪 | 含义 |
|--------|------|------|
| > 50 亿 | 强烈看好 | 外资大举买入 |
| 20-50 亿 | 看好 | 外资持续流入 |
| -20-20 亿 | 中性 | 外资观望 |
| -50-(-20) 亿 | 看空 | 外资流出 |
| < -50 亿 | 强烈看空 | 外资大举出逃 |

### 个股多空情绪分析

#### 1. 资金流向指标

**主力净流入**:
```
主力净流入 = 大单买入 - 大单卖出
```

**解读**:
- 主力净流入 > 0 且价格上涨 = 健康
- 主力净流入 > 0 但价格下跌 = 主力吸筹
- 主力净流出 < 0 且价格上涨 = 主力出货 (警惕)
- 主力净流出 < 0 且价格下跌 = 主力出逃

#### 2. 买卖盘口分析

**买一 - 买五 vs 卖一 - 卖五**:
- 买盘总量 > 卖盘总量 = 多头强势
- 卖盘总量 > 买盘总量 = 空头强势

**大单挂单**:
- 买盘有大单托底 = 主力护盘
- 卖盘有大单压顶 = 主力压价

#### 3. 情绪周期理论

**四个阶段**:

```
1. 冰点期
   - 特征：涨停少，跌停多，连板高度低
   - 操作：观望或轻仓试错

2. 启动期
   - 特征：涨停增加，出现新题材
   - 操作：积极参与龙头

3. 高潮期
   - 特征：百股涨停，连板高度高
   - 操作：持仓或减仓，不追高

4. 退潮期
   - 特征：涨停减少，跌停增加，高位股补跌
   - 操作：空仓或轻仓
```

### 综合情绪评分模型

**公式**:
```
情绪得分 = 
  涨跌比得分 × 20% +
  涨停情绪得分 × 25% +
  北向资金得分 × 20% +
  主力资金得分 × 20% +
  大盘趋势得分 × 15%
```

**评分标准**:
| 得分 | 情绪状态 | 仓位建议 |
|------|----------|----------|
| 80-100 | 极度乐观 | 50-70% (警惕回调) |
| 60-80 | 乐观 | 70-90% |
| 40-60 | 中性 | 50-70% |
| 20-40 | 悲观 | 20-40% |
| 0-20 | 极度悲观 | 0-20% (可能反弹) |

## 模块 3: 成交量 + 情绪 综合应用

### 次日卖出决策模型 (升级版)

**输入数据** (T 日收盘后):
1. 当日成交量 vs 均量 (量比)
2. 当日价格涨跌幅
3. 主力资金流向
4. 北向资金流向
5. 市场情绪得分
6. 个股情绪得分

**决策逻辑**:

```
IF 量比 > 3 AND 价格下跌 AND 主力大幅流出 THEN
  次日坚决卖出 (资金出逃)

ELSE IF 量比 < 0.5 AND 价格下跌 THEN
  次日观望 (可能反弹)

ELSE IF 量比 1.5-3 AND 价格上涨 AND 主力流入 THEN
  次日持有 (健康上涨)

ELSE IF 量比 > 5 AND 价格上涨 THEN
  次日减仓 (警惕见顶)

ELSE
  次日根据开盘情况决定
```

### 实战案例分析

#### 案例：美诺华 (603538) 涨停日分析

**T 日数据**:
- 价格：+10.02% (涨停)
- 成交量：22.7 万手
- 量比：约 2.5 倍 (温和放量)
- 形态：突破平台

**成交量解读**:
- ✅ 量比 2.5 倍 = 健康放量，不是巨量
- ✅ 涨停封死 = 资金一致看好
- ✅ 突破平台 = 技术面好

**情绪分析**:
- 需要查看：主力资金流向、北向资金、龙虎榜

**次日预判**:
- 如果情绪好 (大盘涨 + 板块强) → 大概率高开
- 如果情绪差 (大盘跌 + 板块弱) → 可能低开

**操作策略**:
```
高开 5%+ → 持有，冲高止盈
平开 → 观察 30 分钟
低开 → 反抽卖出
```

#### 案例：百川股份 (002455) 回调日分析

**T 日数据**:
- 价格：-5.32% (回调)
- 成交量：放量 (获利盘出逃)
- 量比：约 2-3 倍
- 形态：连续大涨后回调

**成交量解读**:
- ⚠️ 下跌 + 放量 = 资金出逃
- ⚠️ 连续大涨后回调 = 获利回吐
- ❓ 需要看是否跌破关键支撑

**情绪分析**:
- 连续大涨后情绪可能转弱
- 需要观察次日是否企稳

**次日预判**:
```
IF 收盘价 < 14.73 (前涨停价) THEN 趋势转弱，止损
ELSE IF 缩量企稳 THEN 可能反弹，持有
ELSE 观望
```

## 模块 4: 实战工具与数据源

### 成交量分析工具

**AKShare 数据**:
```python
# 获取历史行情 (含成交量)
df = ak.stock_zh_a_hist(symbol="002455", period="daily")

# 计算量比
df['量比'] = df['成交量'] / df['成交量'].rolling(5).mean()

# 计算成交量均线
df['VOL_MA5'] = df['成交量'].rolling(5).mean()
df['VOL_MA20'] = df['成交量'].rolling(20).mean()
```

**关键指标**:
- 量比
- VOL_MA5 / VOL_MA20 比值
- 成交量创 N 日新高/新低

### 情绪分析数据源

**市场情绪**:
- 涨跌家数比：东方财富/同花顺
- 涨停/跌停家数：每日统计
- 连板高度：短线情绪指标

**资金流向**:
- 北向资金：港交所/东方财富
- 主力资金：AKShare/同花顺
- 龙虎榜：交易所官网

**情绪指标**:
- 恐慌贪婪指数 (如果有)
- VIX 中国 (如果有)
- 融资融券余额变化

## 模块 5: 成交量 + 情绪 量化评分卡

### 个股评分卡 (满分 100)

**成交量维度 (40 分)**:
| 指标 | 评分标准 | 得分 |
|------|----------|------|
| 量比 | 1.5-3 倍 = 10 分，>5 倍或<0.5 倍 = 0 分 | /10 |
| 量价关系 | 量增价涨 = 10 分，量缩价涨 = 5 分，量增价跌 = 0 分 | /10 |
| 成交量趋势 | 连续放量 = 10 分，连续缩量 = 5 分 | /10 |
| 异常成交量 | 无异常 = 10 分，天量/地量 = 0 分 | /10 |

**情绪维度 (60 分)**:
| 指标 | 评分标准 | 得分 |
|------|----------|------|
| 主力资金 | 大幅净流入 = 15 分，净流出 = 0 分 | /15 |
| 北向资金 | 净流入 = 15 分，净流出 = 0 分 | /15 |
| 市场情绪 | 涨跌比>2 = 10 分，<0.5 = 0 分 | /10 |
| 板块情绪 | 板块强势 = 10 分，板块弱势 = 0 分 | /10 |
| 龙虎榜 | 机构买入 = 10 分，机构卖出 = 0 分 | /10 |

**总分解读**:
- 80-100: 强烈看好，持有/加仓
- 60-80: 看好，持有
- 40-60: 中性，观望
- 20-40: 看空，减仓
- 0-20: 强烈看空，清仓

## 核心收获

1. **成交量不会骗人** — 价格可以操纵，成交量很难
2. **量价配合是关键** — 上涨必须放量，下跌最好缩量
3. **情绪是短期驱动力** — 短线就是炒情绪
4. **资金流向是核心** — 跟着主力走，别跟主力对着干
5. **综合评分更可靠** — 单一指标容易误判，多维度综合
