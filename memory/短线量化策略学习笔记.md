## 模块 1: 短线量化核心逻辑

### 短线交易的本质

**定义**: 持仓 1-5 个交易日，赚的是**情绪钱**和**动量钱**

**核心逻辑**:
```
短线盈利 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损
```

**关键**: 不追求高胜率，追求**高盈亏比**

### 短线 vs 中长线

| 维度 | 短线 | 中长线 |
|------|------|--------|
| 持仓时间 | 1-5 天 | 数周 - 数月 |
| 决策依据 | 技术面 + 情绪 | 基本面 + 趋势 |
| 止损幅度 | 3-8% | 10-20% |
| 止盈幅度 | 5-15% | 20-50%+ |
| 胜率要求 | 40%+ 即可 | 50%+ |
| 盈亏比要求 | >1.5 | >2 |

## 模块 2: 次日卖出决策系统 (核心!)

### 决策框架：T+1 卖出判断

**买入日 (T 日) 晚上必须准备好的数据**:
1. 当日 K 线形态 (阳线/阴线/十字星)
2. 成交量变化 (放量/缩量)
3. 收盘价位置 (是否突破/跌破关键位)
4. 当日涨跌幅
5. 龙虎榜数据 (如有)
6. 晚间消息面

**次日 (T+1) 早盘观察要点**:
1. 集合竞价情况 (高开/低开/平开)
2. 竞价成交量
3. 开盘后 5 分钟走势
4. 板块联动情况

### 次日卖出决策树

```
T+1 日开盘前判断:

┌─ 高开 (>2%)
│   ├─ 快速拉升 (>5%) → 持有，设追踪止损
│   ├─ 冲高回落 → 减仓 50%
│   └─ 直接低开 → 警惕，准备止损
│
├─ 平开 (±1%)
│   ├─ 站稳昨日收盘价 → 持有观察
│   └─ 跌破昨日最低价 → 止损卖出
│
└─ 低开 (<-2%)
    ├─ 快速反弹翻红 → 持有
    ├─ 震荡走低 → 减仓或止损
    └─ 直接跌停 → 排队卖出
```

### 量化卖出信号 (满足任一即卖出)

**🔴 止损信号 (必须执行)**:
1. 收盘价跌破买入日最低价 → 趋势转弱
2. 亏损达到 -8% → 纪律止损
3. MACD 死叉 + 价格跌破 MA5 → 技术破位
4. 放量下跌 (>2 倍均量) → 资金出逃

**🟢 止盈信号 (部分或全部)**:
1. 盈利达到 +15% → 至少减半
2. 连续 2 日涨停 → 第三日高开则止盈
3. RSI > 80 → 超买，减仓
4. 突破后回踩确认失败 → 获利了结

**🟡 观望信号 (继续持有)**:
1. 价格在 MA5 上方 → 趋势未破
2. 成交量温和 → 没有异常
3. 盈亏比仍合理 → 值得博弈

## 模块 3: 常见短线形态及次日策略

### 形态 1: 涨停板次日

**T 日**: 涨停 (+10%)

**T+1 策略**:
| 开盘情况 | 操作 | 概率 |
|----------|------|------|
| 高开 5%+ | 持有，冲高止盈 | 连板概率 30% |
| 高开 2-5% | 观察，破昨日涨停价止损 | 震荡概率 50% |
| 平开 | 减仓 50%，等方向 | 回调概率 40% |
| 低开 | 反抽卖出 | 走弱概率 60% |

**关键价位**:
- 支撑：昨日涨停价
- 压力：前高

### 形态 2: 大阳线次日 (>5%)

**T 日**: 大涨 5-9%

**T+1 策略**:
- 高开：持有，看能否涨停
- 平开：观察 30 分钟，不破昨日一半持有
- 低开：反抽不过昨日收盘价卖出

**止损位**: 昨日阳线的 1/2 位置

### 形态 3: 突破形态次日

**T 日**: 突破平台/前高

**T+1 策略**:
- 成功突破：持有，追踪止损
- 假突破 (回落)：立即卖出

**判断标准**: 收盘价是否站稳突破位

### 形态 4: 十字星/小阴线次日

**T 日**: 震荡整理

**T+1 策略**:
- 方向选择日，等明确信号
- 突破昨日高点 → 持有/加仓
- 跌破昨日低点 → 卖出

## 模块 4: 尾盘买入法 (最适合上班族)

### 策略逻辑

**买入时间**: 14:30-14:55
**卖出时间**: 次日 9:30-10:30
**持仓时间**: 隔夜 1 天

**优势**:
- 避免当日盘中波动风险
- 次日早盘决定去留，灵活
- 适合没时间盯盘的人

### 选股条件 (必须同时满足)

1. **趋势向上**: 价格在 MA20 上方
2. **当日强势**: 涨幅 3-7%，不是涨停
3. **成交量**: 放量但不巨量 (1.5-3 倍均量)
4. **收盘位置**: 接近当日最高价 (上影线短)
5. **RSI**: 50-70 区间 (强势但不超买)
6. **无利空**: 晚间无预期利空消息

### 次日卖出规则

**必须卖出的情况**:
1. 低开超过 -3% → 9:35 反抽卖出
2. 开盘后跌破昨日最低价 → 立即卖出
3. 亏损达到 -5% → 无条件止损

**可以持有的情况**:
1. 高开 2%+ 且快速拉升 → 持有到 10:00
2. 平开但站稳昨日收盘价 → 观察 30 分钟

**止盈策略**:
- 盈利 5%+ → 卖出一半
- 盈利 10%+ → 全部止盈

## 模块 5: 量化决策公式

### 次日卖出概率模型

```
卖出概率 = 
  技术面得分 × 40% + 
  情绪面得分 × 35% + 
  资金面得分 × 25%
```

**技术面得分 (0-100)**:
- 价格在 MA5 上方：+20
- MA 多头排列：+20
- MACD 金叉：+20
- RSI < 70: +20
- 未跌破关键支撑：+20

**情绪面得分 (0-100)**:
- 昨日涨停：+30
- 昨日涨幅>5%: +20
- 板块强势：+20
- 无利空消息：+15
- 大盘环境好：+15

**资金面得分 (0-100)**:
- 昨日放量：+20
- 主力净流入：+30
- 北向资金流入：+20
- 龙虎榜机构买入：+30

**决策阈值**:
- 卖出概率 > 70% → 坚决卖出
- 卖出概率 50-70% → 减仓
- 卖出概率 < 50% → 持有

## 模块 6: 实战案例模板

### 案例：美诺华 (603538) 涨停后次日

**T 日 (3 月 13 日) 数据**:
- 收盘价：25.37 元
- 涨跌幅：+10.02% (涨停)
- 成交量：22.7 万手 (放量)
- 形态：突破前期平台

**T+1 日 (3 月 16 日) 策略**:

| 情景 | 开盘 | 操作 | 仓位 |
|------|------|------|------|
| 强势 | 高开 5%+ | 持有，冲高止盈 | 100% → 50% |
| 中性 | 高开 0-5% | 观察，破 25 元减仓 | 100% → 70% |
| 弱势 | 低开 | 反抽卖出 | 100% → 0% |

**关键价位**:
- 支撑：24.50 元 (涨停价一半)
- 压力：26.50 元 (前高)
- 止损：24.00 元

**预期**:
- 连板概率：25%
- 震荡概率：45%
- 回调概率：30%

## 模块 7: 风险控制

### 仓位管理

**单只股票上限**: 不超过总资金 30%
**总仓位上限**: 根据大盘环境调整
- 大盘强势：80-100%
- 大盘震荡：50-70%
- 大盘弱势：0-30%

### 止损纪律

**必须止损的情况**:
1. 亏损达到 -8% → 无条件止损
2. 跌破关键支撑位 → 技术止损
3. 买入逻辑被证伪 → 逻辑止损

**止损后心态**:
- 止损是成本，不是失败
- 连续止损 3 次 → 暂停交易，复盘

### 止盈策略

**分批止盈**:
- 盈利 10% → 卖出 30%
- 盈利 20% → 再卖出 30%
- 剩余 40% → 追踪止损 (最高点回撤 10%)

## 核心收获

1. **次日决策关键在盘前准备** — 晚上做好预案，早上执行
2. **止损比止盈重要** — 先保命，再赚钱
3. **尾盘买入法最适合上班族** — 风险可控，决策简单
4. **量化是概率游戏** — 不追求每次都赢，追求长期正期望
5. **纪律是短线生命线** — 严格按规则执行，不情绪化
