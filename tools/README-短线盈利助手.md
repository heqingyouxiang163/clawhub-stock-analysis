## 📋 系统概述

本系统是一个**专业的隔日超短线选股助手**，结合：
- ✅ 分时图形态分析
- ✅ 历史涨停形态库
- ✅ 每日学习成果
- ✅ 综合评分系统
- ✅ 市场情绪判断

专注于**今天买入、明天出货**的超短线模式，帮助用户快速盈利。

## 🎯 选股逻辑 (隔日超短线专用)

### 基础过滤条件

```python
# 排除条件
if change_pct < 3 or change_pct >= 9:  # 涨幅过小或接近涨停
    continue
if turnover < 3 or turnover > 20:  # 换手不足或过高
    continue
if amount < 300000000:  # 成交额<3 亿
    continue
```

### 核心筛选条件

1. **分时图形态强势** (≥25 分)
   - 股价在分时均线上方
   - 涨幅 3-7% (有上涨空间)
   - 量价配合健康

2. **历史涨停形态** (≥20 分)
   - 近期有涨停记录 (股性活跃)
   - 涨停形态胜率高 (>60%)
   - 有连板记录加分

3. **量能资金充足** (≥15 分)
   - 成交额>5 亿 (流动性好)
   - 换手率 5-15% (充分换手)
   - 量比>1.5 (放量)

4. **市场情绪向好** (≥10 分)
   - 热门概念 (芯片、AI、新能源等)
   - 板块效应强

5. **历史次日溢价** (≥8 分)
   - 该股历史次日表现好
   - 类似形态次日溢价高

## 📈 盈利模式

### 核心逻辑

```
胜率：60%+
盈亏比：1:2 (亏 3% 赚 6%)
月交易次数：20-30 次
月盈利目标：20-30%
```

### 示例计算

```
假设每月交易 25 次：
- 盈利 15 次 (60%) × 平均 6% = +90%
- 亏损 10 次 (40%) × 平均 3% = -30%
- 净利润：+60%

考虑仓位控制 (平均 30% 仓)：
实际月盈利：60% × 30% = 18%
```

### 成功关键

1. **严格止损** (-3% 无条件止损)
2. **及时出货** (次日 9:30-10:00 必须出货)
3. **仓位控制** (单只≤30%，总仓≤60%)
4. **选股质量** (≥80 分才推荐)
5. **心态管理** (宁可空仓，不做弱势)

## 🔧 故障排除

### 问题 1: 无股票推荐

**原因**:
- 市场整体强度不足
- 未达到推送阈值 (≥80 分)
- 非交易时间

**解决**:
- 保持耐心，等待更好机会
- 超短线讲究：宁可空仓，不做弱势
- 检查是否在 9:35-10:35 时段

---

### 问题 2: 推送太多

**原因**:
- 市场行情火爆
- 阈值设置过低

**解决**:
- 提高阈值 (80→85)
- 减少每次推送数量 (3→2)

---

### 问题 3: 胜率不理想

**原因**:
- 止损执行不严格
- 出货不及时 (贪心)
- 仓位控制不当

**解决**:
- 严格执行 -3% 止损
- 次日 9:30-10:00 必须出货
- 单只仓位≤30%
