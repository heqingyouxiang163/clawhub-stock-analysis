## 3️⃣ 止损策略优化

### 当前止损
```
固定止损：-5%
技术止损：破支撑
时间止损：无
```

### 优化后止损
```
固定止损：-5% (不变)
技术止损:
  - 破 5 日线：-3%
  - 破分时均线：-2%
  - 破突破位：-3%
时间止损:
  - 3 天不涨：卖出
  - 不板即卖：次日
动态止损:
  - 盈利>10%: 追踪止损 -5%
  - 盈利>20%: 追踪止损 -10%
```

### 止损执行优化
```
自动止损:
  - 设置条件单
  - 触及自动卖出
  - 不人为干预

止损纪律:
  - 连续止损 3 次→暂停 1 天
  - 单日亏损 -10%→停止交易
  - 总回撤 -15%→空仓 1 周
```
