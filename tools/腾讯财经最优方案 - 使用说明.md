## 📊 核心功能

### 1. 实时数据获取

```python
get_stock_data(codes, use_cache=True)
```

**参数**:
- `codes`: 股票代码列表
- `use_cache`: 是否使用缓存 (默认 True)

**返回**: 股票数据列表

**特性**:
- ✅ 多线程并发 (5 个线程)
- ✅ 智能缓存 (2 分钟 TTL)
- ✅ 自动重试 (最多 3 次)
- ✅ 批量获取 (每批 150 只)

### 2. 涨幅筛选

```python
filter_by_change_pct(stocks, min_pct, max_pct)
```

**参数**:
- `stocks`: 股票数据列表
- `min_pct`: 最小涨幅
- `max_pct`: 最大涨幅

**示例**:
```python
# 筛选涨幅 5%-7% 的主升浪股票
target = filter_by_change_pct(stocks, 5, 7)
```

### 3. 综合评分

```python
calculate_score(stock)
```

**评分维度**:
- 涨幅趋势：40 分
- 成交活跃度：30 分
- 量比：20 分
- 开盘强度：10 分

**示例**:
```python
# 获取高确定性股票 (≥75 分)
high_quality = get_high_quality_stocks(stocks, min_score=75)
```

### 4. 报告生成

```python
generate_report(stocks, title="报告标题")
print_top_stocks(stocks, top_n=20)
print_score_stocks(stocks, min_score=75)
```

## 💡 使用场景

### 场景 1: 持仓监控

```python
# 监控持仓股
holdings = ["002828", "002342", "603778"]
stocks = get_stock_data(holdings)

for s in stocks:
    print(f"{s['name']}: ¥{s['current']} ({s['change_pct']:+.1f}%)")
```

### 场景 2: 选股

```python
# 获取涨幅 5%-7% 的高评分股票
all_stocks = get_stock_data(your_stock_pool)
target = filter_by_change_pct(all_stocks, 5, 7)
high_quality = get_high_quality_stocks(target, min_score=75)

print(f"找到{len(high_quality)}只高确定性股票")
for s in high_quality[:5]:
    print(f"{s['code']} {s['name']} - {s['score']}分")
```

### 场景 3: 全市场扫描

```python
# 从文件读取股票池
with open('stock_pool.txt', 'r') as f:
    codes = [line.strip() for line in f.readlines()]

# 获取全市场数据
stocks = get_stock_data(codes)

# 生成报告
generate_report(stocks, "全市场涨幅统计")
print_top_stocks(stocks, top_n=30)
```

## ⚠️ 注意事项

1. **编码**: 腾讯返回 GBK 编码，脚本已自动处理
2. **频率**: 建议单次请求间隔≥100ms (已自动处理)
3. **批量**: 单次最多 150 只，超过自动分批
4. **超时**: 默认 8 秒超时
5. **缓存**: 默认 2 分钟，可在配置区修改
