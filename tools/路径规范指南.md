## ❌ 常见路径错误

### 错误 1: 使用 `~` 路径

```python
# ❌ 错误
CACHE_FILE = "~/openclaw/workspace/temp/cache.json"
BASE_DIR = "~/openclaw/workspace"
```

**问题**: `~` 在不同环境下可能解析不同，导致文件找不到

---

### 错误 2: 相对路径不一致

```python
# ❌ 错误 (在不同目录执行时路径不同)
with open('temp/cache.json', 'r') as f:
    data = json.load(f)
```

**问题**: 工作目录变化时路径失效

---

### 错误 3: 硬编码用户目录

```python
# ❌ 错误 (不同用户不同)
CACHE_FILE = "/home/admin/openclaw/workspace/temp/cache.json"
```

**问题**: 换用户或环境时代码失效

---

## ✅ 正确做法

### 方案 1: 使用 `os.path.expanduser()` (推荐)

```python
# ✅ 正确
import os

BASE_DIR = os.path.expanduser("~/openclaw/workspace")
CACHE_FILE = os.path.join(BASE_DIR, "temp/cache.json")

# 或使用绝对路径
BASE_DIR = "/home/admin/openclaw/workspace"
CACHE_FILE = os.path.join(BASE_DIR, "temp/cache.json")
```

**优势**: 
- ✅ 自动处理 `~` 展开
- ✅ 跨平台兼容
- ✅ 代码清晰

---

### 方案 2: 使用 `pathlib` (现代 Python)

```python
# ✅ 正确 (Python 3.4+)
from pathlib import Path

BASE_DIR = Path.home() / "openclaw" / "workspace"
CACHE_FILE = BASE_DIR / "temp" / "cache.json"

# 使用
with open(CACHE_FILE, 'r') as f:
    data = json.load(f)
```

**优势**:
- ✅ 面向对象，更优雅
- ✅ 自动处理路径分隔符
- ✅ 支持路径运算

---

### 方案 3: 使用脚本所在目录

```python
# ✅ 正确 (基于脚本位置)
import os

# 获取脚本所在目录
SCRIPT_DIR = os.path.dirname(os.path.abspath(__file__))
BASE_DIR = os.path.join(SCRIPT_DIR, "..", "..")
CACHE_FILE = os.path.join(BASE_DIR, "temp", "cache.json")
```

**优势**:
- ✅ 不依赖工作目录
- ✅ 适合打包分发
- ✅ 相对位置固定

## 🔧 路径检查工具

创建 `tools/路径检查工具.py`:

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
路径检查工具
检查所有文件中的路径是否规范
"""

import os
import re
from pathlib import Path

def check_path_in_file(filepath):
    """检查文件中的路径是否规范"""
    issues = []
    
    with open(filepath, 'r', encoding='utf-8') as f:
        lines = f.readlines()
    
    for i, line in enumerate(lines, 1):
        # 检查~路径 (代码文件中)
        if filepath.endswith('.py'):
            if re.search(r'["\']~/', line) and 'expanduser' not in line:
                issues.append(f"  第{i}行：发现~路径，建议使用 os.path.expanduser()")
        
        # 检查硬编码路径
        if re.search(r'/home/[a-z]+/openclaw', line):
            if 'BASE_DIR' not in line and '配置' not in line:
                issues.append(f"  第{i}行：发现硬编码路径，建议使用常量")
    
    return issues

def scan_directory(directory):
    """扫描目录检查所有文件"""
    print(f"🔍 检查目录：{directory}")
    print()
    
    total_issues = 0
    
    for root, dirs, files in os.walk(directory):
        # 跳过隐藏目录
        dirs[:] = [d for d in dirs if not d.startswith('.')]
        
        for file in files:
            if file.endswith(('.py', '.md', '.json', '.yaml')):
                filepath = os.path.join(root, file)
                issues = check_path_in_file(filepath)
                
                if issues:
                    print(f"⚠️ {filepath}:")
                    for issue in issues:
                        print(issue)
                    print()
                    total_issues += len(issues)
    
    print(f"总计：{total_issues} 个问题")
    return total_issues

if __name__ == "__main__":
    import sys
    
    directory = sys.argv[1] if len(sys.argv) > 1 else "."
    issues = scan_directory(directory)
    
    if issues > 0:
        print("\n❌ 发现路径规范问题，请修复")
        exit(1)
    else:
        print("\n✅ 所有路径规范")
        exit(0)
```

**使用**:
```bash
# 检查当前目录
python3 tools/路径检查工具.py .

# 检查指定目录
python3 tools/路径检查工具.py skills/realtime-monitor-3min/
```

## 📝 路径最佳实践

### 1. 集中管理路径

```python
# config.py
import os
from pathlib import Path

class Config:
    # 基础路径
    BASE_DIR = Path.home() / "openclaw" / "workspace"
    
    # 子目录
    SKILLS_DIR = BASE_DIR / "skills"
    TOOLS_DIR = BASE_DIR / "tools"
    TEMP_DIR = BASE_DIR / "temp"
    MEMORY_DIR = BASE_DIR / "memory"
    
    # 缓存文件
    CACHE_FILE = TEMP_DIR / "cache.json"
    
    # 确保目录存在
    @classmethod
    def init(cls):
        for dir_path in [cls.TEMP_DIR, cls.MEMORY_DIR]:
            dir_path.mkdir(parents=True, exist_ok=True)
```

**使用**:
```python
from config import Config

Config.init()
data = load_cache(Config.CACHE_FILE)
```

### 3. 路径验证

```python
def validate_path(path, must_exist=True):
    """验证路径是否有效"""
    from pathlib import Path
    
    path = Path(path)
    
    if must_exist and not path.exists():
        raise FileNotFoundError(f"路径不存在：{path}")
    
    # 检查权限
    if not os.access(path, os.R_OK):
        raise PermissionError(f"无读取权限：{path}")
    
    return path
```

## 🚨 常见错误及修复

### 错误 1: `FileNotFoundError`

**原因**: `~` 未展开

**修复**:
```python
# ❌ 错误
with open("~/openclaw/workspace/file.txt", 'r') as f:
    content = f.read()

# ✅ 正确
import os
with open(os.path.expanduser("~/openclaw/workspace/file.txt"), 'r') as f:
    content = f.read()
```

---

### 错误 2: `Edit failed` (路径不匹配)

**原因**: 文档中的路径与实际文件路径不一致

**修复**:
```markdown
# ❌ 错误 (文档中)
文件位置：`skills/realtime-monitor-3min/`

# ✅ 正确 (文档中)
文件位置：`/home/admin/openclaw/workspace/skills/realtime-monitor-3min/`
```

---

### 错误 3: 跨平台路径分隔符

**原因**: Windows 用 `\`，Linux/Mac 用 `/`

**修复**:
```python
# ❌ 错误 (Windows 不兼容)
path = "temp/cache.json"

# ✅ 正确 (跨平台)
import os
path = os.path.join("temp", "cache.json")

# 或使用 pathlib
from pathlib import Path
path = Path("temp") / "cache.json"
```
