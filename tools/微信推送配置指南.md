# 🦞 微信推送配置指南

**配置时间**: 2026-03-29
**用途**: 通过微信推送小龙虾推荐、复盘、升级通知

---

## 📱 方案选择

### 方案 1: 企业微信机器人 (推荐 ⭐⭐⭐⭐⭐)

**优点**:
- ✅ 免费
- ✅ 配置简单
- ✅ 支持 Markdown
- ✅ 稳定可靠

**缺点**:
- ❌ 需要企业微信账号

**配置步骤**:

1. **创建企业微信**
   - 访问：https://work.weixin.qq.com/
   - 注册企业微信 (个人也可以创建)

2. **创建机器人**
   - 进入：工作台 → 群机器人 → 添加
   - 设置机器人名称：炒股龙虾
   - 获取 Webhook URL

3. **配置 Webhook**
   ```bash
   # 在 ~/.openclaw/wechat_config.json 中配置
   {
     "wechat_webhook": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY"
   }
   ```

---

### 方案 2: 微信测试号 (推荐 ⭐⭐⭐⭐)

**优点**:
- ✅ 免费
- ✅ 直接推送到个人微信
- ✅ 支持模板消息

**缺点**:
- ❌ 需要关注公众号
- ❌ 配置稍复杂

**配置步骤**:

1. **申请微信测试号**
   - 访问：https://mp.weixin.qq.com/debug/cgi-bin/sandboxinfo
   - 扫码登录

2. **配置模板消息**
   - 模板消息 → 添加模板
   - 选择行业：科技/互联网
   - 添加模板：推送通知

3. **获取配置信息**
   - appid
   - appsecret
   - 模板 ID

4. **配置到文件**
   ```json
   {
     "wechat_appid": "YOUR_APPID",
     "wechat_appsecret": "YOUR_SECRET",
     "wechat_template_id": "YOUR_TEMPLATE_ID",
     "wechat_openid": "YOUR_OPENID"
   }
   ```

### 方案 3: Server 酱 (推荐 ⭐⭐⭐)

**优点**:
- ✅ 配置最简单
- ✅ 无需公众号
- ✅ 免费

**缺点**:
- ❌ 功能较少

**配置步骤**:

1. **获取 SendKey**
   - 访问：https://sct.ftqq.com/
   - 微信扫码登录
   - 获取 SendKey

2. **配置到文件**
   ```json
   {
     "serverchan_sendkey": "YOUR_SENDKEY"
   }
   ```

## 🛠️ 推荐配置：企业微信

### 1. 安装依赖

```bash
pip3 install requests
```

### 2. 创建配置文件

```bash
mkdir -p ~/.openclaw
cat > ~/.openclaw/wechat_config.json << 'EOF'
{
  "platform": "企业微信",
  "webhook": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY"
}
EOF
```

### 3. 测试推送

```python
python3 skills/微信推送.py
```

## 📊 推送内容配置

### 推送类型

| 类型 | 时间 | 内容 |
|------|------|------|
| **早盘候选** | 09:25 | 集合竞价结果 |
| **打板推荐** | 09:30 | 重点推荐股票 |
| **盘中更新** | 10:00/11:00/13:00/14:00 | 盘中扫描结果 |
| **收盘总结** | 15:00 | 今日推荐统计 |
| **复盘报告** | 15:30 | 胜率 + 盈亏 |
| **策略升级** | 20:00 | 版本更新通知 |
| **代码同步** | 22:00 | Git 推送通知 |

### 推送格式

```markdown
🦞 **小龙虾 · 打板推荐**

⏰ 时间：09:30
🎯 类型：早盘推荐

🔥 **重点推荐**
1. 600227 赤天化
   - 综合评分：92
   - 涨停概率：95%
   - 封单：3.5 亿
   - 连板：3 板

⚠️ **风险提示**
- 炸板立即止损
- 回撤≥4% 坚决卖出

---
📊 模型版本：v18.0
```

---

## ⚠️ 注意事项

1. **推送频率** - 避免过于频繁，建议每小时最多 1 次
2. **推送时间** - 交易时段推送，非交易时段静默
3. **推送内容** - 仅推送推荐，不推送具体买卖点
4. **风险提示** - 每次推送必须包含风险提示
