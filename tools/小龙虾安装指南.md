# 🦞 小龙虾·主板打板保活版 安装指南

## 📋 功能说明

- **24 小时保活** - 后台持续运行，自动重启
- **自动打板** - 交易时段自动执行策略
- **只做主板** - 沪深主板 (600/601/603/605/000/001/002/003)
- **无 Token 消耗** - 使用腾讯财经免费接口

---

## 🛠️ 安装步骤

### 方法 1: systemd 服务 (推荐)

#### 1. 复制服务文件

```bash
sudo cp /home/admin/openclaw/workspace/clawhub-stock-analysis/tools/小龙虾.service /etc/systemd/system/
```

#### 2. 重载 systemd

```bash
sudo systemctl daemon-reload
```

#### 3. 启用服务

```bash
sudo systemctl enable 小龙虾.service
```

#### 4. 启动服务

```bash
sudo systemctl start 小龙虾.service
```

#### 5. 查看状态

```bash
sudo systemctl status 小龙虾.service
```

#### 6. 查看日志

```bash
sudo journalctl -u 小龙虾.service -f
```

---

### 方法 2: 手动运行

#### 1. 直接运行

```bash
cd /home/admin/openclaw/workspace/clawhub-stock-analysis
python3 skills/小龙虾_保活版.py
```

#### 2. 后台运行 (nohup)

```bash
nohup python3 skills/小龙虾_保活版.py > temp/小龙虾后台日志.txt 2>&1 &
```

#### 3. 查看进程

```bash
ps aux | grep 小龙虾
```

#### 4. 停止进程

```bash
pkill -f 小龙虾_保活版.py
```

## 🔧 管理命令

## ⚠️ 注意事项

1. **交易时段** - 只在周一至周五 09:25-15:30 执行策略
2. **周末** - 周末不执行交易任务，但心跳正常
3. **节假日** - 需要手动暂停服务
4. **异常重启** - 服务异常会自动重启 (间隔 10 秒)

## 🛡️ 故障排除

### 问题 1: 服务无法启动

```bash
# 查看详细错误
sudo journalctl -u 小龙虾.service -n 50

# 检查 Python 路径
which python3

# 检查文件权限
ls -la skills/小龙虾_保活版.py
```

### 问题 2: 心跳停止

```bash
# 重启服务
sudo systemctl restart 小龙虾.service

# 查看最近日志
tail -100 temp/小龙虾运行日志.txt
```

### 问题 3: 定时任务不执行

```bash
# 检查系统时间
date

# 检查时区
timedatectl

```
