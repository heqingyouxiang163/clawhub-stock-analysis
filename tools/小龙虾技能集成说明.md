## 🎯 集成方式

### 方式 1: 直接调用 Python 脚本

```python
from skills.小龙虾_main_board import 小龙虾

# 初始化
lxl = 小龙虾()

# 扫板选股
targets = lxl.scan(实时行情)

# 学习复盘
lxl.learn(今日交易记录)

# 进化优化
lxl.evolve(一周历史数据)
```

### 方式 2: 定时任务自动执行

```bash
# 添加定时任务
openclaw cron add \
  --name "🦞 小龙虾早盘扫板" \
  --cron "25 9 * * 1-5" \
  --system-event "python3 /home/admin/openclaw/workspace/clawhub-stock-analysis/skills/小龙虾_反馈版.py"
```

### 方式 3: systemd 服务后台运行

```bash
# 安装服务
sudo cp tools/小龙虾.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable 小龙虾.service
sudo systemctl start 小龙虾.service

# 查看状态
sudo systemctl status 小龙虾.service
```

## 🛠️ 使用流程

### 盘前 (09:15-09:25)

1. 系统自动启动 (systemd 服务)
2. 09:25 集合竞价扫描
3. 推送主板候选列表

### 盘中 (09:30-11:30, 13:00-15:00)

1. 09:30 执行打板策略
2. 实时筛选符合条件股票
3. 推送推荐列表 (前 3 强)
4. 每分钟心跳保活

### 盘后 (15:30)

1. 统计今日交易
2. 计算胜率
3. 保存到学习记录
4. 推送复盘报告

### 夜间 (20:00)

1. 加载历史数据
2. 网格搜索最优参数
3. 更新策略参数
4. 推送升级通知

---

## ⚠️ 注意事项

1. **交易时段** - 只在周一至周五 09:25-15:30 执行策略
2. **主板限制** - 创业板 (300)/科创板 (688) 自动过滤
3. **节假日** - 需要手动暂停服务
4. **异常重启** - 服务异常会自动重启 (间隔 10 秒)
