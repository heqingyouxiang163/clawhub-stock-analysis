# 盘中分时图评分推送系统

**创建时间**: 2026-03-21 14:39  
**版本**: v1.0  
**功能**: 每 5 分钟推送高确定性股票 (9:35-10:35)

---

## 📋 系统概述

本系统结合**分时图形态 + 涨幅 + 量能 + 换手**四大维度，每分钟扫描全市场，自动筛选高确定性股票并推送。

### 核心特性

| 特性 | 说明 |
|------|------|
| **推送频率** | 每 5 分钟一次 |
| **推送时段** | 9:35-10:35 (共 1 小时，12 次) |
| **推送阈值** | ≥75 分 (高确定性) |
| **每次数量** | 最多 5 只 |
| **数据源** | 腾讯财经 + 新浪财经 (v2.0) |
| **评分维度** | 分时图 (40 分) + 涨幅 (25 分) + 量能 (20 分) + 换手 (15 分) |


### 2. 设置定时任务 (cron)

```bash
openclaw cron add --name="盘中推送 -9:35" \
  --schedule="cron:35 9 * * 1-5" \
  --command="python3 /home/admin/openclaw/workspace/tools/盘中分时图评分推送.py" \
  --timeout=120
```

### 3. 添加所有推送时点

```bash
# 9:35-10:35，每 5 分钟一次
for minute in 35 40 45 50 55 60; do
  hour=9
  if [ $minute -ge 60 ]; then
    hour=10
    minute=$((minute-60))
  fi
  openclaw cron add --name="盘中推送-${hour}:${minute}" \
    --schedule="cron:${minute} ${hour} * * 1-5" \
    --command="python3 /home/admin/openclaw/workspace/tools/盘中分时图评分推送.py" \
    --timeout=120
done
```

## 📊 评分系统

### 评分维度 (总分 100 分)

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **分时图形态** | 40 分 | 股价在均线上方、均线斜率、涨幅 |

## 🔧 故障排除

### 问题 1: 无股票推荐

**原因**:
- 市场整体强度不足
- 未达到推送阈值 (≥75 分)
- 非交易时间

**解决**:
- 降低阈值 (临时测试用)
- 等待更好机会
- 检查是否在 9:35-10:35 时段

---

### 问题 2: 推送太多

**原因**:
- 阈值设置过低
- 市场行情火爆

**解决**:
- 提高阈值 (75→80)
- 减少每次推送数量 (5→3)

---

### 问题 3: 执行超时

**原因**:
- 网络请求慢
- 股票池过大

**解决**:
- 增加超时时间 (120→180 秒)
- 优化并发线程数 (10→15)
