# 低开跌停风险预警系统

## ⚠️ 这是最危险的短线形态！

**为什么危险？**
- 单日亏损可达 15%+ (昨日涨停→今日跌停)
- 流动性枯竭，想卖卖不掉
- 次日通常继续低开，亏损扩大
- 心态容易崩溃，操作变形

---

## 🎯 风险评估维度

### 1. 连板高度风险 (30 分)
| 连板数 | 风险得分 | 说明 |
|--------|----------|------|
| ≥5 板 | +30 分 | 极高位，随时退潮 |
| 3-4 板 | +20 分 | 高位，注意分化 |
| 1-2 板 | 0 分 | 正常 |

### 2. 涨停质量风险 (25 分)
| 昨日涨停质量 | 风险得分 |
|--------------|----------|
| C 级 (尾盘/巨量/超大单流出) | +25 分 |
| B 级 (午后/量比偏高) | +10 分 |
| A 级 (早盘/健康放量) | 0 分 |

### 3. 资金流向风险 (25 分)
| 超大单流向 | 风险得分 |
|------------|----------|
| <-3000 万 | +25 分 |
| -3000 万至 0 | +10 分 |
| >0 | 0 分 |

### 4. 量比风险 (20 分)
| 量比 | 风险得分 |
|------|----------|
| >5 倍 | +20 分 (巨量警惕) |
| 3-5 倍 | +10 分 |
| <3 倍 | 0 分 |

## 📊 风险等级

| 总分 | 风险等级 | 应对策略 |
|------|----------|----------|
| ≥70 | 🔴🔴🔴 极高风险 | 集合竞价坚决止损 |
| 50-69 | 🔴🔴 高风险 | 反抽坚决卖出 |
| 30-49 | 🔴 中高风险 | 设置止损，准备逃命 |
| <30 | 🟡 中等风险 | 观察集合竞价 |

---

## ⚠️ 必须止损的信号

### 集合竞价阶段 (9:15-9:25)
- 跌停封单 > 10 万手 → 挂跌停价卖出
- 低开 -8% 以下 → 准备止损
- 买盘几乎为零 → 坚决卖出

### 开盘后 5 分钟
- 快速杀向跌停 → 反抽卖出
- 无反抽直接跌 → 市价卖出
- 跌破开盘价 → 止损

### 开盘后 30 分钟
- 仍在 -5% 以下 → 卖出
- 无法翻红 → 减仓
- 再次下跌 → 清仓

---

## 📋 应对预案

### 预案 A: 集合竞价止损 (最果断)
```
适用：极高风险股
操作：9:24 分挂跌停价，全部卖出
```

### 预案 B: 反抽止损 (最常用)
```
适用：高风险股
操作：等待反抽到 -3% 至 0%，卖出 50-100%
```

### 预案 C: 分批止损 (折中)
```
适用：中高风险股
操作：
  集合竞价：卖出 30%
  反抽时：卖出 30%
  剩余 40%：观察
```

## 💡 心态建设

### 亏损回本难度
```
亏损 10% → 需要涨 11% 回本
亏损 20% → 需要涨 25% 回本
亏损 30% → 需要涨 43% 回本
亏损 50% → 需要涨 100% 回本

结论：越早止损，回本越容易！
```

### 正确心态
1. 接受亏损 — 止损是保命，不是失败
2. 果断执行 — 不抱幻想，不侥幸
3. 保护本金 — 留得青山在，不怕没柴烧
4. 总结经验 — 避免下次犯同样错误
