# 🦞 涨停分时图 + 成交量训练系统

**创建时间**: 2026-03-29
**用途**: 基于历史涨停股的分时图和成交量特征，训练涨停预测模型，优化盘中推荐

---

## 📊 训练系统架构

```
历史涨停股数据
    ↓
分时图特征分析
├─ 均线上方运行占比
├─ 价格波动率
├─ 上涨斜率
├─ 均线支撑次数
└─ 量价配合度
    ↓
成交量特征分析
├─ 平均成交量
├─ 成交量波动率
├─ 放量倍数
├─ 缩量比 (涨停后缩量)
└─ 堆量特征 (连续放量)
    ↓
训练涨停预测模型
    ↓
优化盘中推荐评分
```

---

## 🎯 核心特征

### 1. 分时图特征 (40 分)

| 特征 | 说明 | 涨停股典型值 |
|------|------|-------------|
| **均线上方占比** | 价格在均线上方的时间占比 | >80% |
| **价格波动率** | 价格波动程度 | <2% |
| **上涨斜率** | 价格上涨速度 | 正斜率 |
| **均线支撑次数** | 回踩均线后反弹次数 | ≥3 次 |
| **量价配合度** | 价升量增的次数占比 | >60% |

### 2. 成交量特征 (30 分)

| 特征 | 说明 | 涨停股典型值 |
|------|------|-------------|
| **平均成交量** | 平均成交水平 | 基准值 |
| **成交量波动率** | 成交量波动程度 | <50% |
| **放量倍数** | 最大量/平均量 | 3-5 倍 |
| **缩量比** | 涨停后成交量萎缩程度 | <0.3 |
| **堆量特征** | 连续放量次数占比 | >20% |

### 3. 其他特征 (30 分)

| 特征 | 权重 | 说明 |
|------|------|------|
| **连板数** | 15 分 | ≥4 板 15 分，≥3 板 12 分，≥2 板 8 分 |
| **封单额** | 10 分 | ≥5 亿 10 分，≥3 亿 7 分，≥2 亿 4 分 |
| **板块效应** | 5 分 | 板块≥5 只涨停 5 分 |

## 🛠️ 使用方法

### 1. 收集历史数据

```python
from skills.涨停分时图训练 import LimitUpTrainer

trainer = LimitUpTrainer()

# 历史涨停股数据 (需要包含 intraday_data)
历史数据 = [
    {
        '代码': '600227',
        '名称': '赤天化',
        '连板数': 3,
        '封单额亿': 3.5,
        'intraday_data': [
            {'time': '093000', 'price': 4.00, 'volume': 1000, 'avg_price': 4.00},
            {'time': '093100', 'price': 4.10, 'volume': 1200, 'avg_price': 4.05},
            # ... 更多分时数据
        ]
    },
    # ... 更多涨停股
]
```

### 2. 训练模型

```python
# 训练涨停预测模型
trainer.train(历史数据)

# 模型自动保存到 memory/涨停预测模型.json
```

### 3. 预测评分

```python
# 预测当前股票的涨停概率
股票数据 = {
    '代码': '600890',
    '名称': '法尔胜',
    '连板数': 4,
    '封单额亿': 2.8,
    'intraday_data': [...]  # 实时分时数据
}

score = trainer.predict_score(股票数据)
print(f"涨停概率评分：{score}")

# ≥80 分：重点推荐
# 70-79 分：可关注
# <70 分：观望
```

### 4. 集成到盘中推荐

```python
from skills.小龙虾_main_board import 小龙虾
from skills.涨停分时图训练 import LimitUpTrainer

lxl = 小龙虾()
trainer = LimitUpTrainer()

# 加载训练好的模型
trainer.load_model()

# 扫板选股
targets = lxl.scan(实时行情)

# 对每个目标进行涨停概率预测
for target in targets:
    score = trainer.predict_score(target)
    target['涨停概率'] = score

# 按涨停概率排序
targets.sort(key=lambda x: x['涨停概率'], reverse=True)

# 推荐前 3 强
print("重点推荐:")
for i, t in enumerate(targets[:3], 1):
    print(f"{i}. {t['代码']} {t['名称']} (涨停概率：{t['涨停概率']})")
```

## 📈 训练流程

### 步骤 1: 数据收集

收集过去 1-3 个月的涨停股数据：
- 股票代码
- 连板数
- 封单额
- 分时图数据 (价格、成交量、均线)
- 次日溢价 (用于验证)

### 步骤 2: 特征提取

对每只涨停股提取特征：
- 分时图特征 (5 个维度)
- 成交量特征 (5 个维度)
- 其他特征 (连板、封单、板块)

### 步骤 3: 模型训练

计算平均特征作为标准模板：
- 均线上方占比均值
- 价格波动率均值
- 成交量特征均值
- ...

### 步骤 4: 模型验证

使用最近 1 周的涨停股验证模型：
- 计算预测准确率
- 调整评分权重
- 优化推荐阈值

### 步骤 5: 持续更新

每日盘后自动更新：
- 添加当日涨停股数据
- 重新训练模型
- 优化预测准确率

## ⚠️ 注意事项

1. **数据质量** - 需要完整的分时图数据 (至少 240 个点/天)
2. **数据量** - 至少需要 50 只涨停股才能训练
3. **更新频率** - 建议每周重新训练一次
4. **验证机制** - 用最近数据验证，避免过拟合
