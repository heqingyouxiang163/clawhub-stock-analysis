## 🎯 核心形态

### 1. 涨停连板形态

**评估维度**:
- 涨停时间 (早盘/午后/尾盘)
- 封单金额
- 量比
- 超大单流向

**分级**:
- A 级：连板概率 60%+
- B 级：连板概率 30-60%
- C 级：连板概率<30%

### 2. 五日线首阴反包形态

**前置条件**:
- 趋势向上 (MA5>MA10>MA20)
- 之前至少 3 连阳
- 今日首阴 (-1% 至 -5%)
- 缩量回调 (<0.8 倍)
- 5 日线支撑有效

**评分系统** (100 分):
- 趋势强度：25 分
- 缩量程度：25 分
- 5 日线支撑：20 分
- 资金流向：15 分
- 板块强度：15 分

**概率对应**:
- ≥80 分：反包概率 70%+
- 60-79 分：反包概率 50-70%
- 40-59 分：反包概率 30-50%
- <40 分：放弃

### 3. 低开跌停形态

**风险维度**:
- 连板高度 (≥5 板高危)
- 涨停质量 (C 级高危)
- 资金流向 (超大单流出高危)
- 量比 (>5 倍高危)

**预警等级**:
- 🔴🔴🔴 极高风险：集合竞价止损
- 🔴🔴 高风险：反抽卖出
- 🔴 中高风险：设置止损

## ⚠️ 预警系统

### 三级预警

| 等级 | 触发条件 | 应对 |
|------|----------|------|
| **🔴🔴🔴 一级** | 跌停/跌超 -8% | 立即止损 |
| **🔴🔴 二级** | 跌超 -5%/超大单流出>5000 万 | 反抽卖出 |
| **🔴 三级** | 跌破开盘价 | 减仓止损 |
| **🟢 机会** | 涨停/涨超 5% | 持有/打板 |
