# 盘中实时监控系统

## 🎯 系统功能

基于短线高手操盘手法，实时监控持仓股和市场情绪，自动预警。

## ⏰ 监控时间

| 时间 | 任务 | 说明 |
|------|------|------|
| **9:25** | 盘前分析 | 集合竞价结束后发送 |
| **10:00** | 盘中监控 1 号 | 开盘后 30 分钟，确认趋势 |
| **14:00** | 盘中监控 2 号 | 午后开盘，尾盘决策前 |

## 📊 监控内容

### 1. 市场情绪 (权重 40%)

| 指标 | 权重 | 说明 |
|------|------|------|
| 涨停家数 | 30% | >80 高潮，<20 冰点 |
| 跌停家数 | 25% | >50 恐慌 |
| 涨跌比 | 20% | >3 强势 |
| 北向资金 | 15% | >50 亿看好 |
| 炸板率 | 10% | >40% 退潮 |

**情绪周期四阶段**:
- **高潮期** (≥70 分): 止盈为主，不追高
- **强势期** (50-69 分): 积极操作
- **震荡期** (30-49 分): 谨慎操作
- **冰点期** (<30 分): 空仓/试错首板

### 2. 持仓股监控 (权重 60%)

| 指标 | 说明 |
|------|------|
| 实时价格 | 涨跌幅 |
| 开盘价 | 最高/最低价 |
| 持仓盈亏 | 盈亏比例 |
| 资金流向 | 超大单/大单/中单/小单 |

## ⚠️ 预警系统

### 一级预警 (极度危险) 🔴🔴🔴

**触发条件**:
- 跌停 (≤-9.5%)
- 低开/下跌超 -8%

**应对**: 立即止损，能卖出就卖出

### 二级预警 (高风险) 🔴🔴

**触发条件**:
- 低开/下跌超 -5%
- 超大单流出 >5000 万

**应对**: 准备止损，等待反抽

### 三级预警 (注意) 🔴

**触发条件**:
- 跌破开盘价且下跌

**应对**: 减仓，设置止损

### 机会预警 🟢

**触发条件**:
- 涨停 (≥9.5%)
- 大涨超 5%

**应对**: 持有，观察封单/准备打板
