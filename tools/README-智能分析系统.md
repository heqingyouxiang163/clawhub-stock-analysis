# 智能股票分析系统

## 🎯 核心功能

**根据当前时间自动判断使用盘中逻辑还是盘后逻辑**，无需手动选择！

---

## ⏰ 时间判断逻辑

### 交易日 (周一至周五)

| 时间段 | 模式 | 分析逻辑 | 内容 |
|--------|------|----------|------|
| **00:00-09:00** | 盘前准备 | 盘后逻辑 | 日 K 线、形态识别、盘前预案 |
| **09:00-09:25** | 集合竞价前 | 盘后逻辑 | 昨日回顾、观察要点、预警位 |
| **09:25-11:30** | 盘中监控 (早盘) | **盘中逻辑** | 实时数据、分时图、资金流向、盘中选股 |
| **11:30-13:00** | 午间休市 | 盘中逻辑 | 上午复盘、午后预案 |
| **13:00-15:00** | 盘中监控 (午后) | **盘中逻辑** | 实时数据、分时图、涨停潜力、尾盘决策 |
| **15:00-20:00** | 盘后分析 (傍晚) | 盘后逻辑 | 日 K 线、涨停质量、资金流向、次日预案 |
| **20:00-24:00** | 盘后复盘 (晚间) | 盘后逻辑 | 深度复盘、龙虎榜、周计划 |

### 非交易日 (周末/节假日)

| 时间段 | 模式 | 分析逻辑 |
|--------|------|----------|
| **全天** | 盘后复盘 | 盘后逻辑 |

## 📊 两种分析逻辑对比

### 盘中逻辑 (09:25-15:00)

**核心**: 实时监控、动态决策

**数据源**:
- 实时行情 (每秒更新)
- 分时图数据
- 实时资金流向
- 集合竞价数据

**分析内容**:
- ✅ 分时图战法 (股价 vs 均线)
- ✅ 实时涨跌幅
- ✅ 盘中预警 (三级预警)
- ✅ 资金流向 (超大单/大单)
- ✅ 盘中选股 (动态筛选)
- ✅ 涨停潜力判断

**输出**:
```
【实时数据】
  现价：XX 元 (XX%)
  开盘：XX 元
  最高：XX 元
  最低：XX 元

🔥【分时图战法】
  形态评级：🟢🟢🟢 极强
  形态评分：92/100
  概率预测：涨停概率 70%+

⚠️【预警】
  🟢 机会预警：大涨超 5%，持有！
```

### 盘后逻辑 (15:00-次日 09:25)

**核心**: 深度分析、形态识别、预案制定

**数据源**:
- 日 K 线数据
- 收盘数据
- 龙虎榜数据
- 资金流向统计

**分析内容**:
- ✅ 日 K 线形态识别
- ✅ 涨停质量评估 (A/B/C 级)
- ✅ 五日线首阴反包
- ✅ 低开跌停风险
- ✅ 技术指标 (MA/MACD/RSI)
- ✅ 次日操作预案

**输出**:
```
【基础数据】
  收盘价：XX 元 (XX%)
  持仓盈亏：+XX 元 (+XX%)

【技术指标】
  MA5: XX 元
  MA10: XX 元
  均线排列：🟢 多头排列

【形态识别】
  🔥 涨停板
  🟡 五日线首阴形态

【次日预案】
  IF 高开 5-8% → 持有观望
  IF 高开>8% → 持有，冲高止盈
  IF 低开 → 反抽卖出
```

## 🤖 自动判断逻辑

### Python 代码实现

```python
def get_current_mode():
    now = datetime.now()
    current_time = now.time()
    weekday = now.weekday()
    
    # 非交易日
    if weekday >= 5:
        return {"mode": "盘后复盘", "logic": "盘后"}
    
    # 盘中时段 (使用盘中逻辑)
    if time(9, 25) <= current_time < time(11, 30):
        return {"mode": "盘中监控 (早盘)", "logic": "盘中"}
    
    if time(13, 0) <= current_time < time(15, 0):
        return {"mode": "盘中监控 (午后)", "logic": "盘中"}
    
    # 其他时段 (使用盘后逻辑)
    return {"mode": "盘后分析", "logic": "盘后"}
```
