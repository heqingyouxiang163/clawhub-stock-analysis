# 持仓分析自动化系统

## 📋 系统功能

每日自动分析持仓股的**成交量**和**多空情绪**，生成次日操作建议。

## ⏰ 运行时间

**每周一至周五 9:25** (集合竞价结束后，开盘前 5 分钟)

- ✅ 基于真实开盘价（不是预测）
- ✅ 可见集合竞价封单金额
- ✅ 可见竞价成交量
- ✅ 更准确的低开跌停风险判断
- ✅ 还有 5 分钟准备开盘操作

## 📊 分析内容

### 1. 成交量指标 (40 分)
- 量比 (当日成交量 / 20 日均量)
- 量价关系 (量增价涨/量缩价涨/量增价跌等)
- 成交量状态 (天量/地量/正常)
- 成交量趋势 (连续放量/缩量)

### 2. 技术面指标 (25 分)
- 均线排列 (多头/空头/纠缠)
- MACD (金叉/死叉)
- RSI (超买/超卖/中性)
- 布林带位置

### 3. 情绪面指标 (35 分) 🎯 核心
- **超大单净流入** (25 分) ← 最重要
- **大单净流入** (15 分) ← 核心参考
- 北向资金流向 (15 分)
- 主力 - 散户差值 (10 分)
- 市场涨跌比 (10 分)

## 📈 评分体系

| 总分 | 评级 | 仓位建议 |
|------|------|----------|
| 80-100 | 🟢🟢 强烈看好 | 50-70% |
| 60-80 | 🟢 看好 | 70-90% |
| 40-60 | 🟡 中性 | 50-70% |
| 20-40 | 🔴 看空 | 20-40% |
| 0-20 | 🔴🔴 强烈看空 | 0-20% |

## 📝 输出内容

每只股票的分析报告包括：

1. **基础数据**: 现价、成本、盈亏
2. **成交量分析**: 量比、量价关系、趋势
3. **技术分析**: 均线、MACD、RSI、布林带
4. **情绪分析**: 超大单、大单、北向资金、主力 - 散户差值
5. **综合评分**: 成交量 + 技术面 + 情绪面
6. **评级建议**: 持有/加仓/减仓/止损
7. **当日预案**: IF-THEN 操作规则

## 📅 定时任务

已设置 cron 任务，**每交易日 9:25** 自动运行。

**为什么是 9:25？**
- 集合竞价刚结束 (9:15-9:25)
- 可以看到真实开盘价、封单、竞价量
- 比 9:15 更准确，比 9:30 更从容
- 还有 5 分钟准备开盘操作

查看任务状态:
```bash
openclaw cron list
```

手动触发:
```bash
openclaw cron run <job-id>
```

## ⚠️ 注意事项

1. **数据源稳定性**: AKShare 接口偶尔不稳定，脚本有 3 次重试机制
2. **网络问题**: 如遇网络错误，报告会标注数据获取失败
3. **周末/节假日**: cron 表达式设置为仅工作日运行 (1-5)
4. **数据延迟**: 使用前一日收盘数据，不影响盘前分析

## 🎯 使用建议

1. **每日 9:15 查看**: 开盘前查看报告，了解持仓股状态
2. **执行预案**: 按照 IF-THEN 预案执行，不情绪化交易
3. **纪律止损**: 触及止损位无条件执行，不抱侥幸心理
4. **定期复盘**: 每周/每月回顾评分准确性，优化参数
