## 📊 盈利模式

### 核心逻辑

```
胜率：60%+
盈亏比：1:2 (亏 3% 赚 6%)
月交易次数：20-30 次
月盈利目标：20-30%
```

### 示例计算

```
假设每月交易 25 次：
- 盈利 15 次 (60%) × 平均 6% = +90%
- 亏损 10 次 (40%) × 平均 3% = -30%
- 净利润：+60%

考虑仓位控制 (平均 30% 仓)：
实际月盈利：60% × 30% = 18%

加上复利效应：
月盈利 20-30% 可实现
```

### 成功关键

1. **严格止损** (-3% 无条件止损)
2. **及时出货** (次日 9:30-10:00 必须出货)
3. **仓位控制** (单只≤30%，总仓≤60%)
4. **选股质量** (≥80 分才推荐)
5. **心态管理** (宁可空仓，不做弱势)


### 2. 自动推送 (定时任务)

```bash
# 设置 9:35-10:35 的定时任务
for minute in 35 40 45 50 55 60; do
  hour=9
  [ $minute -ge 60 ] && hour=10 && minute=$((minute-60))
  openclaw cron add --name="短线盈利-${hour}:${minute}" \
    --schedule="cron:${minute} ${hour} * * 1-5" \
    --command="python3 /home/admin/openclaw/workspace/tools/短线盈利助手.py" \
    --timeout=120
done
```

### 3. 查看推送
