### 2️⃣ 路径自动转换

**新增函数**: `get_absolute_path()`

**功能**:
- ✅ 自动将 `~` 路径转换为绝对路径
- ✅ 启动时验证所有路径
- ✅ 避免路径错误导致失败

**代码**:
```python
def get_absolute_path(path):
    """将 `~` 路径转换为绝对路径"""
    if path.startswith("~/"):
        return "/home/admin" + path[1:]
    elif path.startswith("~/."):
        return "/home/admin" + path[2:]
    return path

# 使用
BASE_DIR = get_absolute_path("~/openclaw/workspace")
# 结果：/home/admin/openclaw/workspace
```
