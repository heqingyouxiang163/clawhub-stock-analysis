# 数据校验层技能 v1.0

## 功能描述
对所有获取的股票数据进行合理性校验，防止错误数据影响决策。

---

## 校验规则

### A 股硬约束 (必须遵守)

| 规则 | 范围 | 说明 |
|------|------|------|
| **涨跌幅** | -10% to +10% | 涨跌停限制 (ST 股±5%) |
| **换手率** | 0% to 50% | 正常范围，>50% 可疑 |
| **成交量** | >0 | 不能为负 |
| **成交额** | >0 | 不能为负 |
| **现价** | >0 | 不能为负 |
| **最高价** | >= 现价 | 最高价不能低于现价 |
| **最低价** | <= 现价 | 最低价不能高于现价 |
| **开盘价** | -10% to +10% vs 昨收 | 开盘限制 |

---

## 校验流程

### Step 1: 获取数据

```
数据源：MCP API / 雪球浏览器
```

### Step 2: 合理性校验

```python
def validate_stock_data(data):
    errors = []
    
    # 涨跌幅校验
    if data['change_pct'] > 10:
        errors.append(f"涨跌幅 {data['change_pct']}% 超过 10% 限制")
    if data['change_pct'] < -10:
        errors.append(f"涨跌幅 {data['change_pct']}% 低于 -10% 限制")
    
    # 换手率校验
    if data['turnover_rate'] > 50:
        errors.append(f"换手率 {data['turnover_rate']}% 过高，可能数据错误")
    if data['turnover_rate'] < 0:
        errors.append(f"换手率 {data['turnover_rate']}% 不能为负")
    
    # 价格校验
    if data['high'] < data['price']:
        errors.append(f"最高价 {data['high']} < 现价 {data['price']}")
    if data['low'] > data['price']:
        errors.append(f"最低价 {data['low']} > 现价 {data['price']}")
    
    # 量价校验
    if data['volume'] <= 0:
        errors.append(f"成交量 {data['volume']} 不能<=0")
    if data['amount'] <= 0:
        errors.append(f"成交额 {data['amount']} 不能<=0")
    
    return errors
```

### Step 3: 错误处理

| 错误等级 | 处理方式 |
|----------|----------|
| **A 类** (涨跌幅>10%) | 立即标记错误，切换数据源重新获取 |
| **B 类** (换手率>50%) | 标记可疑，多数据源对比 |
| **C 类** (其他) | 记录日志，继续处理 |

### Step 4: 多数据源对比

```
数据源 1: MCP API
数据源 2: 雪球浏览器

if abs(数据 1 - 数据 2) > 5%:
    标记为"数据不一致"
    使用雪球数据 (优先级高)
    记录差异日志
```

## 数据源优先级

| 数据类型 | 首选 | 备选 | 验证 |
|----------|------|------|------|
| **实时价格** | 雪球浏览器 | MCP API | 涨跌幅校验 |
| **涨跌幅** | 雪球浏览器 | MCP API | ±10% 校验 |
| **换手率** | 雪球浏览器 | MCP API | 0-50% 校验 |
| **成交量** | 雪球浏览器 | MCP API | >0 校验 |
| **主力流向** | 雪球浏览器 | 东方财富 | 合理性校验 |
