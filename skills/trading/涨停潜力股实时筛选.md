# 涨停潜力股实时筛选技能 v1.0

## 场景
当用户需要筛选**可买入的涨停潜力股**时（非已涨停），使用此技能进行实时筛选和推荐。

---

## 核心流程

### Step 1: 数据源获取（按优先级）

| 优先级 | 数据源 | 耗时 | 验证码 |
|--------|--------|------|--------|
| **1️⃣** | **雪球浏览器** | 3-5 秒 | ✅ 无 |
| **2️⃣** | **MCP API** | 1-2 秒 | ✅ 无 |
| **3️⃣** | 东方财富浏览器 | 5-10 秒 | ❌ 有 |

**雪球 URL 格式**:
```
https://xueqiu.com/hq/detail?order=desc&orderBy=percent&type=sha&market=CN
```

**MCP API 命令**:
```bash

```

---

### Step 2: 初步筛选条件

**涨幅范围**: `+5% 至 +9%`（排除已涨停的）

**排除条件**:
- ❌ 涨幅 ≥ +9.5%（已涨停或接近涨停，买不进去）
- ❌ 涨幅 < +5%（强度不足）
- ❌ 市值 < 30 亿 或 > 500 亿（太小或太大）
- ❌ 市盈率 < 0（亏损股，风险高）

**保留条件**:
- ✅ 涨幅 +5% 至 +9%
- ✅ 市值 50-300 亿（最佳区间）
- ✅ 成交额 > 5 亿（流动性充足）

| **弱转强板** | 80%+ | 昨日烂板→今日高开 3-5%→快速封板 | 昨日烂板，今日高开 |
| **回封真龙板** | 78%+ | 首次封板→炸板→5 分钟内回封 | 炸板后快速回封 |
| **二板加速板** | 70%+ | 昨日涨停→今日高开>5%→秒板 | 连板股 |
| **反包板** | 68%+ | 昨日阴线→今日高开→放量反包 | 昨日下跌，今日反包 |

#### 核心要素对照表

| 要素 | 标准 | 权重 | 说明 |
|------|------|------|------|
| **换手率** | 8-15% | ⭐⭐⭐⭐⭐ | **最关键指标** |
| **主力流入** | >1 亿 | ⭐⭐⭐⭐⭐ | 主力强度 |
| **振幅** | >7% | ⭐⭐⭐ | 波动性 |
| **成交额** | >5 亿 | ⭐⭐⭐⭐ | 流动性 |
| **市值** | 50-300 亿 | ⭐⭐⭐⭐ | 适中最好 |
| **量比** | >2 | ⭐⭐⭐ | 放量程度 |
| **板块助攻** | ≥3 家 | ⭐⭐⭐⭐ | 板块效应 |

#### 分时形态匹配

| 形态 | 特征 | 胜率 | 操作 |
|------|------|------|------|
| **阶梯上涨** | 上涨→盘整→上涨→盘整 | 85%+ | ✅ 突破盘整买入 |
| **直线拉升** | >70 度角，5-15 分钟 | 80%+ | ✅ 拉升初期追入 |
| **高位横盘** | 涨幅>5% 后横盘>1 小时 | 70%+ | ✅ 横盘不破持有 |
| **V 型反转** | 下跌→快速反转→上涨 | 75%+ | ✅ 反转确认买入 |
| **W 底** | 第二低≈第一低，第二底缩量 | 72%+ | ✅ 第二底确认买入 |

#### 禁止分时形态 ❌

| 形态 | 特征 | 操作 |
|------|------|------|
| **阶梯下跌** | 下跌→盘整→下跌→盘整 | ❌ 不抄底 |
| **直线跳水** | >70 度角下跌 | ❌ 立即卖出 |
| **M 顶** | 第二高≈第一高，第二顶缩量 | ❌ 卖出 |
| **尾盘跳水** | 14:30 后放量跳水 | ❌ 清仓 |

### Step 5: 评分体系

#### 综合评分公式

```
总分 = 换手率 (25 分) + 主力流入 (25 分) + 振幅 (10 分) + 成交额 (15 分) + 市值 (10 分) + 形态匹配 (15 分)
```

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| **换手率** | 25 分 | 8-15%=25 分，5-8%=20 分，15-20%=15 分，>20%=5 分 |
| **主力流入** | 25 分 | >3 亿=25 分，1-3 亿=20 分，0.5-1 亿=10 分，<0.5 亿=0 分 |
| **振幅** | 10 分 | >10%=10 分，7-10%=8 分，5-7%=5 分，<5%=0 分 |
| **成交额** | 15 分 | >20 亿=15 分，10-20 亿=12 分，5-10 亿=8 分，<5 亿=0 分 |
| **市值** | 10 分 | 50-150 亿=10 分，150-300 亿=8 分，30-50 亿=5 分，其他=0 分 |
| **形态匹配** | 15 分 | 符合高胜率形态=15 分，一般=8 分，不符合=0 分 |

#### 评级标准

| 总分 | 评级 | 涨停概率 | 操作建议 |
|------|------|----------|----------|
| **90-100 分** | S 级 | 85%+ | 重仓 4-5 成 |
| **80-89 分** | A 级 | 75-85% | 中仓 3-4 成 |
| **70-79 分** | B 级 | 65-75% | 轻仓 2-3 成 |
| **60-69 分** | C 级 | 55-65% | 观察 1-2 成 |
| **<60 分** | D 级 | <55% | 放弃 |

## 关键注意事项

### ⚠️ 红线（必须遵守）

1. **换手率>25%** → 直接放弃（高风险）
2. **涨幅≥9.5%** → 已涨停，买不进去
3. **涨幅<5%** → 强度不足，放弃
4. **市值<30 亿或>500 亿** → 放弃（太小或太大）
5. **主力流出** → 放弃（无主力）

### ✅ 黄金标准

1. **换手率 8-15%** → 完美区间
2. **主力流入>1 亿** → 强度足够
3. **市值 50-150 亿** → 最佳区间
4. **成交额>10 亿** → 流动性好
5. **振幅>7%** → 波动性足够

### 📚 形态优先级

1. **分歧转一致板** (75%+) → 最常见

3. **首板龙头板** (72%+) → 需要板块数据
4. **弱转强板** (80%+) → 需要昨日数据

## 核心要点总结

### 🎯 筛选流程

```
1. 雪球获取实时数据（3-5 秒）
2. 筛选涨幅 +5-9% 的股票
3. 对照 10 种涨停形态
4. 对照分时图战法要素
5. 综合评分（满分 100 分）
6. 按评级给出操作建议
```

### 📊 评分权重

| 指标 | 权重 | 说明 |
|------|------|------|
| 换手率 | 25% | **最关键**，8-15% 完美 |
| 主力流入 | 25% | 强度指标 |
| 成交额 | 15% | 流动性 |
| 形态匹配 | 15% | 涨停形态 |
| 市值 | 10% | 适中最好 |
| 振幅 | 10% | 波动性 |

### 🦞 龙虾提示

**换手率是核心中的核心！**
- 8-15% = 黄金区间（25 分）
- >20% = 高风险（5 分）
- >25% = 直接放弃（0 分）

**根据实战经验，换手率过高的股票（如 49.61%），即使主力流入再强，也不建议参与！**
