# 龙头股识别技能 v1.0

## 技能描述
识别市场龙头股，区分真龙头和伪龙头，提高打板胜率。

## 真龙头核心特征

### ✅ 五大核心特征
| 特征 | 说明 | 权重 |
|------|------|------|
| 板块最先涨停 | 同板块中第一个封板 | ⭐⭐⭐⭐⭐ |
| 封单最强 | 封单金额/封成比最高 | ⭐⭐⭐⭐ |
| 连板数最高 | 市场连板高度标杆 | ⭐⭐⭐⭐⭐ |
| 带动板块效应 | 涨停后板块跟风≥3 家 | ⭐⭐⭐⭐⭐ |
| 抗跌性最强 | 分歧日最后开板/最先回封 | ⭐⭐⭐⭐ |

### ✅ 辅助特征
- 成交量适中（换手 8-15%）
- 市值适中（50-300 亿）
- 题材纯正（核心受益）
- 股性活跃（历史涨停多）

## 伪龙头识别（❌ 禁止打板）

| 类型 | 特征 | 风险 |
|------|------|------|
| 跟风板 | 板块涨停后第 4 个+封板 | 易炸板 |
| 尾盘偷鸡板 | 14:30 后封板 | 次日低开 |
| 无板块助攻 | 独苗涨停 | 持续性差 |
| 缩量秒板 | 换手<5% | 获利盘未出清 |
| 高位加速板 | 5 板以上加速 | 监管风险 + 获利盘 |

## 龙头股打分表

| 条件 | 分值 | 说明 |
|------|------|------|
| 板块第 1 个涨停 | +30 分 | 核心指标 |
| 封单>2 亿 | +20 分 | 强度指标 |
| 连板数市场最高 | +25 分 | 地位指标 |
| 板块涨停≥5 家 | +15 分 | 效应指标 |
| 换手 8-15% | +10 分 | 健康指标 |
| **总分≥80 分** | **真龙头** | 可打板 |
| 总分 60-79 分 | 疑似龙头 | 观察 |
| 总分<60 分 | 伪龙头 | 放弃 |

## 使用方式
```
1. 涨停后对照特征打分
2. ≥80 分确认真龙头
3. 结合情绪周期决定仓位
4. 次日验证龙头地位
```
