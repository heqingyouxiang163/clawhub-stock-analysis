# 分时图战法 v1.0

## 功能描述
通过分时图识别强势股、主力意图、买卖点。

---

## 分时图核心要素

---

### 白线（分时价）

| 位置 | 含义 | 操作 |
|------|------|------|
| **黄线上方** | 强势 | 持有/买入 |
| **黄线下方** | 弱势 | 观望/卖出 |
| **穿越黄线** | 转强/转弱 | 关注 |

---

### 黄线（分时均价）

| 位置 | 含义 | 操作 |
|------|------|------|
| **向上倾斜** | 多头 | 持有 |
| **向下倾斜** | 空头 | 卖出 |
| **走平** | 震荡 | 观望 |

---

### 成交量柱

| 形态 | 含义 | 操作 |
|------|------|------|
| **红柱放大** | 主力买入 | 跟随 |
| **绿柱放大** | 主力卖出 | 回避 |
| **红绿交替** | 分歧 | 观望 |
| **缩量** | 无主力 | 观望 |

---

## 强势分时形态（6 种）

---

### 形态 1：阶梯上涨 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 上涨→盘整→上涨→盘整 |
| **角度** | 45-60 度角上涨 |
| **盘整** | 不破前低 |
| **成交量** | 上涨放量，盘整缩量 |
| **胜率** | 85%+ |

**操作**：
```
✅ 突破盘整平台→买入
✅ 回踩不破前低→加仓
✅ 跌破平台→卖出
```

---

### 形态 2：直线拉升 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 几乎垂直上涨 |
| **角度** | >70 度角 |
| **时间** | 5-15 分钟 |
| **成交量** | 巨量 |
| **胜率** | 80%+ |

**操作**：
```
✅ 拉升初期→追入
✅ 拉升末期→不追
✅ 拉升后盘整→持有
```

---

### 形态 3：V 型反转 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 下跌→快速反转→上涨 |
| **时间** | 30-60 分钟完成 |
| **成交量** | 反转时放量 |
| **胜率** | 75%+ |

**操作**：
```
✅ 反转确认→买入
✅ 突破下跌起点→加仓
✅ 跌破低点→止损
```

---

### 形态 4：W 底 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 下跌→反弹→下跌→反弹 |
| **低点** | 第二低≈第一低 |
| **成交量** | 第二底缩量 |
| **胜率** | 72%+ |

**操作**：
```
✅ 第二底确认→买入
✅ 突破颈线→加仓
✅ 跌破低点→止损
```

---

### 形态 5：高位横盘 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 大涨后横盘 |
| **位置** | 涨幅>5% 后 |
| **时间** | 横盘>1 小时 |
| **成交量** | 横盘缩量 |
| **胜率** | 70%+ |

**操作**：
```
✅ 横盘不破→持有
✅ 突破横盘→加仓
✅ 跌破横盘→卖出
```

---

### 形态 6：尾盘拉升 ⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **走势** | 尾盘 30 分钟拉升 |
| **位置** | 14:30 后 |
| **成交量** | 放量 |
| **胜率** | 60%+ |

**操作**：
```
⚠️ 谨慎参与
⚠️ 可能是偷袭板
✅ 配合板块→可参与
❌ 无板块→放弃
```

## 分时买卖点

---

### 买入点（5 个）

| 买点 | 条件 | 胜率 |
|------|------|------|
| **突破买点** | 突破盘整平台 + 放量 | 80%+ |
| **回踩买点** | 回踩不破前低 + 缩量 | 75%+ |
| **反转买点** | V 型反转确认 + 放量 | 75%+ |
| **W 底买点** | 第二底确认 + 缩量 | 72%+ |
| **追涨买点** | 直线拉升初期 + 巨量 | 70%+ |

---

### 卖出点（5 个）

| 卖点 | 条件 | 说明 |
|------|------|------|
| **跌破卖点** | 跌破平台/支撑 | 止损 |
| **M 顶卖点** | 第二顶确认 | 止盈 |
| **跳水卖点** | 直线跳水 | 立即卖出 |
| **尾盘卖点** | 尾盘跳水 | 不持仓过夜 |
| **炸板卖点** | 涨停炸板不回封 | 清仓 |

## 分时监控流程

---

### 9:30-10:00（开盘 30 分钟）

```
【监控重点】
□ 开盘方向（高开/低开）
□ 前 5 分钟走势
□ 成交量变化
□ 是否快速封板

【操作】
✅ 快速拉升→关注
✅ 快速封板→打板
❌ 快速跳水→回避
```

---

### 10:00-11:30（上午盘）

```
【监控重点】
□ 分时形态形成
□ 黄线白线关系
□ 成交量配合
□ 板块联动

【操作】
✅ 阶梯上涨→持有/加仓
✅ 高位横盘→持有
❌ 阶梯下跌→卖出
```

---

### 13:00-14:30（下午盘）

```
【监控重点】
□ 午后走势
□ 是否突破上午高点
□ 成交量变化
□ 炸板风险

【操作】
✅ 突破上午高点→持有
✅ 横盘稳定→持有
❌ 跌破上午低点→卖出
```

---

### 14:30-15:00（尾盘）

```
【监控重点】
□ 尾盘拉升/跳水
□ 是否封板
□ 封单稳定性
□ 次日预期

【操作】
✅ 尾盘封板→持有过夜
✅ 尾盘拉升→谨慎
❌ 尾盘跳水→清仓
```

## 分时案例

---

### 成功案例

```
【案例 1】璞泰来 603659（2026-03-16）
- 9:30 高开 4%
- 9:35 直线拉升（5 分钟）
- 9:40 封板
- 成交量：拉升巨量
- 封单：3.5 亿
- 结果：涨停 +10%

【操作】
✅ 9:35 追涨买入
✅ 封板持有
✅ 次日高开止盈
```

---

```
【案例 2】XXX（2026-02-15）

- 10:30 跳水 -5%
- 成交量：炸板巨量
- 结果：收盘 -3%

【教训】
❌ 炸板后未立即卖出
❌ 等待回封幻想
✅ 应炸板 15 分钟不回封清仓
```
