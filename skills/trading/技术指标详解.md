## 趋势指标

---

### 均线系统（MA）⭐⭐⭐⭐⭐

| 均线 | 周期 | 用途 | 说明 |
|------|------|------|------|
| **MA5** | 5 日 | 超短线 | 短线生命线 |
| **MA10** | 10 日 | 短线 | 短线趋势线 |
| **MA20** | 20 日 | 中线 | 月线/波段线 |
| **MA60** | 60 日 | 中长线 | 季线/决策线 |
| **MA120** | 120 日 | 长线 | 半年线 |
| **MA250** | 250 日 | 超长线 | 年线 |

**用法**：
```
【多头排列】
MA5>MA10>MA20>MA60 → 强势上涨 ✅
操作：持有/加仓

【空头排列】
MA5<MA10<MA20<MA60 → 强势下跌 ❌
操作：空仓/卖出

【金叉】
MA5 上穿 MA10 → 买入信号 ✅
MA10 上穿 MA20 → 买入信号 ✅

【死叉】
MA5 下穿 MA10 → 卖出信号 ❌
MA10 下穿 MA20 → 卖出信号 ❌

【支撑/压力】
股价回踩 MA5/MA10 不破 → 支撑 ✅
股价上攻 MA60/MA120 不过 → 压力 ❌
```

---

### MACD（平滑异同移动平均线）⭐⭐⭐⭐

| 组成 | 说明 |
|------|------|
| **DIF（白线）** | 快线，12 日 EMA-26 日 EMA |
| **DEA（黄线）** | 慢线，DIF 的 9 日 EMA |
| **MACD 柱（红绿柱）** | DIF-DEA 的 2 倍 |

**用法**：
```
【金叉】
DIF 上穿 DEA → 买入信号 ✅
零轴上方金叉 → 强势买入 ✅✅
零轴下方金叉 → 弱势买入 ⚠️

【死叉】
DIF 下穿 DEA → 卖出信号 ❌
零轴上方死叉 → 回调卖出 ⚠️
零轴下方死叉 → 强势卖出 ❌❌

【顶背离】
股价新高，MACD 不新高 → 见顶信号 ❌
操作：减仓/清仓

【底背离】
股价新低，MACD 不新低 → 见底信号 ✅
操作：关注/买入

【红绿柱】
红柱放大 → 多头强势 ✅
红柱缩小 → 多头减弱 ⚠️
绿柱放大 → 空头强势 ❌
绿柱缩小 → 空头减弱 ⚠️
```

---

### KDJ（随机指标）⭐⭐⭐

| 组成 | 说明 | 阈值 |
|------|------|------|
| **K 线（白线）** | 快线 | - |
| **D 线（黄线）** | 慢线 | - |
| **J 线（紫线）** | 超快线 | - |

**用法**：
```
【超买区】
K/D/J>80 → 超买 ⚠️
J>100 → 严重超买 ❌
操作：减仓/卖出

【超卖区】
K/D/J<20 → 超卖 ⚠️
J<0 → 严重超卖 ✅
操作：关注/买入

【金叉】
K 上穿 D → 买入信号 ✅
20 以下金叉 → 强烈买入 ✅✅

【死叉】
K 下穿 D → 卖出信号 ❌
80 以上死叉 → 强烈卖出 ❌❌

【钝化】
连续超买/超卖 → 指标失效 ⚠️
操作：结合其他指标
```

## 量能指标

---

### 量比 ⭐⭐⭐⭐⭐

| 值 | 说明 | 操作 |
|------|------|------|
| **<0.8** | 缩量 | ⚠️ 观望 |
| **0.8-1.2** | 平量 | ⚠️ 观望 |
| **1.2-2.0** | 温和放量 | ✅ 关注 |
| **2.0-3.0** | 明显放量 | ✅ 跟随 |
| **3.0-5.0** | 巨量 | ✅ 强势 |
| **>5.0** | 天量 | ⚠️ 谨慎 |

---

### 换手率 ⭐⭐⭐⭐⭐

| 值 | 说明 | 操作 |
|------|------|------|
| **<1%** | 极度缩量 | ⚠️ 观望 |
| **1-3%** | 缩量 | ⚠️ 观望 |
| **3-5%** | 正常 | ⚠️ 观望 |
| **5-10%** | 活跃 | ✅ 关注 |
| **10-15%** | 高度活跃 | ✅ 跟随 |
| **15-20%** | 过热 | ⚠️ 谨慎 |
| **>20%** | 危险 | ❌ 卖出 |

---

### OBV（能量潮）⭐⭐⭐

**原理**：成交量与价格关系

**用法**：
```
【OBV 上升 + 股价上升】
量价配合 → 强势 ✅
操作：持有

【OBV 上升 + 股价下跌】
底背离 → 主力吸筹 ✅
操作：关注/买入

【OBV 下降 + 股价上升】
顶背离 → 主力出货 ❌
操作：减仓/卖出

【OBV 下降 + 股价下跌】
量价配合 → 弱势 ❌
操作：空仓
```

## 波动指标

---

### BOLL（布林线）⭐⭐⭐⭐

| 组成 | 说明 |
|------|------|
| **上轨** | 20 日 MA+2 倍标准差 |
| **中轨** | 20 日 MA |
| **下轨** | 20 日 MA-2 倍标准差 |

**用法**：
```
【股价在上轨附近】
超买 → 可能回调 ⚠️
突破上轨 → 强势上涨 ✅

【股价在中轨附近】
震荡 → 观望 ⚠️
突破中轨向上 → 买入 ✅

【股价在下轨附近】
超卖 → 可能反弹 ✅
跌破下轨 → 弱势下跌 ❌

【收口】
上下轨收窄 → 变盘前兆 ⚠️
操作：等待突破方向

【开口】
上下轨扩大 → 趋势形成 ✅
操作：跟随趋势
```

---

### RSI（相对强弱指标）⭐⭐⭐

| 值 | 说明 | 操作 |
|------|------|------|
| **>80** | 超买 | ⚠️ 减仓 |
| **50-80** | 强势 | ✅ 持有 |
| **20-50** | 弱势 | ⚠️ 观望 |
| **<20** | 超卖 | ✅ 关注 |

**用法**：
```
【RSI>80】
超买 → 可能回调 ⚠️
操作：减仓

【RSI<20】
超卖 → 可能反弹 ✅
操作：关注

【顶背离】
股价新高，RSI 不新高 → 见顶 ❌
操作：卖出

【底背离】
股价新低，RSI 不新低 → 见底 ✅
操作：买入
```

## 短线指标组合

---

### 最佳组合（5 个）

```
【趋势判断】
✅ MA5/MA10 多头排列

【动能判断】
✅ MACD 零轴上方金叉

【超买超卖】
✅ KDJ 20-50 区间

【量能判断】
✅ 量比 2-3 + 换手 10-15%

【波动判断】
✅ BOLL 中轨向上突破
```

---

### 买入信号组合

```
【强烈买入】5 个信号共振
□ MA5/MA10 多头排列 ✅
□ MACD 零轴上方金叉 ✅
□ KDJ 20-50 金叉 ✅
□ 量比>2 + 换手>10% ✅
□ BOLL 突破中轨 ✅

【操作】
- 5 个共振 → 重仓 7 成
- 4 个共振 → 中仓 5 成
- 3 个共振 → 轻仓 3 成
- <3 个 → 观望
```

---

### 卖出信号组合

```
【强烈卖出】5 个信号共振
□ MA5/MA10 空头排列 ❌
□ MACD 零轴下方死叉 ❌
□ KDJ 80 以上死叉 ❌
□ 量比>5（天量） ❌
□ BOLL 跌破中轨 ❌

【操作】
- 5 个共振 → 清仓
- 4 个共振 → 减仓 80%
- 3 个共振 → 减仓 50%
- <3 个 → 减仓 20%
```

## 指标陷阱识别

---

### 常见陷阱

| 陷阱 | 说明 | 识别方法 |
|------|------|----------|
| **指标钝化** | 连续超买/超卖失效 | 结合价格判断 |
| **滞后性** | 指标落后于价格 | 结合 K 线形态 |
| **假信号** | 金叉/死叉后反向 | 结合量能验证 |
| **单一指标** | 只用一个指标 | 多指标共振 |

---

### 避免方法

```
✅ 多指标共振（至少 3 个）
✅ 量价配合验证
✅ K 线形态确认
✅ 情绪周期判断
✅ 板块效应验证
❌ 不依赖单一指标
❌ 不盲目相信金叉死叉
❌ 不忽视量能
```

## 指标使用流程

---

### 盘前分析

```
【趋势】
□ MA5/MA10 方向
□ MACD 零轴位置

【量能】
□ 昨日量比
□ 昨日换手率

【波动】
□ BOLL 位置
□ RSI 位置

【输出】
- 趋势：多头/空头/震荡
- 买点：有/无
- 仓位：X 成
```

---

### 盘中监控

```
【每 30 分钟】
□ MA5 是否破位
□ MACD 是否背离
□ KDJ 是否超买/超卖
□ 量比变化
□ BOLL 位置

【操作】
- 指标恶化 → 减仓
- 指标改善 → 加仓
- 指标中性 → 持有
```
