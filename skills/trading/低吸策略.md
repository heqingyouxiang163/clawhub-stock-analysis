# 低吸策略 v1.0

## 功能描述
专注 -3% 至 -7% 区间低吸，支撑位附近介入，博取反弹收益。

---

## 选股条件

### 基础条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **跌幅** | -3% 至 -7% | 超跌区间 |
| **市值** | 30-500 亿 | 放宽，避免错过 |
| **换手率** | 3-25% | 放宽，增加机会 |
| **成交量** | 缩量回调 | 主力未出逃 |

### 技术条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **支撑位** | 5 日/10 日线 | 关键均线支撑 |
| **前期平台** | 突破后回踩 | 支撑有效 |
| **缺口** | 回补不破 | 缺口支撑 |
| **主力流向** | 未大幅流出 | <5000 万流出 |

### 排除条件
| 条件 | 说明 |
|------|------|
| **跌停股** | 避免接飞刀 |
| **连续大跌** | 3 日跌幅>20% 排除 |
| **主力大幅流出** | >1 亿流出排除 |
| **利空公告** | 有利空排除 |
| **ST 股** | 风险过高排除 |

| 信号 | 说明 | 胜率 |
|------|------|------|
| **W 底确认** | 第二次探底回升 | 75% |

| **缩量企稳** | 成交量萎缩后走平 | 65% |

| 位置 | 操作 | 仓位 |
|------|------|------|

| **-5% 附近** | 正式买入 | 2 成 |
| **-7% 附近** | 重仓买入 | 3 成 |

## 止盈止损

### 止盈策略
| 目标 | 操作 | 仓位 |
|------|------|------|
| **+3%** | 止盈 30% | 70% |
| **+5%** | 止盈 50% | 50% |
| **+7%** | 止盈 80% | 20% |
| **涨停** | 持有过夜 | 20% |

### 止损策略
| 条件 | 操作 | 说明 |
|------|------|------|
| **-7%** | 止损 50% | 第一止损位 |
| **-10%** | 清仓 | 最后止损位 |
| **跌破支撑** | 立即止损 | 支撑失效 |
| **主力大幅流出** | 立即止损 | >5000 万 |

### 总仓位

|------|--------|

## 成功率优化

### 提高胜率的因素
| 因素 | 胜率提升 |
|------|----------|
| **板块效应** | +10% (≥3 家涨停) |
| **主力流入** | +10% (>1 亿) |
| **早盘低吸** | +5% (10:30 前) |
| **回踩确认** | +10% (2 次确认) |

### 降低胜率的因素
| 因素 | 胜率降低 |
|------|----------|
| **尾盘低吸** | -10% (14:00 后) |
| **无板块效应** | -10% (独苗) |
| **主力流出** | -15% (>5000 万) |
| **连续大跌** | -20% (3 日>20%) |

## 实战案例

### 成功案例
```
【案例】XX 股份 (2026-03-15)
- 跌幅：-5.2%
- 支撑：10 日线
- 买入：-5% 介入 2 成
- 反弹：+6%
- 止盈：+5% 出局
- 盈利：+10% (2 成仓位)
```

### 失败案例
```
【案例】XX 科技 (2026-03-10)
- 跌幅：-6%
- 支撑：前期平台
- 买入：-6% 介入 3 成
- 跌破：支撑失效
- 止损：-7% 出局
- 亏损：-7% (3 成仓位)

【教训】支撑位跌破应立即止损，不应等待
```
