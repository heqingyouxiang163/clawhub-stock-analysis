# 仓位管理技能 v1.0

## 核心原则

```
1. 单笔最大风险 ≤ 总资产 1%
2. 单票最大仓位 ≤ 30%
3. 板块集中度 ≤ 50%
4. 永远保留 20% 现金
5. 亏损股不加仓
```

## 情绪周期仓位表

| 周期 | 总仓位 | 单只上限 | 操作频率 |
|------|--------|----------|----------|
| 主升期 | 70-80% | 3 成 | 高频 |
| 分歧期 | 30-50% | 2 成 | 中频 |
| 退潮期 | 0-20% | 1 成 | 低频 |
| 冰点期 | 0% | 空仓 | 等待 |

## 持仓股管理

### 盈利股
| 状态 | 操作 |
|------|------|
| +0-5% | 持有观察 |
| +5-10% | 持有，设保护止损 |
| +10% 以上 | 分批止盈，让利润奔跑 |

### 亏损股
| 状态 | 操作 |
|------|------|
| -0-3% | 持有观察 |
| -3-5% | 准备止损 |
| -5% 以上 | 无条件止损 |
| **铁律** | **亏损股不加仓** |

## 加仓规则

### ✅ 允许加仓
- 盈利状态
- 符合预期走势
- 情绪周期主升期
- 加仓后单只≤30%

### ❌ 禁止加仓
- 亏损状态
- 走势弱于预期
- 情绪周期退潮期
- 已满仓单只 30%

## 现金管理

```
【现金用途】
1. 应对波动（避免被动）
2. 捕捉新机会（打板）
3. 补仓优质股（盈利股）
4. 极端行情避险

【现金比例】
- 主升期：20%
- 分歧期：30-50%
- 退潮期：50-80%
- 冰点期：100%
```
