# 止损止盈策略 v1.0

## 功能描述
精细化止损止盈规则，保住利润，控制亏损。

---

## 止损策略

---

### 打板股止损

| 情况 | 止损位 | 操作 | 说明 |
|------|--------|------|------|
| **当日炸板** | -3% | 次日开盘卖出 | 不等待反包 |
| **次日低开** | -2% | 反抽卖出 | 不抱幻想 |
| **次日平开** | 0% | 冲高卖出 | 不连板就走 |
| **次日高开** | +3% | 不连板卖出 | 保住利润 |
| **连续 2 日不连板** | 任意价 | 清仓 | 弱势股 |

**核心原则**：
```
✅ 打板股是短线，不是长线
✅ 不连板就是弱势
✅ 快速止损，不拖泥带水
✅ 保住本金第一
```

---

### 持仓股止损

| 情况 | 止损位 | 操作 | 说明 |
|------|--------|------|------|
| **成本价 -5%** | -5% | 无条件止损 | 铁律 |
| **成本价 -8%** | -8% | 强制止损 | 最后机会 |
| **成本价 -10%** | -10% | 清仓 + 停止交易 3 天 | 严重错误 |

**核心原则**：
```
✅ -5% 是底线，必须止损
✅ -8% 是最后机会，必须走
✅ -10% 说明策略有问题，停止交易反思
✅ 永不补仓摊低成本
```

---

### 特殊止损情况

| 情况 | 操作 | 说明 |
|------|------|------|
| **炸板不回封** | 立即卖出 | >15 分钟不回封 |
| **板块集体走弱** | 减仓/清仓 | 板块涨停减少>50% |
| **大盘跳水** | 减仓/清仓 | 指数跌>2% |
| **突发利空** | 立即卖出 | 个股/行业利空 |
| **龙虎榜机构大卖** | 次日卖出 | 机构出货 |
| **游资集体出货** | 次日卖出 | 多家游资卖出 |

## 止盈策略

---

### 打板股止盈

| 情况 | 止盈位 | 操作 | 说明 |
|------|--------|------|------|
| **连板** | 持有 | 继续持有 | 连板预期 |
| **不连板高开** | +5% | 冲高止盈 | 不贪心 |
| **不连板平开** | 0% | 冲高止盈 | 不连板就走 |
| **不连板低开** | -2% | 反抽止盈 | 反抽走人 |
| **涨停封单减弱** | 持有 | 观察 | 封单减少>50% 减仓 |

**连板股止盈**：
```
【2 连板】
- 继续持有
- 观察封单
- 炸板止盈

【3 连板】
- 继续持有
- 观察板块
- 板块减弱止盈

【4 连板】
- 减仓 50%
- 锁定利润
- 剩余博 5 连板

【5 连板+】
- 清仓
- 不博更高
- 利润最大化
```

---

### 持仓股止盈

| 情况 | 止盈位 | 操作 | 说明 |
|------|--------|------|------|
| **盈利 +10%** | +10% | 止盈 50% | 锁定利润 |
| **盈利 +20%** | +20% | 止盈 80% | 大部分锁定 |
| **盈利 +30%** | +30% | 清仓 | 全部止盈 |
| **趋势破位** | 任意价 | 清仓 | 趋势结束 |

**核心原则**：
```
✅ 分批止盈，不一次性卖光
✅ 锁定利润，不坐过山车
✅ 趋势破位，全部清仓
✅ 宁可少赚，不亏回去
```

## 止损止盈执行流程

---

### 盘前设置

```
【每只股票设置】
□ 止损价（成本价 -5%）
□ 第一止盈价（+10%）
□ 第二止盈价（+20%）
□ 第三止盈价（+30%）
□ 预警价格（提前 1% 提醒）

【记录到交易记录】
- 股票名称/代码
- 买入价格
- 止损价
- 止盈价
- 仓位
```

---

### 盘中监控

```
【每 30 分钟检查】
□ 当前价格 vs 止损价
□ 当前价格 vs 止盈价
□ 封单变化
□ 板块走势
□ 大盘走势

【触发预警】
- 接近止损价 1% → 准备卖出
- 接近止盈价 1% → 准备止盈
- 封单减少>50% → 准备减仓
- 板块走弱 → 准备清仓
```

---

### 执行纪律

```
【止损纪律】
✅ 触及止损价，立即卖出
✅ 不等待反弹
✅ 不抱幻想
✅ 承认错误

【止盈纪律】
✅ 触及止盈价，执行止盈
✅ 不贪心
✅ 锁定利润
✅ 不坐过山车

【禁止行为】
❌ 移动止损价（向下）
❌ 补仓摊低成本
❌ 止盈后追高买回
❌ 止损后报复性交易
```

## 止损止盈案例

---

### 成功案例

```
【案例 1】璞泰来 603659
- 买入价：58.80 元
- 止损价：55.86 元（-5%）
- 第一止盈：64.68 元（+10%）
- 第二止盈：70.56 元（+20%）
- 结果：次日涨停 64.68 元，止盈 50%
- 第三日继续涨停 71.15 元，止盈剩余
- 总盈利：+21%

【教训】
✅ 分批止盈正确
✅ 不贪心正确
✅ 锁定利润正确
```

---

### 失败案例

```
【案例 2】XXX
- 买入价：20.00 元
- 止损价：19.00 元（-5%）
- 次日低开 18.50 元（-7.5%）
- 未止损，等待反弹
- 继续下跌至 17.00 元（-15%）
- 最终止损，亏损 15%

【教训】
❌ 未执行止损纪律
❌ 等待反弹是幻想
❌ 亏损扩大
❌ 应 -5% 立即止损
```
