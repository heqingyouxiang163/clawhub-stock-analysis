# 集合竞价监控技能 v1.0

## 功能描述
在交易日 9:20 自动运行，监控集合竞价（9:15-9:25），筛选强势股，9:25 前输出报告。

---

## 运行时间
** cron**: `0 20 9 * * 1-5`（周一至周五 9:20）
**时区**: Asia/Shanghai

---

## 监控流程

### Step 1: 获取竞价数据（9:20-9:22）

**数据源**: 雪球浏览器
```
URL: https://xueqiu.com/hq/detail?order=desc&orderBy=percent&type=sha&market=CN
```

**获取内容**:
- 竞价涨幅前 50 名
- 竞价量比前 50 名
- 竞价成交额前 50 名

---

### Step 2: 筛选条件（9:22-9:23）

| 条件 | 标准 | 说明 |
|------|------|------|
| **竞价涨幅** | +3% 至 +5% | 高开适中，避免一字板 |
| **竞价量比** | >3 | 放量明显 |
| **竞价成交额** | >1000 万 | 流动性充足 |
| **市值** | 50-300 亿 | 适中最好 |
| **昨日涨停** | 是 | 弱转强预期 |

---

### Step 3: 形态识别（9:23-9:24）

**重点形态**:

| 形态 | 特征 | 胜率 |
|------|------|------|
| **弱转强板** | 昨日烂板 + 今日高开 3-5% + 量比>3 | 80%+ |
| **二板加速板** | 昨日首板 + 今日高开>5% + 秒板 | 70%+ |
| **反包板** | 昨日阴线 + 今日高开>2% + 放量 | 68%+ |

---

### Step 4: 输出报告（9:24-9:25）

**报告格式**:
```markdown
## 📊 集合竞价监控报告 (MM-DD 9:25)

### 🎯 重点推荐（1-3 只）

#### 1️⃣ [股票名] ([代码])
| 指标 | 数值 | 标准 |
|------|------|------|
| 竞价涨幅 | +X.XX% | +3-5% |
| 竞价量比 | X.X | >3 |
| 竞价成交额 | XXXX 万 | >1000 万 |
| 市值 | XX 亿 | 50-300 亿 |
| 昨日表现 | 涨停/烂板/阴线 | - |
| 形态 | 弱转强/二板/反包 | - |

**操作建议**:
```
💰 9:30 开盘可买入
🎯 涨停价：XX 元
📈 空间：+X%
仓位：X 成
止损：-3%
```

### ⚠️ 观察名单（3-5 只）
[略]

### 📊 市场情绪
- 竞价涨停家数：XX 家
- 竞价跌停家数：XX 家
- 竞价炸板家数：XX 家
```

---

## 红线（禁止参与）

❌ 竞价涨幅>8%（一字板，买不进去）
❌ 竞价涨幅<2%（强度不足）
❌ 竞价量比<2（无量）
❌ 竞价成交额<500 万（流动性差）
❌ 市值<30 亿或>500 亿（太小或太大）
