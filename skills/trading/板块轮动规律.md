# 板块轮动规律 v1.0

## 功能描述
识别主线/支线板块，把握板块切换节奏。

---

## 板块分类

---

### 主线板块 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **涨停家数** | ≥5 家/日，连续 3 日 + |
| **龙头股** | 有明确龙一，连板 3 板+ |
| **板块指数** | 连续上涨，突破平台 |
| **成交量** | 持续放大 |
| **消息催化** | 有政策/行业利好 |
| **持续时间** | 1-4 周 |

**操作策略**：
```
✅ 重仓龙一
✅ 参与龙二龙三
✅ 持有至板块走弱
✅ 不轻易切换
```

---

### 支线板块 ⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **涨停家数** | 2-4 家/日，不连续 |
| **龙头股** | 龙一 1-2 板 |
| **板块指数** | 震荡上涨 |
| **成交量** | 温和放大 |
| **消息催化** | 次要利好 |
| **持续时间** | 3-7 天 |

**操作策略**：
```
✅ 轻仓参与龙一
✅ 快进快出
✅ 不恋战
✅ 主线回归切回主线
```

---

### 轮动板块 ⭐

| 特征 | 说明 |
|------|------|
| **涨停家数** | 1-2 家/日，单日 |
| **龙头股** | 首板，无连板 |
| **板块指数** | 一日游 |
| **成交量** | 单日放大 |
| **消息催化** | 突发消息 |
| **持续时间** | 1 天 |

**操作策略**：
```
⚠️ 观望为主
⚠️ 不追高
⚠️ 等待确认
❌ 不参与
```

### 阶段 4：分化期

## 板块切换信号

---

### 旧主线退潮信号

| 信号 | 阈值 | 操作 |
|------|------|------|
| 涨停家数减少 | 10 家 → 3 家 | 减仓 |
| 龙头炸板 | 龙一炸板 | 清仓 |
| 板块指数跌 | 跌>3% | 清仓 |
| 成交量萎缩 | 萎缩>50% | 清仓 |

---

### 新主线启动信号

| 信号 | 阈值 | 操作 |
|------|------|------|
| 涨停家数增加 | 3 家 → 8 家 | 试错 |
| 新龙头出现 | 3 连板 | 加仓 |
| 板块指数突破 | 突破平台 | 重仓 |
| 成交量放大 | 放大>100% | 重仓 |

## 板块轮动规律

---

### 规律 1：大小轮动

```
【大盘股搭台】
- 银行/保险/券商启动
- 指数上涨
- 持续 3-5 天

【小盘股唱戏】
- 题材股活跃
- 涨停家数增加
- 持续 1-2 周

【操作】
- 大盘股搭台：轻仓观望
- 小盘股唱戏：重仓参与
```

---

### 规律 2：高低切换

```
【高位股见顶】
- 连板股集体杀跌
- 炸板率上升
- 资金流出

【低位股启动】
- 首板股增加
- 低位板块活跃
- 资金流入

【操作】
- 高位股见顶：清仓
- 低位股启动：试错新方向
```

---

### 规律 3：政策驱动

```
【政策发布】
- 国家政策利好
- 相关行业受益
- 持续 1-2 周

【操作】
- 政策发布当日：打板龙一
- 发酵期：参与龙二龙三
- 高潮期：持有
- 退潮期：清仓
```

---

### 规律 4：业绩驱动

```
【财报季】
- 业绩超预期股涨停
- 行业龙头受益
- 持续 2-4 周

【操作】
- 业绩超预期：打板
- 行业龙头：持有
- 业绩暴雷：回避
```

### 板块强度评分

```
【评分公式】
= 涨停家数 (40 分) + 连板数 (20 分) + 成交量 (20 分) + 消息催化 (10 分) + 持续性 (10 分)

【涨停家数】
- ≥10 家：40 分
- 5-9 家：30 分
- 3-4 家：20 分
- 1-2 家：10 分
- 0 家：0 分

【连板数】
- ≥5 板：20 分
- 3-4 板：15 分
- 2 板：10 分
- 1 板：5 分
- 0 板：0 分

【成交量】
- 放大>200%：20 分
- 放大 100-200%：15 分
- 放大 50-100%：10 分
- 放大<50%：5 分
- 缩量：0 分

【消息催化】
- 国家政策：10 分
- 行业利好：7 分
- 公司利好：3 分
- 无消息：0 分

【持续性】
- 连续 5 日+：10 分
- 连续 3-4 日：7 分
- 连续 2 日：3 分
- 单日：0 分

【决策】
- ≥80 分：主线板块，重仓
- 60-79 分：支线板块，轻仓
- <60 分：轮动板块，观望
```
