# 风险控制系统 v1.0

## 功能描述
实时监控风险，防止大幅回撤。

## 风险等级

### 🟢 低风险（正常操作）
| 指标 | 阈值 |
|------|------|
| 单日亏损 | < -2% |
| 连续亏损 | 0-1 笔 |
| 情绪周期 | 主升期 |
| 仓位上限 | 80% |

### 🟡 中风险（降低仓位）
| 指标 | 阈值 |
|------|------|
| 单日亏损 | -2% 至 -5% |
| 连续亏损 | 2 笔 |
| 情绪周期 | 分歧期 |
| 仓位上限 | 50% |

### 🔴 高风险（强制空仓）
| 指标 | 阈值 |
|------|------|
| 单日亏损 | > -5% |
| 连续亏损 | 3 笔 |
| 情绪周期 | 退潮期/冰点期 |
| 仓位上限 | 0-20% |

## 止损规则

### 单笔止损
| 类型 | 止损位 | 操作 |
|------|--------|------|
| 打板股 | -3% | 无条件止损 |
| 持仓股 | -5% | 无条件止损 |
| 炸板股 | 不回封 | 立即卖出 |

### 总止损
| 类型 | 止损位 | 操作 |
|------|--------|------|
| 单日总亏损 | -5% | 停止交易 |
| 单周总亏损 | -10% | 停止交易 3 天 |
| 单月总亏损 | -15% | 停止交易 1 周 |

## 风控检查清单

### 开盘前检查
```
□ 昨日盈亏统计
□ 当前情绪周期
□ 今日仓位上限
□ 持仓股止损位
□ 可用资金
```

### 盘中监控
```
□ 封单变化（每 30 分钟）
□ 炸板预警（实时）
□ 板块轮动（每小时）
□ 大盘情绪（每小时）
```

### 收盘后检查
```
□ 今日盈亏统计
□ 连续亏损计数
□ 总回撤统计
□ 明日仓位调整
```

## 炸板预警

### 炸板信号
| 信号 | 说明 | 操作 |
|------|------|------|
| 封单骤降 | 封单减少>50% | 准备卖出 |
| 反复开板 | 开板≥2 次 | 减仓 50% |
| 板块走弱 | 板块涨停减少 | 观察 |
| 大盘跳水 | 指数跌>1% | 减仓 |

### 回封确认
| 条件 | 说明 | 操作 |
|------|------|------|
| 快速回封 | <5 分钟 | 持有 |
| 慢速回封 | 5-15 分钟 | 减仓 |
| 不回封 | >15 分钟 | 清仓 |

## 异常波动处理

### 个股异常
| 情况 | 操作 |
|------|------|
| 突发利空 | 立即卖出 |
| 停牌公告 | 等待复牌 |
| 业绩暴雷 | 次日开盘卖出 |
| 监管问询 | 减仓 50% |

### 市场异常
| 情况 | 操作 |
|------|------|
| 千股跌停 | 空仓 |
| 指数跌>3% | 仓位<20% |
| 涨停<10 家 | 空仓 |
| 炸板>50% | 空仓 |
