# 主力行为识别 v1.0

## 功能描述
识别主力吸筹、洗盘、拉升、出货行为，跟随主力操作。

---

## 主力操盘四阶段

---

### 阶段 1：吸筹 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 低位（跌幅>30%） |
| **价格** | 小幅上涨/横盘 |
| **成交量** | 温和放量 |
| **时间** | 2-4 周 |
| **K 线** | 小阳小阴交替 |

**识别信号**：
```
✅ 底部放量（成交量放大 2-3 倍）
✅ 价格不跌（主力护盘）
✅ 大单净流入（超大单持续买入）
✅ 龙虎榜机构买入
```

**操作**：
```
✅ 底部放量→买入 3 成
✅ 突破平台→加仓至 5 成
✅ 主升浪→加仓至 7 成
```

---

### 阶段 2：洗盘 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 上涨途中（涨幅 20-50%） |
| **价格** | 回调 10-20% |
| **成交量** | 缩量 50%+ |
| **时间** | 1-2 周 |
| **K 线** | 阴线为主 |

**识别信号**：
```
✅ 回调缩量（成交量萎缩 50%+）
✅ 不破支撑（10 日/20 日线）
✅ 大单未流出（主力未走）
✅ 板块仍强（主线未变）
```

**操作**：
```
✅ 缩量企稳→加仓
✅ 突破洗盘→加仓
❌ 放量下跌→卖出（非洗盘）
```

---

### 阶段 3：拉升 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 主升浪 |
| **价格** | 连续上涨 30%+ |
| **成交量** | 持续放量 |
| **时间** | 1-3 周 |
| **K 线** | 大阳/涨停 |

**识别信号**：
```
✅ 量增价涨（健康量价）
✅ 连板/大阳（主力进攻）
✅ 封单稳定（>2 亿）
✅ 板块助攻（≥5 家涨停）
```

**操作**：
```
✅ 拉升初期→重仓 7 成
✅ 连板确认→持有
✅ 缩量加速→持有
⚠️ 高位放量→减仓
```

---

### 阶段 4：出货 ⚠️

| 特征 | 说明 |
|------|------|
| **位置** | 高位（涨幅>50%） |
| **价格** | 滞涨/震荡 |
| **成交量** | 天量/放量下跌 |
| **时间** | 1-2 周 |
| **K 线** | 长上影/大阴 |

**识别信号**：
```
❌ 高位天量（成交量 5 倍 +）
❌ 滞涨/长上影（主力出货）
❌ 大单净流出（超大单卖出）
❌ 龙虎榜机构卖出
```

**操作**：
```
❌ 不追高
✅ 持有者止盈
✅ 天量滞涨→清仓
❌ 不博反弹
```

## 主力行为信号

---

### 主力吸筹信号（6 个）

| 信号 | 阈值 | 说明 |
|------|------|------|
| **底部放量** | 成交量放大 2-3 倍 | 主力进场 |
| **价格不跌** | 放量但价格稳定 | 主力护盘 |
| **大单净流入** | 超大单连续 3 日 + 流入 | 主力吸筹 |
| **龙虎榜机构买入** | 机构专用买入 | 机构建仓 |
| **股东人数减少** | 股东人数连续减少 | 筹码集中 |
| **大宗交易溢价** | 大宗交易溢价成交 | 主力抢筹 |

---

### 主力洗盘信号（5 个）

| 信号 | 阈值 | 说明 |
|------|------|------|
| **回调缩量** | 成交量萎缩 50%+ | 主力未走 |
| **不破支撑** | 10 日/20 日线支撑 | 主力护盘 |
| **大单未流出** | 超大单稳定 | 主力锁仓 |
| **板块仍强** | 主线板块未变 | 趋势未变 |
| **快速企稳** | 回调 3-5 日后企稳 | 洗盘结束 |

---

### 主力拉升信号（6 个）

| 信号 | 阈值 | 说明 |
|------|------|------|
| **量增价涨** | 成交量放大 + 价格上涨 | 主力进攻 |
| **连板/大阳** | 连板或大阳 5%+ | 主力强势 |
| **封单稳定** | 封单>2 亿 | 主力锁仓 |
| **板块助攻** | 板块≥5 家涨停 | 主力合力 |
| **龙虎榜游资买入** | 一线游资买入 | 游资接力 |
| **加速上涨** | 角度>60 度 | 主升浪 |

---

### 主力出货信号（7 个）

| 信号 | 阈值 | 说明 |
|------|------|------|
| **高位天量** | 成交量 5 倍 + | 主力出货 |
| **滞涨/长上影** | 高位不涨 | 主力不做多 |
| **大单净流出** | 超大单连续流出 | 主力卖出 |
| **龙虎榜机构卖出** | 机构专用卖出 | 机构出货 |
| **放量下跌** | 下跌 + 放量 | 主力砸盘 |
| **跌破支撑** | 跌破 10 日/20 日线 | 主力放弃 |
| **板块走弱** | 板块涨停减少 | 主力撤离 |

## 主力类型识别

---

### 机构主力 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **资金量** | 10 亿+ |
| **风格** | 价值投资、趋势配置 |
| **持仓周期** | 数周 - 数月 |
| **操盘手法** | 分批建仓、趋势持有 |
| **识别方法** | 龙虎榜机构专用、大宗交易 |

**跟随策略**：
```
✅ 机构大买→跟随买入
✅ 趋势持有→持有
❌ 机构大卖→清仓
```

---

### 游资主力 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **资金量** | 1-10 亿 |
| **风格** | 短线打板、连板接力 |
| **持仓周期** | 1-5 天 |
| **操盘手法** | 快速封板、连板接力 |
| **识别方法** | 龙虎榜游资席位 |

**跟随策略**：
```
✅ 游资大买→跟随打板
✅ 连板确认→持有
❌ 游资出货→清仓
```

---

### 散户主力 ⭐

| 特征 | 说明 |
|------|------|
| **资金量** | <1 亿 |
| **风格** | 追涨杀跌 |
| **持仓周期** | 1-3 天 |
| **操盘手法** | 跟风、无章法 |
| **识别方法** | 小单净流入、无龙虎榜 |

**跟随策略**：
```
⚠️ 不跟随
⚠️ 散户合力难持续
❌ 散户集中→反向
```

## 主力动向监控

---

### 盘后监控（16:00-17:00）

```
【龙虎榜】
□ 机构买入/卖出
□ 游资席位买入/卖出
□ 机构 + 游资合力

【大宗交易】
□ 大宗交易成交价
□ 折价/溢价
□ 买方席位

【资金流向】
□ 超大单净流入/出
□ 大单净流入/出
□ 散户净流入/出
```

---

### 盘中监控（每 30 分钟）

```
【资金流向】
□ 超大单变化
□ 大单变化
□ 主力净流入/出

【封单变化】
□ 封单金额
□ 封单增减
□ 主力意图

【板块联动】
□ 板块涨停家数
□ 板块资金流向
□ 主力是否合力
```

## 主力行为案例

---

### 成功案例

```
【案例 1】璞泰来 603659（2026-03-16）

【吸筹阶段】2026-02-15 至 2026-03-01
- 底部放量：成交量放大 2 倍
- 价格不跌：横盘震荡
- 大单净流入：连续 5 日 + 流入
- 操作：买入 3 成

【洗盘阶段】2026-03-02 至 2026-03-08
- 回调缩量：成交量萎缩 60%
- 不破支撑：20 日线支撑
- 大单未流出：主力锁仓
- 操作：加仓至 5 成

【拉升阶段】2026-03-09 至 2026-03-16
- 量增价涨：成交量放大 3 倍
- 连板：2 连板
- 封单稳定：3.5 亿
- 操作：持有至 7 成

【结果】
- 吸筹价：50 元
- 拉升价：58.80 元
- 盈利：+17.6%
```

---

### 失败案例

```
【案例 2】XXX（2026-02-15）

【拉升阶段】2026-01-15 至 2026-02-10
- 量增价涨：成交量放大 3 倍
- 连板：5 连板
- 涨幅：+80%

【出货阶段】2026-02-11 至 2026-02-15
- 高位天量：成交量 5 倍
- 滞涨：长上影
- 大单净流出：连续 3 日流出
- 操作：未识别，未止盈

【结果】
- 最高价：100 元
- 出货价：70 元
- 亏损：-30%（坐过山车）

【教训】
❌ 高位天量未识别
❌ 滞涨未止盈
❌ 大单流出未关注
✅ 应天量滞涨清仓
```

## 主力行为评分

---

### 主力强度评分

```
【评分公式】
= 资金流向 (30 分) + 量价关系 (25 分) + 龙虎榜 (20 分) + 板块效应 (15 分) + 趋势 (10 分)

【资金流向】
- 超大单净流入>5000 万：30 分
- 超大单净流入 1000-5000 万：20 分
- 超大单净流入<1000 万：10 分
- 超大单净流出：0 分

【量价关系】
- 量增价涨：25 分
- 量缩价涨：15 分
- 量缩价跌：10 分
- 量增价跌：0 分

【龙虎榜】
- 机构 + 游资大买：20 分
- 机构大买：15 分
- 游资大买：10 分
- 机构/游资大卖：0 分

【板块效应】
- 板块≥10 家涨停：15 分
- 板块 5-10 家涨停：10 分
- 板块 3-5 家涨停：5 分
- 板块<3 家涨停：0 分

【趋势】
- 主升期：10 分
- 分歧期：5 分
- 退潮期/冰点期：0 分

【决策】
- ≥80 分：主力强烈做多→重仓 7 成
- 60-79 分：主力做多→中仓 5 成
- 40-59 分：主力观望→轻仓 3 成
- <40 分：主力做空→空仓
```
