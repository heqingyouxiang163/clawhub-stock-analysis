# 半路策略 v1.0

## 功能描述
专注涨幅 3-7% 时介入，突破关键价位后追击，博取涨停收益。

---

## 选股条件

### 基础条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **涨幅** | 3-8% | 放宽，增加机会 |
| **市值** | 30-500 亿 | 放宽，避免错过 |
| **换手率** | 3-25% | 放宽，增加机会 |
| **成交量** | 持续放大 | 量价配合 |

### 技术条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **突破** | 关键价位 | 前高/平台/均线 |
| **分时图** | 阶梯上涨 | 上涨→盘整→上涨 |
| **主力流入** | >1 亿 | 主力强势 |
| **板块效应** | ≥3 家涨停 | 有板块助攻 |

### 排除条件
| 条件 | 说明 |
|------|------|
| **涨幅>8%** | 接近涨停，空间不足 |
| **缩量拉升** | 量价背离，风险高 |
| **主力流出** | >5000 万流出排除 |
| **独苗股** | 无板块效应排除 |
| **尾盘拉升** | 14:00 后排除 |

## 买入时机

### 分时图信号
| 信号 | 说明 | 胜率 |
|------|------|------|
| **突破平台** | 横盘后突破 | 75% |
| **阶梯上涨** | 上涨→盘整→上涨 | 80% |
| **回踩确认** | 突破后回踩不破 | 70% |
| **大单买入** | 连续大单买入 | 75% |

### 买入点位
| 涨幅 | 操作 | 仓位 |
|------|------|------|
| **3-4%** | 试探性买入 | 1 成 |
| **5-6%** | 正式买入 | 2 成 |
| **7%** | 重仓买入 | 3 成 |

## 止盈止损

### 止盈策略
| 目标 | 操作 | 仓位 |
|------|------|------|
| **涨停** | 持有过夜 | 100% |
| **+8%** | 止盈 50% | 50% |
| **+10%** | 止盈 80% | 20% |
| **炸板** | 立即止盈 | 0% |

### 止损策略
| 条件 | 操作 | 说明 |
|------|------|------|
| **-3%** | 止损 50% | 第一止损位 |
| **-5%** | 清仓 | 最后止损位 |
| **跌破突破位** | 立即止损 | 突破失效 |
| **主力大幅流出** | 立即止损 | >1 亿 |

## 仓位管理

### 单笔仓位
| 市场情绪 | 仓位上限 |
|----------|----------|
| **主升期** | 4 成 |
| **分歧期** | 2 成 |
| **退潮期** | 1 成 |
| **冰点期** | 空仓 |

### 总仓位
| 情况 | 总仓位 |
|------|--------|
| **连续盈利 3 笔** | +1 成 (上限 5 成) |
| **连续亏损 3 笔** | -1 成 (下限 1 成) |
| **单日亏损>-5%** | 停止交易 1 天 |

## 成功率优化

### 提高胜率的因素
| 因素 | 胜率提升 |
|------|----------|
| **板块龙一** | +15% (最早涨停) |
| **主力流入** | +10% (>2 亿) |
| **早盘半路** | +10% (10:30 前) |
| **突破确认** | +10% (2 次确认) |

### 降低胜率的因素
| 因素 | 胜率降低 |
|------|----------|
| **尾盘半路** | -15% (14:00 后) |
| **无板块效应** | -15% (独苗) |
| **主力流出** | -20% (>5000 万) |
| **缩量拉升** | -25% (量价背离) |

## 实战案例

### 成功案例
```
【案例】XX 电子 (2026-03-15)
- 涨幅：5.5%
- 突破：前期平台
- 买入：5.5% 介入 3 成
- 涨停：+10%
- 次日溢价：+5%
- 总盈利：+15% (3 成仓位)
```

### 失败案例
```
【案例】XX 软件 (2026-03-12)
- 涨幅：6%
- 突破：均线压力位
- 买入：6% 介入 2 成
- 回落：跌破突破位
- 止损：-5% 出局
- 亏损：-5% (2 成仓位)

【教训】突破位跌破应立即止损，不应等待反抽
```
