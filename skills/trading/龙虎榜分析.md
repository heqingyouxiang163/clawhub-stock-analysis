# 龙虎榜分析技能 v1.0

## 功能描述
每日 17:00 后获取龙虎榜数据，分析游资动向，追踪主力建仓。

---

## 运行时间
** cron**: `0 17 * * 1-5`（交易日 17:00）
**时区**: Asia/Shanghai

---

## 核心逻辑

### 为什么看龙虎榜？

| 信息 | 价值 | 说明 |
|------|------|------|
| **游资席位** | 追踪主力 | 知道谁在买 |
| **买入金额** | 判断强度 | 真金白银 |
| **机构席位** | 判断趋势 | 机构偏好 |
| **营业部联动** | 判断合力 | 多家游资 |

---

## 分析流程

### Step 1: 获取龙虎榜数据（17:00-17:10）

**数据源**: 东方财富/同花顺
```
URL: http://data.eastmoney.com/stock/tradedetail.html
```

**获取内容**:
- 当日龙虎榜股票列表
- 买入/卖出前 5 席位
- 买入/卖出金额

### Step 2: 识别游资动向（17:10-17:20）

**分析维度**:
- 哪些游资在买入？
- 哪些游资在卖出？
- 是否有机构参与？
- 买入金额是否集中？

### Step 3: 判断次日溢价（17:20-17:25）

| 条件 | 次日溢价 | 说明 |
|------|----------|------|
| **知名游资买入>5000 万** | +5%+ | 强势 |
| **机构 + 游资共同买入** | +8%+ | 超强 |
| **多家游资联动** | +5%+ | 合力 |
| **仅散户席位** | 0-3% | 一般 |
| **游资大额卖出** | -3% - | 风险 |

### Step 4: 输出报告（17:25-17:30）

**报告格式**:
```markdown
## 📊 龙虎榜分析 (MM-DD)

### 🎯 重点关注（1-3 只）

#### 1️⃣ [股票名] ([代码])
| 指标 | 数值 |
|------|------|
| 涨停原因 | [主线/消息/技术] |
| 买入金额 | X.XX 亿 |
| 卖出金额 | X.XX 亿 |
| 净买入 | +X.XX 亿 |
| 知名游资 | [席位名] |
| 机构席位 | 有/无 |

**次日预期**: +X%
**操作建议**: 高开可追/低吸/回避

### 📊 游资动向
- 最活跃游资：[席位名]
- 买入最多：[席位名]，X.XX 亿
- 卖出最多：[席位名]，X.XX 亿
```
