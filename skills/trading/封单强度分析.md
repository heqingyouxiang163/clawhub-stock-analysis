# 封单强度分析技能 v1.0

## 技能描述
分析涨停板封单强度，判断封板质量和炸板风险。

## 封单强度分级

### 🟢 强封单（可排队/接力）
| 指标 | 阈值 |
|------|------|
| 封单金额 | >3 亿 |
| 封成比 | >15% |
| 封单/流通市值 | >0.5% |
| 撤单频率 | 低 |

**特征**：一字板/秒板，买一封单稳定
**操作**：排队打板或次日接力
**炸板风险**：<10%

### 🟡 中封单（确认后打板）
| 指标 | 阈值 |
|------|------|
| 封单金额 | 1-3 亿 |
| 封成比 | 10-15% |
| 封单/流通市值 | 0.2-0.5% |
| 撤单频率 | 中等 |

**特征**：T 字板/换手板，封单时有波动
**操作**：确认板块效应后打板
**炸板风险**：10-30%

### 🔴 弱封单（观望）
| 指标 | 阈值 |
|------|------|
| 封单金额 | <1 亿 |
| 封成比 | <10% |
| 封单/流通市值 | <0.2% |
| 撤单频率 | 高 |

**特征**：反复开板，封单频繁变化
**操作**：观望，易炸板
**炸板风险**：>30%

## 封单陷阱识别

### ❌ 虚假封单
- 特征：大单封板后频繁撤单
- 识别：观察 5 分钟封单变化
- 风险：主力诱多出货

### ❌ 尾盘突击封单
- 特征：14:30 后突然大单封板
- 识别：查看涨停时间
- 风险：次日低开概率>70%

### ❌ 对倒封单
- 特征：封单中有明显对倒痕迹
- 识别：龙虎榜验证
- 风险：主力自导自演

## 封成比计算
```
封成比 = 封单量 / 当日成交量 × 100%

示例：
- 封单 10 万手，当日成交 50 万手
- 封成比 = 10/50 × 100% = 20%（强封单）
```

## 使用方式
```
1. 涨停后查看买一封单
2. 计算封成比
3. 对照分级判断强度
4. 结合板块效应决策
```
