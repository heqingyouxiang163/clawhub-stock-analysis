# 尾盘选股策略 v1.0

## 功能描述
14:30-15:00 尾盘时段筛选强势股，博取次日溢价收益。

---

## 选股条件

### 基础条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **涨幅** | 3-7% | 强势但非涨停 |
| **市值** | 50-300 亿 | 适中 |
| **换手率** | 8-15% | 充分换手 |
| **成交量** | 尾盘放大 | 资金进场 |

### 技术条件
| 条件 | 标准 | 说明 |
|------|------|------|
| **日内走势** | 高位横盘 | 强势整理 |
| **均线位置** | 5 日线上方 | 多头排列 |
| **主力流入** | >5000 万 | 主力进场 |
| **板块效应** | ≥2 家涨停 | 有板块助攻 |

### 排除条件
| 条件 | 说明 |
|------|------|
| **已涨停** | 买不进去 |
| **尾盘跳水** | 跌幅>3% 排除 |
| **主力流出** | >5000 万流出排除 |
| **尾盘偷袭拉升** | 14:50 后突然拉升排除 |
| **独苗股** | 无板块效应排除 |

---

## 买入时机

### 时间窗口
| 时间段 | 操作 | 说明 |
|--------|------|------|
| **14:30-14:40** | 观察 | 筛选候选股 |
| **14:40-14:50** | 买入 | 确认强势 |
| **14:50-15:00** | 补仓 | 加仓强势股 |

### 买入信号
| 信号 | 说明 | 胜率 |
|------|------|------|
| **尾盘放量** | 成交量放大 20%+ | 70% |
| **高位横盘** | 涨幅 3-7% 横盘 30 分钟 | 75% |
| **主力流入** | 14:30 后流入>3000 万 | 80% |
| **板块强势** | 板块≥3 家涨停 | 85% |

---

## 止盈止损

### 次日止盈
| 目标 | 操作 | 仓位 |
|------|------|------|
| **高开>3%** | 持有观察 | 100% |
| **高开 0-3%** | 开盘止盈 | 100% |
| **平开** | 持有 5 分钟 | 100% |
| **低开** | 反抽止盈 | 100% |

### 止损策略
| 条件 | 操作 | 说明 |
|------|------|------|
| **低开>-3%** | 反抽止损 | 正常回调 |
| **低开>-5%** | 立即止损 | 弱势明显 |
| **低开<-5%** | 竞价止损 | 严重弱势 |

## 仓位管理

### 单笔仓位
| 市场情绪 | 仓位上限 |
|----------|----------|

| **分歧期** | 2 成 |

| 情况 | 持股上限 |
|------|----------|

## 成功率优化

### 提高胜率因素
| 因素 | 胜率提升 |
|------|----------|
| **板块龙一** | +15% (最早涨停) |
| **主力流入** | +10% (>1 亿) |
| **高位横盘** | +10% (30 分钟+) |
| **次日高开** | +20% (预期) |

### 降低胜率因素
| 因素 | 胜率降低 |
|------|----------|
| **尾盘偷袭** | -20% (14:50 后拉升) |
| **无板块效应** | -15% (独苗) |
| **主力流出** | -20% (>5000 万) |
| **次日低开** | -25% (预期) |

## 实战案例

### 成功案例
```
【案例】XX 股份 (2026-03-15)
14:40 观察：涨幅 5%，高位横盘
14:45 买入：3 成仓 @ 15.0 元
次日：高开 4% @ 15.6 元
止盈：15.6 元
获利：+4% (3 成仓位)
```

### 失败案例
```
【案例】XX 科技 (2026-03-12)
14:45 买入：2 成仓 @ 25.0 元
次日：低开 -3% @ 24.25 元
止损：24.25 元
亏损：-3% (2 成仓位)

【教训】次日低开应果断止损
```
