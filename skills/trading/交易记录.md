# 交易记录系统 v1.0

## 功能描述
记录每笔交易，用于盘后复盘和策略进化。

## 记录字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 交易日期 | YYYY-MM-DD | 2026-03-16 |
| 股票名称 | 标的名称 | 璞泰来 |
| 股票代码 | 6 位代码 | 603659 |
| 操作类型 | 买入/卖出 | 买入 |
| 交易价格 | 成交价 | 58.80 元 |
| 仓位 | 占总资金比例 | 30% |
| 交易理由 | 打板逻辑 | 储能龙一 + 放量换手板 |
| 封单强度 | 强/中/弱 | 强 |
| 情绪周期 | 主升/分歧/退潮/冰点 | 主升期 |
| 置信度 | 0-100% | 92% |

## 结果记录

| 字段 | 说明 | 示例 |
|------|------|------|
| 卖出日期 | YYYY-MM-DD | 2026-03-17 |
| 卖出价格 | 成交价 | 64.68 元 |
| 盈亏比例 | (卖出 - 买入)/买入 | +10% |
| 盈亏金额 | 实际盈亏 | +1764 元 |
| 持仓天数 | 持有天数 | 1 天 |
| 卖出理由 | 止盈/止损/其他 | 止盈 |

## 统计指标

### 每日统计
```
- 交易次数
- 胜率（盈利交易/总交易）
- 平均盈亏
- 最大单笔盈利
- 最大单笔亏损
- 总盈亏
```

### 每周统计
```
- 周交易次数
- 周胜率
- 周总盈亏
- 周最大回撤
- 周最佳交易
- 周最差交易
```

### 每月统计
```
- 月交易次数
- 月胜率
- 月总盈亏
- 月最大回撤
- 月最佳交易
- 月最差交易
- 盈亏比（平均盈利/平均亏损）
```

## 使用方式

### 买入时记录
```
【交易记录 - 买入】
日期：2026-03-16
股票：璞泰来 (603659)
操作：买入
价格：58.80 元
仓位：3 成
理由：储能龙一 + 放量换手板 + 板块 8 家涨停助攻
封单：强 (3.5 亿)
情绪：主升期
置信度：92%
```

### 卖出时记录
```
【交易记录 - 卖出】
日期：2026-03-17
股票：璞泰来 (603659)
操作：卖出
价格：64.68 元
盈亏：+10% (+1764 元)
持仓：1 天
理由：冲高止盈
```

### 盘后统计
```
【2026-03-16 交易统计】
交易次数：3
胜率：100% (3/3)
总盈亏：+30% (浮盈)
最佳交易：璞泰来 (+10%)
最差交易：无亏损
情绪周期：主升期
仓位：70%
```
