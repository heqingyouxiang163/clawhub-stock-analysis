# 股票数据获取技能 v2.0

## 场景
当需要获取 A 股股票实时数据时（价格、涨跌幅、资金流向、换手率等）。

---

## 数据源优先级（按使用场景排序）

### 🚀 **场景 1: 需要实时数据（打板/监控）**

#### 1️⃣ **雪球浏览器**（首选，实时 + 无验证码）

**命令**:
```bash
# 使用 browser 工具导航
browser(action=navigate, targetUrl="https://xueqiu.com/S/SZ<股票代码>")
browser(action=snapshot, targetId="<会话 ID>", refs="aria")
```

**示例**:
```javascript
// 查询贝肯能源
browser(action=navigate, targetUrl="https://xueqiu.com/S/SZ002828")
browser(action=snapshot, targetId="B4625F8DD88C0FE15C3A08F2ACCB32AC")

// 查询巨力索具
browser(action=navigate, targetUrl="https://xueqiu.com/S/SZ002342")
browser(action=snapshot, targetId="B4625F8DD88C0FE15C3A08F2ACCB32AC")
```

**URL 格式**:
```
https://xueqiu.com/S/SZ<股票代码>  // 深交所
https://xueqiu.com/S/SH<股票代码>  // 上交所
```

**数据提取** (从 snapshot 中):
```
最新价：strong [ref=e129]: ¥14.26
涨跌幅：generic [ref=e130]: +0.38 +2.74%
最高：cell "最高：14.91"
最低：cell "最低：14.15"
今开：cell "今开：14.47"
昨收：cell "昨收：13.88"
成交量：cell "成交量：33.62 万手"
成交额：cell "成交额：4.87 亿"
换手率：cell "换手：17.37%"
振幅：cell "振幅：5.48%"
量比：cell "量比：1.42"
市盈率：cell "市盈率 (动)：74.09"
总市值：cell "总市值：28.66 亿"
```

**优点**:
- ✅ **实时数据**（无延迟）
- ✅ **无验证码**（雪球无需验证）
- ✅ **速度快**（3-5 秒完成）
- ✅ **数据全**（价格、成交量、换手率等）
- ✅ **有社区讨论**（可看市场情绪）

**缺点**:
- ⚠️ 需要浏览器会话
- ⚠️ 需要解析 snapshot 数据

#### 2️⃣ **MCP API**（备选，快速查询）

**命令**:
```bash
cd /home/admin/.openclaw/skills/finance-data/tools
bash query.sh stock <股票代码> 2>&1 | head -50
```

**示例**:
```bash
# 查询贝肯能源
bash query.sh stock 002828 2>&1 | head -50

# 查询巨力索具
bash query.sh stock 002342 2>&1 | head -50

# 并行查询多只股票
(bash query.sh stock 002828 2>&1 &)
(bash query.sh stock 002342 2>&1 &)
```

**优点**:
- ✅ 速度快（1-2 秒）
- ✅ 数据全（价格、资金、换手、PE/PB 等）
- ✅ JSON 格式，易解析
- ✅ 不需要浏览器

**缺点**:
- ⚠️ 偶尔 API 不稳定（RemoteDisconnected）
- ⚠️ 数据延迟 15 分钟

## 常见问题 FAQ

### Q1: MCP API 返回空数据？
**A**: 检查股票代码格式：
- 深交所：`002828` → `SZ002828`
- 上交所：`600519` → `SH600519`

### Q2: 雪球浏览器加载失败？
**A**: 
1. 检查网络连接
2. 切换 MCP API 获取数据（延迟 15 分钟）
3. 用户手动在雪球/东方财富 APP 查看

### Q3: 数据延迟多久？
**A**: 
- 雪球浏览器：实时（无延迟）
- MCP API: 延迟 15 分钟

### Q4: 如何获取实时数据？
**A**: 使用雪球浏览器，3-5 秒获取实时数据！

## 核心要点总结

### 🎯 数据源选择策略

| 场景 | 首选 | 备选 | 耗时 |
|------|------|------|------|
| **实时数据**（打板/监控） | 雪球浏览器 | MCP API | 3-5 秒 |
| **盘后复盘** | MCP API | - | 1-2 秒 |

### ✅ 关键改进（v2.0）

1. **雪球浏览器作为实时数据首选** - 无验证码，数据实时
2. **移除东方财富浏览器** - 滑块验证码无法自动处理
3. **MCP API 用于盘后复盘** - 速度快，数据全
4. **5 秒内获取实时数据** - 雪球 3-5 秒，MCP1-2 秒

### 📝 实战经验

- ✅ 雪球无验证码，适合自动化监控
- ✅ MCP API 适合盘后复盘
- ❌ 东方财富滑块验证无法自动处理，已弃用
