# 回撤控制策略 v1.0

## 功能描述
控制最大回撤，保护本金，实现稳定盈利。

---

## 回撤分级

### 轻度回撤 (-5% 至 -10%)
**触发条件**: 总资金回撤 5-10%

**应对措施**:
| 措施 | 操作 |
|------|------|
| **仓位降低** | 降至 3 成 |
| **选股收紧** | 只选 S 级/A 级 |
| **止损加严** | -5% 止损 |
| **频率降低** | 每日≤1 笔 |

### 中度回撤 (-10% 至 -20%)
**触发条件**: 总资金回撤 10-20%

**应对措施**:
| 措施 | 操作 |
|------|------|
| **仓位降低** | 降至 2 成 |
| **选股收紧** | 只选 S 级 |
| **止损加严** | -3% 止损 |
| **频率降低** | 每 2 日 1 笔 |
| **暂停交易** | 休息 1-3 天 |

### 重度回撤 (-20% 至 -30%)
**触发条件**: 总资金回撤 20-30%

**应对措施**:
| 措施 | 操作 |
|------|------|
| **仓位降低** | 降至 1 成 |
| **停止交易** | 休息 1 周 |
| **复盘总结** | 找出问题 |
| **策略调整** | 优化策略 |

### 极端回撤 (>30%)
**触发条件**: 总资金回撤>30%

**应对措施**:
| 措施 | 操作 |
|------|------|
| **空仓** | 停止交易 |
| **深度复盘** | 全面反思 |
| **重新学习** | 强化训练 |
| **模拟交易** | 恢复信心 |

|------|------|------|

## 实战应用

### 案例 1: 轻度回撤
```
【情况】总资金 -8%
【措施】
1. 仓位降至 3 成
2. 只选 S 级/A 级股票
3. 止损加严至 -5%
4. 每日≤1 笔交易

【结果】3 天后回升至 -3%
```

### 案例 2: 中度回撤
```
【情况】总资金 -15%
【措施】
1. 仓位降至 2 成
2. 只选 S 级股票
3. 止损加严至 -3%
4. 每 2 日 1 笔
5. 休息 2 天

【结果】1 周后回升至 -5%
```

## 心理建设

### 回撤时的心态
1. **接受回撤**: 回撤是交易的一部分
2. **不要报复性交易**: 越亏越重仓是大忌
3. **严格执行风控**: 风控是保护伞
4. **保持耐心**: 等待机会比抓住机会更重要

### 恢复信心
1. **小仓位开始**: 1 成仓找回感觉
2. **选择高胜率**: 只做 S 级机会
3. **记录成功**: 积累正反馈
4. **逐步加仓**: 信心恢复后加仓
