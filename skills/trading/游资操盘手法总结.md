# 游资操盘手法总结 v1.0

## 一线游资识别

---

### 游资 1：中信上海分公司 ⭐⭐⭐⭐⭐

| 项目 | 说明 |
|------|------|
| **席位** | 中信证券上海分公司 |
| **风格** | 龙头战法、主线板块 |
| **资金量** | 10 亿+ |
| **胜率** | 75%+ |
| **持有周期** | 1-3 天 |

**操盘特征**：
```
✅ 偏好：主线板块龙一
✅ 封单：通常>3 亿（大封单）
✅ 入场：首板/二板
✅ 加仓：连板确认后续加
✅ 出货：高位放量出货
✅ 止损：果断，不拖泥带水
```

**识别方法**：
```
【龙虎榜】
- 买入金额：通常 5000 万 -2 亿
- 封单金额：3 亿+
- 板块：主线板块

【盘面】
- 封板坚决
- 炸板快速回封
- 连板意愿强
```

**跟随策略**：
```
【买入】
- 中信上海大买→跟随
- 封单>3 亿→打板
- 龙一地位→重仓

【卖出】
- 中信上海出货→跟随卖出
- 高位放量→清仓
- 炸板不回封→止损
```

## 机构席位识别

---

### 机构专用 ⭐⭐⭐⭐⭐

| 项目 | 说明 |
|------|------|
| **席位** | 机构专用 |
| **风格** | 价值投资、趋势配置 |
| **资金量** | 100 亿+ |
| **胜率** | 85%+ |
| **持有周期** | 数周 - 数月 |

**操盘特征**：
```
✅ 偏好：业绩股、行业龙头
✅ 封单：通常不封板，低吸为主
✅ 入场：分批建仓
✅ 加仓：趋势确认后加仓
✅ 出货：高位分批出货
✅ 止损：较少止损，趋势破位才走
```

**识别方法**：
```
【龙虎榜】
- 买入金额：5000 万 -5 亿
- 封单：通常不大
- 板块：业绩股/行业龙头

【盘面】
- 趋势向上
- 成交量稳定
- 较少涨停
```

**跟随策略**：
```
【买入】
- 机构大买→强烈看好
- 业绩股→跟随
- 趋势向上→持有

【卖出】
- 机构大卖→减仓
- 趋势破位→清仓
- 业绩暴雷→立即走人
```

## 游资 + 机构合力 ⭐⭐⭐⭐⭐

| 组合 | 说明 | 胜率 |
|------|------|------|
| **机构大买 + 游资大买** | 机构 + 游资合力 | 85%+ |
| **机构大买 + 游资小买** | 机构主导 | 80%+ |
| **机构大买 + 游资大卖** | 机构接盘 | 75%+ |
| **游资大买 + 机构小买** | 游资主导 | 70%+ |
| **游资大买 + 机构大卖** | 游资接盘 | 50%⚠️ |
| **机构大卖 + 游资大卖** | 集体出货 | 20%❌ |

**操作策略**：
```
【强烈看好】
- 机构大买 + 游资大买→重仓
- 机构大买 + 游资小买→中仓

【谨慎参与】
- 游资大买 + 机构小买→轻仓
- 游资大买 + 机构大卖→观望

【坚决回避】
- 机构大卖 + 游资大卖→清仓
```

## 游资操盘规律总结

---

### 规律 1：封单大小与持有周期

| 封单金额 | 游资类型 | 持有周期 | 操作 |
|----------|----------|----------|------|
| >3 亿 | 中信上海 | 1-3 天 | 重仓跟随 |
| 1-3 亿 | 华泰益田路 | 2-5 天 | 中仓跟随 |
| 5000 万 -1 亿 | 国泰君安江苏路 | 1-2 天 | 轻仓跟随 |
| <5000 万 | 小游资 | 1 天 | 观望 |

---

### 规律 2：游资入场时机

| 连板数 | 游资类型 | 操作 |
|--------|----------|------|
| 首板 | 国泰君安江苏路 | 跟随 |
| 二板 | 华泰益田路 | 跟随 |
| 三板 | 中信上海 | 跟随 |
| 四板 + | 多家游资 | 谨慎 |
| 六板 + | 高危 | 回避 |

---

### 规律 3：游资出货信号

| 信号 | 说明 | 操作 |
|------|------|------|
| 高位放量 | 成交量放大 2 倍+ | 减仓 |
| 炸板不回封 | 炸板后>15 分钟不回 | 清仓 |
| 龙虎榜大卖 | 游资席位大卖 | 跟随卖出 |
| 板块集体走弱 | 板块涨停减少 | 清仓 |

## 使用流程

### 盘后分析（16:00-17:00）

```
【步骤】
1. 获取龙虎榜数据
2. 识别游资席位
3. 分析买卖力度
4. 判断主力意图
5. 制定明日策略
```

### 盘中监控（实时）

```
【监控】
1. 封单变化（是否游资风格）
2. 炸板回封（游资是否回封）
3. 板块联动（游资是否做多）
```
