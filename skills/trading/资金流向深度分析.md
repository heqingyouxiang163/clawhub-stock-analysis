# 资金流向深度分析 v1.0

## 功能描述
深度分析资金流向，识别主力动向。

## 资金分类

| 类型 | 单笔金额 | 说明 |
|------|----------|------|
| 超大单 | >100 万 | 主力/机构 |
| 大单 | 20-100 万 | 大户/游资 |
| 中单 | 4-20 万 | 中户 |
| 小单 | <4 万 | 散户 |

## 核心指标

### 1. 净流入分析

| 指标 | 阈值 | 说明 |
|------|------|------|
| 超大单净流入 | >5000 万 | 主力大幅流入 |
| 超大单净流入 | 1000-5000 万 | 主力中幅流入 |
| 超大单净流入 | <1000 万 | 主力小幅流入 |
| 超大单净流出 | >5000 万 | 主力大幅流出 |
| 超大单净流出 | 1000-5000 万 | 主力中幅流出 |
| 超大单净流出 | <1000 万 | 主力小幅流出 |

### 2. 资金分化

| 情况 | 说明 | 后市预期 |
|------|------|----------|
| 超大单 + 大单流入 | 主力 + 大户合力 | ⭐⭐⭐⭐⭐ 强烈看好 |
| 超大单流入 + 大单流出 | 主力进场大户出货 | ⭐⭐⭐⭐ 看好 |
| 超大单流出 + 大单流入 | 主力出货大户接盘 | ⭐⭐ 谨慎 |
| 超大单 + 大单流出 | 主力 + 大户出货 | ⭐ 看空 |

### 3. 散户动向

| 情况 | 说明 | 后市预期 |
|------|------|----------|
| 小单大幅流出 | 散户离场 | ⭐⭐⭐⭐ 看好（反向） |
| 小单大幅流入 | 散户进场 | ⭐⭐ 谨慎（反向） |
| 小单稳定 | 散户观望 | ⭐⭐⭐ 中性 |

## 资金流向信号

### ✅ 利好信号
| 信号 | 阈值 | 操作 |
|------|------|------|
| 超大单净流入 | >5000 万 | 跟随买入 |
| 超大单连续 3 日流入 | 持续流入 | 加仓 |
| 超大单流入 + 小单流出 | 主力进散户出 | 强烈看好 |
| 封板时超大单激增 | 主力封板 | 持有 |

### ❌ 利空信号
| 信号 | 阈值 | 操作 |
|------|------|------|
| 超大单净流出 | >5000 万 | 卖出 |
| 超大单连续 3 日流出 | 持续流出 | 清仓 |
| 超大单流出 + 小单流入 | 主力出散户进 | 强烈看空 |
| 炸板时超大单骤减 | 主力撤单 | 立即卖出 |

## 使用流程

### 盘中监控（每 30 分钟）
```
1. 获取实时资金流向
2. 分析超大单变化
3. 判断主力意图
4. 调整策略
```

### 盘后分析（16:00-17:00）
```
1. 获取全天资金流向
2. 分析主力净流入/出
3. 对比历史数据
4. 制定明日策略
```

## 输出格式

```
【资金流向分析】璞泰来 603659

【全天资金流向】
| 类型 | 净流入 | 占比 |
|------|--------|------|
| 超大单 | +8000 万 | 45% ✅ |
| 大单 | +5000 万 | 28% ✅ |
| 中单 | +2000 万 | 11% |
| 小单 | -3000 万 | -16% ✅ |

【分析】
- 超大单 + 大单大幅流入 ✅
- 散户流出，主力进场 ✅
- 主力净流入 1.3 亿，强烈看好

【结论】
- 主力强烈看好，继续持有
- 明日可加仓
```
