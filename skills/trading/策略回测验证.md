# 策略回测验证技能 v1.0

## 功能描述
每周日回测策略有效性，持续优化胜率。

---

## 运行时间
** cron**: `0 20 * * 0`（每周日 20:00）
**时区**: Asia/Shanghai

---

## 回测内容

### 策略列表

| 策略 | 胜率标准 | 实际胜率 | 状态 |
|------|----------|----------|------|
| **主升龙一板** | 85%+ | 待回测 | - |
| **弱转强板** | 80%+ | 待回测 | - |
| **分歧转一致板** | 75%+ | 待回测 | - |
| **首板龙头板** | 72%+ | 待回测 | - |
| **突破板** | 65%+ | 待回测 | - |

### 回测维度

| 维度 | 说明 |
|------|------|
| **胜率** | 盈利次数/总次数 |
| **盈亏比** | 平均盈利/平均亏损 |
| **最大回撤** | 最大连续亏损 |
| **夏普比率** | 风险调整后收益 |

---

## 回测流程

### Step 1: 获取历史数据（20:00-20:30）

**数据源**: 雪球/同花顺/东方财富

**获取内容**:
- 过去 3 个月涨停股列表
- 次日涨跌幅
- 3 日后涨跌幅
- 5 日后涨跌幅

### Step 2: 策略回测（20:30-21:00）

**回测方法**:
- 按策略条件筛选股票
- 统计买入后收益
- 计算胜率/盈亏比

### Step 3: 策略优化（21:00-21:30）

**优化方向**:
- 调整筛选条件
- 优化仓位配置
- 改进止损止盈

### Step 4: 输出报告（21:30-22:00）

**报告格式**:
```markdown
## 📊 策略回测报告 (MM-DD)

### 🎯 回测周期：过去 3 个月

### 📈 策略表现

| 策略 | 胜率 | 盈亏比 | 最大回撤 | 状态 |
|------|------|--------|----------|------|
| 主升龙一板 | XX% | X:X | -XX% | ✅/⚠️/❌ |
| 弱转强板 | XX% | X:X | -XX% | ✅/⚠️/❌ |
| 分歧转一致板 | XX% | X:X | -XX% | ✅/⚠️/❌ |

### 📊 优化建议
- [策略名]：调整 [条件] 从 X 到 Y
- [策略名]：增加 [条件]
- [策略名]：淘汰/保留

### 📈 下周策略
- 重点使用：[策略名]
- 谨慎使用：[策略名]
- 暂停使用：[策略名]
```
