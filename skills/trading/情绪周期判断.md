# 情绪周期判断技能 v1.0

## 技能描述
判断 A 股市场情绪周期阶段，指导仓位管理和操作策略。

## 四阶段判断标准

### 1️⃣ 主升期
| 指标 | 阈值 |
|------|------|
| 连板高度 | ≥5 板 |
| 涨停家数 | >50 家 |
| 炸板率 | <30% |
| 涨跌家数 | 涨>跌 |

**操作策略**：重仓打龙，2-3 成/只
**胜率预期**：75%+

### 2️⃣ 分歧期
| 指标 | 阈值 |
|------|------|
| 连板高度 | 3-4 板 |
| 涨停家数 | 30-50 家 |
| 炸板率 | 30-50% |
| 涨跌家数 | 涨≈跌 |

**操作策略**：轻仓试错，1-2 成/只，只做龙一
**胜率预期**：60-70%

### 3️⃣ 退潮期
| 指标 | 阈值 |
|------|------|
| 连板高度 | ≤2 板 |
| 涨停家数 | <30 家 |
| 炸板率 | >50% |
| 涨跌家数 | 跌>涨 |

**操作策略**：防守，0-1 成或空仓
**胜率预期**：<50%

### 4️⃣ 冰点期
| 指标 | 阈值 |
|------|------|
| 连板高度 | 1 板 |
| 涨停家数 | <10 家 |
| 炸板率 | >70% |
| 涨跌家数 | 普跌 |

**操作策略**：空仓等待，或试错首板
**胜率预期**：不确定

## 使用方式
```
1. 9:25 集合竞价后收集数据
2. 对照阈值判断周期阶段
3. 执行对应仓位策略
4. 盘后验证判断准确性
```

## 数据来源
- 东方财富涨停板数据
- 同花顺情绪指标
- 手动统计连板高度
