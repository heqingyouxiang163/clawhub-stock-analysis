# 实时数据抓取技能 v1.0

## 功能描述
通过雪球浏览器获取实时股票数据，延迟<5 秒。

---

## 数据源优先级

| 优先级 | 数据源 | 延迟 | 验证码 | 状态 |
|--------|--------|------|--------|------|
| **1️⃣** | 雪球浏览器 | <5 秒 | ✅ 无 | ✅ 首选 |
| **2️⃣** | MCP API | 1-2 秒 | ✅ 无 | ⚠️ 延迟 15 分钟 |
| **3️⃣** | 东方财富 | 5-10 秒 | ❌ 有 | ⚠️ 备选 |

---

## 使用方法

### 浏览器自动化

```python
# 导航到雪球股票页面
browser(action=navigate, targetUrl="https://xueqiu.com/SZ002828")

# 获取页面快照
browser(action=snapshot, targetId="<会话 ID>", refs="aria")

# 解析数据
current_price = parse_price(snapshot)
change_pct = parse_percent(snapshot)
```

### 数据校验

```python
# A 股硬约束
assert -10 <= change_pct <= 10, "涨跌幅异常"
assert 0 <= turnover_rate <= 50, "换手率异常"
assert high >= current >= low, "价格关系异常"
```
