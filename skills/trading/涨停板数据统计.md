# 涨停板数据统计 v1.0

## 功能描述
统计涨停板数据，分析市场情绪和主线。

---

## 核心统计指标

---

### 涨停家数 ⭐⭐⭐⭐⭐

| 数值 | 情绪 | 仓位 | 说明 |
|------|------|------|------|
| **>80 家** | 极热 | 90% | 市场狂热 |
| **50-80 家** | 火热 | 70-80% | 主升期 |
| **30-50 家** | 正常 | 50-70% | 分歧期 |
| **10-30 家** | 偏冷 | 20-30% | 退潮期 |
| **<10 家** | 极冷 | 0% | 冰点期 |

---

### 连板高度 ⭐⭐⭐⭐⭐

| 高度 | 情绪 | 说明 |
|------|------|------|
| **≥7 板** | 疯狂 | 可能见顶 ⚠️ |
| **5-6 板** | 高潮 | 主升末期 ⚠️ |
| **3-4 板** | 健康 | 主升期 ✅ |
| **2 板** | 一般 | 分歧期 ⚠️ |
| **1 板** | 低迷 | 退潮/冰点 ❌ |

---

### 炸板率 ⭐⭐⭐⭐⭐

| 炸板率 | 情绪 | 说明 |
|--------|------|------|
| **<15%** | 极好 | 封板质量高 ✅ |
| **15-25%** | 正常 | 封板质量一般 ✅ |
| **25-35%** | 偏差 | 封板质量下降 ⚠️ |
| **35-50%** | 差 | 封板质量差 ❌ |
| **>50%** | 极差 | 集体炸板 ❌❌ |

**计算公式**：
```
炸板率 = 炸板家数 / (涨停家数 + 炸板家数) × 100%
```

---

### 涨跌家数比 ⭐⭐⭐⭐

| 比值 | 情绪 | 说明 |
|------|------|------|
| **>5:1** | 极热 | 普涨行情 ✅ |
| **3-5:1** | 火热 | 多数上涨 ✅ |
| **1-3:1** | 正常 | 分化行情 ⚠️ |
| **1:1-1:3** | 偏冷 | 多数下跌 ⚠️ |
| **<1:5** | 极冷 | 普跌行情 ❌ |

## 情绪周期判断

---

### 综合情绪评分

```
【情绪评分公式】
= 涨停家数 (30 分) + 连板高度 (25 分) + 炸板率 (20 分) + 涨跌比 (15 分) + 板块效应 (10 分)

【涨停家数】(30 分)
- >80 家：30 分
- 50-80 家：25 分
- 30-50 家：20 分
- 10-30 家：10 分
- <10 家：0 分

【连板高度】(25 分)
- ≥7 板：25 分
- 5-6 板：20 分
- 3-4 板：15 分
- 2 板：10 分
- 1 板：0 分

【炸板率】(20 分)
- <15%：20 分
- 15-25%：15 分
- 25-35%：10 分
- 35-50%：5 分
- >50%：0 分

【涨跌比】(15 分)
- >5:1：15 分
- 3-5:1：12 分
- 1-3:1：8 分
- 1:1-1:3：5 分
- <1:5：0 分

【板块效应】(10 分)
- ≥3 个板块涨停≥5 家：10 分
- 2 个板块涨停≥5 家：7 分
- 1 个板块涨停≥5 家：5 分
- 无板块涨停≥5 家：0 分

【决策】
- ≥80 分：主升期 → 仓位 70-90%
- 60-79 分：分歧期 → 仓位 30-50%
- 40-59 分：退潮期 → 仓位 0-20%
- <40 分：冰点期 → 仓位 0%
```

## 数据获取方式

---

### 同花顺

```
1. 打开同花顺→数据中心→涨停分析
2. 查看涨停家数、连板高度、炸板率
3. 查看板块涨停分布
4. 查看涨停时间分布
```

---

### 东方财富

```
1. 打开东方财富→数据中心→涨停板
```
