# 量价关系分析 v1.0

## 功能描述
通过成交量与价格关系，识别主力动向、趋势强弱。

---

## 量价核心原理

---

### 量价配合（4 种）

| 关系 | 价格 | 成交量 | 含义 | 操作 |
|------|------|--------|------|------|
| **量增价涨** | 上涨 | 放大 | 主力买入 | ✅ 跟随 |
| **量缩价涨** | 上涨 | 缩小 | 主力控盘/诱多 | ⚠️ 谨慎 |
| **量增价跌** | 下跌 | 放大 | 主力出货 | ❌ 卖出 |
| **量缩价跌** | 下跌 | 缩小 | 洗盘/无人问津 | ⚠️ 观望 |

---

## 健康量价关系

---

### 上涨健康量价 ⭐⭐⭐⭐⭐

| 阶段 | 价格 | 成交量 | 说明 |
|------|------|--------|------|
| **启动** | 上涨 3-5% | 放大 50-100% | 主力进场 |
| **发酵** | 上涨 5-8% | 放大 100-200% | 主力加仓 |
| **高潮** | 涨停 | 放大 200%+ | 主力锁仓 |
| **盘整** | 横盘 | 缩量 50% | 洗盘 |
| **再涨** | 上涨 | 再次放大 | 第二波 |

**操作**：
```
✅ 启动期→买入
✅ 发酵期→加仓
✅ 高潮期→持有
✅ 盘整期→持有/加仓
✅ 再涨期→持有/止盈
```

---

### 下跌健康量价 ⭐⭐⭐

| 阶段 | 价格 | 成交量 | 说明 |
|------|------|--------|------|
| **回调** | 下跌 3-5% | 缩量 30-50% | 洗盘 |
| **企稳** | 横盘 | 缩量 50%+ | 主力未走 |
| **反弹** | 上涨 | 放大 | 主力回流 |

**操作**：
```
✅ 回调缩量→持有
✅ 企稳缩量→观察
✅ 反弹放量→加仓
❌ 回调放量→卖出
```

## 异常量价关系

---

### 量价背离（危险信号）⚠️

| 类型 | 价格 | 成交量 | 含义 | 操作 |
|------|------|--------|------|------|
| **顶背离** | 新高 | 低于前高 | 主力出货 | ❌ 卖出 |
| **底背离** | 新低 | 高于前低 | 主力吸筹 | ✅ 关注 |
| **量缩价涨** | 上涨 | 持续缩量 | 诱多 | ❌ 卖出 |
| **量增价跌** | 下跌 | 持续放量 | 出货 | ❌ 卖出 |

---

### 天量天价 ⚠️

| 特征 | 说明 |
|------|------|
| **成交量** | 历史天量（5 日+最高） |
| **价格** | 高位（涨幅>50%） |
| **含义** | 主力出货 |
| **后续** | 大概率见顶 |

**操作**：
```
❌ 不追高
✅ 持有者止盈
✅ 次日不涨清仓
```

---

### 地量地价 ⭐

| 特征 | 说明 |
|------|------|
| **成交量** | 历史地量（20 日 + 最低） |
| **价格** | 低位（跌幅>30%） |
| **含义** | 主力吸筹/无人问津 |
| **后续** | 可能见底 |

**操作**：
```
✅ 关注
✅ 放量上涨→买入
❌ 不急于抄底
```

## 成交量分级

---

### 成交量等级

| 等级 | 量比 | 说明 | 操作 |
|------|------|------|------|
| **缩量** | <0.8 | 无人问津 | ⚠️ 观望 |
| **平量** | 0.8-1.2 | 正常 | ⚠️ 观望 |
| **温和放量** | 1.2-2.0 | 主力进场 | ✅ 关注 |
| **明显放量** | 2.0-3.0 | 主力加仓 | ✅ 跟随 |
| **巨量** | 3.0-5.0 | 主力大动作 | ✅ 强势 |
| **天量** | >5.0 | 见顶信号 | ⚠️ 谨慎 |

---

### 换手率等级

| 等级 | 换手率 | 说明 | 操作 |
|------|--------|------|------|
| **缩量** | <3% | 无人问津 | ⚠️ 观望 |
| **正常** | 3-5% | 正常交易 | ⚠️ 观望 |
| **活跃** | 5-10% | 主力参与 | ✅ 关注 |
| **高度活跃** | 10-15% | 主力加仓 | ✅ 跟随 |
| **过热** | 15-20% | 分歧加大 | ⚠️ 谨慎 |
| **危险** | >20% | 见顶信号 | ❌ 卖出 |

## 量价形态（8 种）

---

### 形态 1：底部放量 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 低位（跌幅>30%） |
| **成交量** | 突然放大 2-3 倍 |
| **价格** | 上涨 3-5% |
| **含义** | 主力吸筹 |
| **胜率** | 75%+ |

**操作**：
```
✅ 放量当日→买入
✅ 回踩缩量→加仓
✅ 突破平台→加仓
```

---

### 形态 2：突破放量 ⭐⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 平台突破 |
| **成交量** | 放大 2-3 倍 |
| **价格** | 突破平台 5%+ |
| **含义** | 主力进攻 |
| **胜率** | 80%+ |

**操作**：
```
✅ 突破当日→买入
✅ 回踩平台→加仓
✅ 持有至缩量
```

---

### 形态 3：缩量回调 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 上涨途中 |
| **成交量** | 缩量 50%+ |
| **价格** | 回调 3-5% |
| **含义** | 洗盘 |
| **胜率** | 70%+ |

**操作**：
```
✅ 缩量企稳→买入
✅ 反弹放量→加仓
❌ 放量回调→卖出
```

---

### 形态 4：高位放量 ⚠️

| 特征 | 说明 |
|------|------|
| **位置** | 高位（涨幅>50%） |
| **成交量** | 放大 3-5 倍 |
| **价格** | 滞涨/下跌 |
| **含义** | 主力出货 |
| **胜率** | 80% 下跌 |

**操作**：
```
❌ 不追高
✅ 持有者止盈
❌ 次日不涨清仓
```

---

### 形态 5：连续放量 ⭐⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 上涨初期 |
| **成交量** | 连续 3 日 + 放量 |
| **价格** | 连续上涨 |
| **含义** | 主力持续加仓 |
| **胜率** | 75%+ |

**操作**：
```
✅ 第 1-2 日→买入
✅ 第 3-4 日→持有
⚠️ 第 5 日 +→谨慎
```

---

### 形态 6：缩量横盘 ⭐⭐⭐

| 特征 | 说明 |
|------|------|
| **位置** | 上涨途中/高位 |
| **成交量** | 缩量 50%+ |
| **价格** | 横盘 3-5 日 |
| **含义** | 洗盘/主力休息 |
| **胜率** | 65%+ |

**操作**：
```
✅ 横盘不破→持有
✅ 突破横盘→加仓
❌ 跌破横盘→卖出
```

---

### 形态 7：放量下跌 ❌

| 特征 | 说明 |
|------|------|
| **位置** | 任何位置 |
| **成交量** | 放大 2 倍 + |
| **价格** | 下跌 3%+ |
| **含义** | 主力出货 |
| **胜率** | 80% 继续跌 |

**操作**：
```
❌ 不抄底
❌ 不补仓
✅ 立即卖出
```

---

### 形态 8：天量见顶 ❌

| 特征 | 说明 |
|------|------|
| **位置** | 高位（涨幅>50%） |
| **成交量** | 历史天量 |
| **价格** | 滞涨/长上影 |
| **含义** | 主力出货完毕 |
| **胜率** | 85% 见顶 |

**操作**：
```
❌ 不追高
✅ 持有者清仓
❌ 不博反弹
```

## 量价监控流程

---

### 9:30-10:00（开盘 30 分钟）

```
【监控重点】
□ 开盘成交量（对比昨日）
□ 前 5 分钟量价关系
□ 是否放量上涨

【操作】
✅ 放量上涨→关注
✅ 缩量上涨→观望
❌ 放量下跌→回避
```

---

### 10:00-11:30（上午盘）

```
【监控重点】
□ 量价配合
□ 成交量趋势
□ 是否持续放量

【操作】
✅ 量增价涨→持有/加仓
✅ 量缩价涨→谨慎
❌ 量增价跌→卖出
```

---

### 13:00-14:30（下午盘）

```
【监控重点】
□ 午后成交量
□ 是否放量突破
□ 量价是否背离

【操作】
✅ 放量突破→加仓
✅ 量价配合→持有
❌ 量价背离→减仓
```

---

### 14:30-15:00（尾盘）

```
【监控重点】
□ 尾盘成交量
□ 是否放量拉升
□ 天量信号

【操作】
✅ 温和放量→持有
⚠️ 天量→止盈
❌ 放量下跌→清仓
```

## 量价案例

---

### 成功案例

```
【案例 1】璞泰来 603659（2026-03-16）
- 9:30 开盘：成交量放大 150%
- 9:35 拉升：成交量放大 200%
- 9:40 封板：成交量放大 300%
- 量价关系：量增价涨✅
- 结果：涨停 +10%

【操作】
✅ 9:35 放量拉升→买入
✅ 封板持有
✅ 次日高开止盈
```

---

### 失败案例

```
【案例 2】XXX（2026-02-15）
- 位置：高位（涨幅 60%）
- 成交量：历史天量（5 倍）
- 价格：滞涨（+1%）
- 量价关系：天量滞涨❌
- 结果：次日跌停 -10%

【教训】
❌ 高位天量不追
❌ 天量滞涨是出货信号
✅ 应清仓回避
```
