**止损**: 锤头最低点

- 第三天：大阳线

**买入信号**: 第三天收盘确认
**胜率**: 80%
**止损**: 十字星最低点

### 4. 双底 (W 底) ⭐⭐⭐⭐
**形态特征**:
- 两个低点基本持平
- 中间有反弹
- 突破颈线确认

## 使用要点

### ✅ 提高准确率
1. **配合成交量**: 形态确认时成交量放大
2. **配合均线**: 形态与均线方向一致
3. **配合指标**: MACD/KDJ 等指标确认
4. **多次确认**: 2 次以上确认更可靠

### ❌ 避免错误
1. **不要提前介入**: 等待形态完全确认
2. **不要忽视成交量**: 无量突破多假突破
3. **不要忽视趋势**: 逆势形态胜率低
4. **严格止损**: 形态失败立即止损

## 实战案例

### 成功案例
```
【案例】早晨之星 (2026-03-15)
第一天：大阴线 -5%
第二天：十字星 -1%
第三天：大阳线 +6%
买入：第三天收盘
止损：十字星最低点
胜率：75%
```

### 失败案例
```
【案例】锤头线假信号 (2026-03-10)
锤头线：下影线 3 倍
次日：继续下跌 -4%
止损：锤头最低点
亏损：-3%

【教训】单根 K 线可靠性低，需配合其他信号
```
