# T+0 策略 v1.0

## 功能描述
利用底仓进行 T+0 交易，降低持仓成本，增厚收益。

---

## 适用条件

### 底仓要求
| 条件 | 标准 | 说明 |
|------|------|------|
| **底仓市值** | >5 万元 | 有足够操作空间 |
| **底仓成本** | 浮盈>5% | 安全垫足够 |
| **股票流动性** | 日成交>5 亿 | 便于进出 |
| **波动率** | 日均振幅>3% | 有操作空间 |

### 市场条件
| 条件 | 说明 |
|------|------|
| **震荡市** | 最适合 T+0 |
| **上升趋势** | 可做正 T |
| **下降趋势** | 可做反 T |
| **单边行情** | 不适合 T+0 |

---

## 操作模式

### 正 T (先买后卖)
**适用场景**: 预期当日上涨

| 步骤 | 操作 | 说明 |
|------|------|------|
| **低位买入** | -2% 至 -3% | 买入 1 成仓 |
| **高位卖出** | +2% 至 +3% | 卖出 1 成仓 |
| **价差** | 4-6% | 当日获利 |

**案例**:
```
底仓：10000 股，成本 10 元
-9:35 买入：1000 股 @ 9.8 元 (-2%)
-10:30 卖出：1000 股 @ 10.2 元 (+2%)
价差：0.4 元/股
获利：400 元
成本降低：0.04 元/股
```

### 反 T (先卖后买)
**适用场景**: 预期当日下跌

| 步骤 | 操作 | 说明 |
|------|------|------|
| **高位卖出** | +2% 至 +3% | 卖出 1 成仓 |
| **低位买入** | -2% 至 -3% | 买入 1 成仓 |
| **价差** | 4-6% | 当日获利 |

**案例**:
```
底仓：10000 股，成本 10 元
-9:35 卖出：1000 股 @ 10.2 元 (+2%)
-14:00 买入：1000 股 @ 9.8 元 (-2%)
价差：0.4 元/股
获利：400 元
成本降低：0.04 元/股
```

## 时间窗口

### 最佳操作时间
| 时间段 | 适合操作 | 说明 |
|--------|----------|------|
| **9:30-10:00** | 正 T 买入 | 早盘低点 |
| **10:00-11:00** | 正 T 卖出 | 早盘高点 |
| **13:00-14:00** | 反 T 卖出 | 午后高点 |
| **14:00-15:00** | 反 T 买入 | 尾盘低点 |

### 避免操作时间
| 时间段 | 原因 |
|--------|------|
| **9:15-9:25** | 集合竞价，波动大 |
| **11:30-13:00** | 午休，流动性差 |
| **14:50-15:00** | 收盘前，不确定性高 |

## 仓位控制

| 底仓规模 | T+0 仓位 |
|----------|----------|
| **5-10 万** | 0.5 成 (5%) |
| **10-50 万** | 1 成 (10%) |
| **50-100 万** | 2 成 (20%) |
| **100 万+** | 3 成 (30%) |

|----------|----------|

| **中波动** | 2 次 |
| **低波动** | 1 次 |
| **极低波动** | 不做 |

|------|------|

## 风险提示

### ⚠️ 常见错误
1. **T 飞了**: 卖出后继续大涨，不敢追回
2. **T 反了**: 买入后继续下跌，越套越深
3. **忘记平仓**: 当日未平仓，变成加仓
4. **过度交易**: 频繁 T+0，手续费侵蚀利润

### ⚠️ 注意事项
1. **当日必须平仓**: T+0 是日内交易，不能过夜
2. **控制仓位**: 单次不超过底仓的 30%
3. **设置止损**: 反向波动 2% 必须止损
4. **记录成本**: 每次 T+0 后更新持仓成本

## 实战案例

### 成功案例
```
【案例】XX 能源 (2026-03-15)
底仓：5000 股，成本 14.5 元

9:35 买入：1000 股 @ 14.0 元 (-3.4%)
10:30 卖出：1000 股 @ 14.6 元 (+0.7%)
价差：0.6 元/股
获利：600 元
成本降低：0.12 元/股

新成本：14.38 元 (原 14.5 元)
```

### 失败案例
```
【案例】XX 科技 (2026-03-12)
底仓：3000 股，成本 25 元

9:40 买入：500 股 @ 24.5 元 (-2%)
继续下跌：24.0 元 (-4%)
止损卖出：500 股 @ 24.0 元
亏损：250 元

【教训】反向波动 2% 应果断止损
```
