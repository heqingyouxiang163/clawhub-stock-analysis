# 心态控制手册 v1.0

## 功能描述
交易心态管理，避免情绪化交易。

---

## 交易心态核心

---

### 正确心态 ✅

```
【关于亏损】
✅ 亏损是交易的一部分
✅ 止损是保护本金
✅ 承认错误，不抱幻想
✅ 亏损后冷静，不报复

【关于盈利】
✅ 盈利是市场的奖励
✅ 不赚最后一个铜板
✅ 分批止盈，不贪心
✅ 盈利后不骄傲

【关于交易】
✅ 交易是概率游戏
✅ 不追求 100% 胜率
✅ 坚持策略，不随意变
✅ 耐心等待机会

【关于自己】
✅ 我是交易者，不是赌徒
✅ 我有纪律，不情绪化
✅ 我承认不足，不断学习
✅ 我控制风险，不控制市场
```

---

```

❌ 盈利后骄傲自满

❌ 频繁交易，不等待
❌ 随意改变策略
❌ 听消息，不独立判断

【关于自己】
❌ 我是股神，能预测市场
❌ 我没有纪律，随情绪

```

## 情绪化交易识别

---

### 愤怒交易

| 信号 | 说明 | 应对 |
|------|------|------|
| 止损后立即买回 | 不服气 | 离开电脑 10 分钟 |
| 亏损后加仓 | 想翻本 | 强制空仓 1 天 |
| 频繁交易 | 想快速回本 | 停止交易 3 天 |
| 重仓单只股票 | 赌一把 | 降低仓位至 1 成 |

---

### 恐惧交易

| 信号 | 说明 | 应对 |
|------|------|------|
| 不敢开仓 | 怕亏损 | 轻仓试错 1 成 |
| 过早止盈 | 怕利润回吐 | 分批止盈 |
| 空仓过久 | 怕错过 | 设定固定交易时间 |
| 追涨杀跌 | 怕踏空 | 等待回调 |

---

### 贪婪交易

| 信号 | 说明 | 应对 |
|------|------|------|
| 不止盈 | 想赚更多 | 设定固定止盈位 |
| 满仓 | 想赚快钱 | 强制保留 20% 现金 |
| 连板不卖 | 想翻倍 | 5 连板强制清仓 |
| 借款交易 | 想暴富 | 立即停止交易 |

---

### 骄傲交易

| 信号 | 说明 | 应对 |
|------|------|------|
| 随意开仓 | 觉得自己厉害 | 回顾亏损记录 |
| 不听建议 | 自以为是 | 找人交流 |
| 重仓单只 | 相信自己判断 | 强制分散仓位 |
| 忽视风险 | 觉得不会亏 | 阅读风险控制文档 |

## 心态控制方法

---

### 方法 1：交易前冥想

```
【步骤】
1. 深呼吸 5 次
2. 问自己：今天状态好吗？
3. 问自己：今天市场好吗？
4. 问自己：有符合策略的机会吗？
5. 回答都是"是"才交易

【时间】
- 每笔交易前 1 分钟
- 开盘前 5 分钟
```

---

### 方法 2：亏损后冷静

```
【步骤】
1. 立即止损
2. 离开电脑 5 分钟
3. 深呼吸 10 次
4. 回顾止损纪律
5. 问自己：还适合交易吗？
6. 状态不好→停止交易
7. 状态好→继续，但降低仓位

【时间】
- 每笔亏损后
- 连续亏损 2 笔后强制停止 1 小时
```

---

### 方法 3：盈利后冷静

```
【步骤】
1. 执行止盈
2. 不立即开新仓
3. 休息 10 分钟
4. 回顾盈利原因（策略 vs 运气）
5. 问自己：还有机会吗？
6. 有符合策略的机会→参与
7. 没有→等待

【时间】
- 每笔盈利后
- 连续盈利 3 笔后降低仓位 10%
```

---

### 方法 4：每日复盘心态

```
【问题】
1. 今天执行纪律了吗？
2. 今天情绪化交易了吗？
3. 今天有报复性交易吗？
4. 今天有骄傲自满吗？
5. 今天心态好吗？

【评分】
- 5 分：完美
- 4 分：良好
- 3 分：一般
- 2 分：较差
- 1 分：很差

【改进】
- <4 分：找出问题，明日改进
- 连续 3 日<3 分：停止交易 3 天
```

## 心态控制案例

---

### 成功案例

```
【案例 1】连续亏损后冷静
- 连续亏损 2 笔，亏损 -8%
- 立即停止交易
- 离开电脑，出去散步
- 复盘亏损原因
- 发现：不符合策略，情绪化交易
- 次日：轻仓 1 成试错
- 盈利 +3%，心态恢复
- 第 3 日：正常仓位，盈利 +5%

【教训】
✅ 亏损后停止交易正确
✅ 复盘找原因正确
✅ 轻仓试错恢复信心正确
```

---

### 失败案例

```
【案例 2】亏损后报复性交易
- 亏损 -5%，止损
- 不服气，立即买回
- 继续亏损 -5%
- 加仓摊低成本
- 再亏损 -10%
- 总亏损 -20%
- 心态崩溃，清仓

【教训】
❌ 止损后买回错误
❌ 加仓摊平错误
❌ 不控制情绪错误
❌ 应停止交易，冷静反思
```

## 心态控制清单

---

### 每日检查

```
【盘前】
□ 今天状态好吗？（睡眠/精神）
□ 今天市场好吗？（情绪周期）
□ 有符合策略的机会吗？
□ 心态平静吗？

【盘中】
□ 有情绪化交易吗？
□ 有报复性交易吗？
□ 有骄傲自满吗？
□ 执行纪律了吗？

【盘后】
□ 今日心态评分（1-5 分）
□ 有情绪化交易吗？
□ 明日如何改进？
```

---

### 强制停止交易条件

```
【条件】
□ 连续亏损 3 笔
□ 单日亏损 -5%
□ 单周亏损 -10%
□ 心态评分连续 3 日<3 分
□ 生病/疲劳/情绪不好
□ 重大生活事件

【操作】
- 满足任一条件：停止交易
- 休息 1-3 天
- 调整心态
- 复盘反思
- 状态恢复后轻仓试错
```
