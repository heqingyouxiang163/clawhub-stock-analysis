# 自动止损止盈监控技能 v1.0

## 功能描述
实时监控持仓股，自动触发止损止盈提醒，严格执行交易纪律。

---

## 运行时间
** cron**: `*/1 9-11,13-15 * * 1-5`（交易日每 1 分钟）
**时区**: Asia/Shanghai

---

## 监控对象

### 持仓股列表
从 `memory/YYYY-MM-DD.md` 读取当日持仓股

**示例**:
```markdown
## 持仓股
- 贝肯能源 (002828): 成本 14.85 元，仓位 3 成
- 巨力索具 (002342): 成本 14.253 元，仓位 3 成
```

---

## 监控规则

### 止损规则（严格执行）

| 条件 | 操作 | 说明 |
|------|------|------|
| **跌破 -3%** | 立即提醒止损 | 短线铁律 |
| **跌破 -5%** | 强烈提醒止损 | 最后机会 |
| **跌破 -7%** | 强制清仓提醒 | 避免大亏 |
| **炸板 15 分钟不回封** | 提醒清仓 | 避免跳水 |

### 止盈规则（灵活执行）

| 条件 | 操作 | 说明 |
|------|------|------|
| **达到 +5%** | 提醒可止盈 | 锁定利润 |
| **达到 +7%** | 建议止盈 | 落袋为安 |
| **达到 +10% (涨停)** | 持有过夜 | 次日溢价 |
| **触及涨停后炸板** | 提醒止盈 | 避免回落 |

---

## 运行流程

### Step 1: 获取实时数据（每 1 分钟）

**数据源**: 雪球浏览器 / MCP API

**获取内容**:
- 现价
- 涨跌幅
- 是否涨停
- 是否炸板

### Step 2: 计算盈亏（每 1 分钟）

**公式**:
```
盈亏% = (现价 - 成本价) / 成本价 * 100
```

### Step 3: 触发条件判断（每 1 分钟）

**判断逻辑**:
```
if 盈亏 <= -3%:
    触发止损提醒
elif 盈亏 <= -5%:
    触发强烈止损提醒
elif 盈亏 <= -7%:
    触发强制清仓提醒
elif 盈亏 >= +5% and 盈亏 < +7%:
    触发止盈提醒
elif 盈亏 >= +7% and 盈亏 < +10%:
    触发建议止盈提醒
elif 盈亏 >= +10%:
    触发涨停持有提醒
    
if 涨停后炸板 and 炸板时间 > 15 分钟:
    触发炸板清仓提醒
```

### Step 4: 发送提醒（触发时）

**提醒格式**:
```markdown
## ⚠️ [止损/止盈] 提醒

**股票**: [股票名] ([代码])
**成本**: XX 元
**现价**: XX 元
**盈亏**: -X.XX% / +X.XX%

### 📊 建议操作
[止损/止盈/持有]

### 💰 仓位：X 成

### ⏰ 时间：HH:MM

**理由**: [触发条件说明]
```

---

## 红线（必须执行）

❌ 跌破 -3% 不止损 → 可能大亏
❌ 炸板 15 分钟不回封 → 可能跳水
❌ 退潮期重仓 → 逆势操作
❌ 单笔亏损>7% → 严重违规
