# 涨停形态每日迭代学习技能 v1.0

## 功能描述
每日收盘后自动收集涨停股数据，分析形态特征，保存到记忆库，持续迭代优化胜率。

---

## 运行时间
** cron**: `30 15 * * 1-5`（交易日 15:30）
**时区**: Asia/Shanghai

---

## 学习流程

### Step 1: 获取涨停股列表（15:30-15:35）

**数据源**: 雪球/同花顺/东方财富

**获取内容**:
- 今日所有涨停股列表
- 每只股票的涨停时间
- 每只股票的封单金额
- 每只股票的换手率
- 每只股票的成交额
- 每只股票的市值
- 每只股票所属板块

### Step 2: 分析形态特征（15:35-15:45）

**分析维度**:

| 维度 | 标准 | 说明 |
|------|------|------|
| **换手率** | 8-15% 完美 | 过高/过低标记 |
| **主力流入** | >1 亿强 | 流入/流出标记 |
| **市值** | 50-300 亿适中 | 太小/太大标记 |
| **成交额** | >5 亿达标 | 流动性判断 |
| **涨停时间** | <10:30 早盘 | 早盘/午盘/尾盘 |
| **封单金额** | >1.5 亿强 | 封单强度 |
| **板块效应** | ≥3 家涨停 | 板块热度 |
| **连板数** | 1 板/2 板/3 板 + | 连板高度 |

### Step 3: 对照 10 种涨停形态（15:45-15:50）

**形态分类**:

| 形态 | 特征 | 标准胜率 |
|------|------|----------|
| **主升龙一板** | 主升期 + 龙一 + 封单>3 亿 | 85%+ |
| **弱转强板** | 昨日烂板 + 今日高开 3-5% | 80%+ |
| **回封真龙板** | 炸板→5 分钟内回封 | 78%+ |
| **分歧转一致板** | 早盘分歧→午后上攻 | 75%+ |
| **首板龙头板** | 板块首板 + 封单>2 亿 | 72%+ |
| **二板加速板** | 昨日涨停 + 今日高开>5% | 70%+ |
| **反包板** | 昨日阴线 + 今日高开 + 放量 | 68%+ |
| **突破板** | 突破平台 + 成交量 2 倍 + | 65%+ |
| **尾盘偷袭板** ❌ | 涨停时间>14:30 | 45% 禁止 |
| **缩量秒板** ❌ | 换手<5% + 秒板 | 40% 禁止 |

## 次日更新（次日 15:30）

### 更新昨日涨停股次日溢价

**流程**:
1. 获取昨日涨停股今日涨跌幅
2. 计算次日溢价 = (今日收盘价 - 昨日涨停价) / 昨日涨停价
3. 更新到 `memory/涨停形态库/YYYY-MM-DD.md`
4. 更新 `memory/涨停形态迭代统计.md` 的胜率
