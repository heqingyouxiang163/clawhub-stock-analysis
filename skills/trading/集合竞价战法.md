# 集合竞价战法 v1.0

## 功能描述
9:15-9:25 集合竞价期间快速选股，识别强势股和主线板块。

## 集合竞价时间轴

| 时间 | 阶段 | 关注点 |
|------|------|--------|
| 9:15-9:20 | 可撤单阶段 | 观察主力意图，虚假申报多 |
| 9:20-9:25 | 不可撤单阶段 | 真实申报，重点关注 |
| 9:25 | 竞价结束 | 确定开盘价，快速决策 |
| 9:30 | 连续竞价 | 执行交易 |

## 竞价强弱判断标准

### 🟢 强势竞价（可打）

| 指标 | 阈值 | 说明 |
|------|------|------|
| 高开幅度 | 3-5% | 最佳区间 |
| 竞价量比 | >3 | 放量明显 |
| 竞价金额 | >1000 万 | 主力进场 |
| 竞价换手 | >2% | 充分换手 |
| 板块助攻 | ≥3 家涨停 | 板块效应 |

**操作**：重点关注，准备打板

---

### 🟡 中性竞价（观察）

| 指标 | 阈值 | 说明 |
|------|------|------|
| 高开幅度 | 1-3% | 偏弱 |
| 竞价量比 | 1.5-3 | 量能一般 |
| 竞价金额 | 500-1000 万 | 主力观望 |
| 竞价换手 | 1-2% | 换手不足 |
| 板块助攻 | 1-2 家涨停 | 板块弱 |

**操作**：观察开盘后走势

---

### 🔴 弱势竞价（放弃）

| 指标 | 阈值 | 说明 |
|------|------|------|
| 高开幅度 | <1% 或低开 | 弱势 |
| 竞价量比 | <1.5 | 缩量 |
| 竞价金额 | <500 万 | 无主力 |
| 竞价换手 | <1% | 换手不足 |
| 板块助攻 | 0 家涨停 | 无板块 |

**操作**：放弃，不关注

## 竞价选股公式

### 核心公式

```
【竞价强度评分】
= 高开幅度 (30 分) + 竞价量比 (25 分) + 竞价金额 (20 分) + 竞价换手 (15 分) + 板块助攻 (10 分)

【评分标准】
- 80-100 分：🟢 强势，可打
- 60-79 分：🟡 中性，观察
- <60 分：🔴 弱势，放弃
```

## 9:15-9:25 操作流程

### 9:15-9:20（观察阶段）

```
【任务】
1. 查看涨停股竞价情况
2. 观察主线板块竞价
3. 记录候选股（5-10 只）

【注意】
- 此阶段可撤单，虚假申报多
- 不急于决策，继续观察
- 记录大单动向
```

### 9:20-9:25（决策阶段）

```
【任务】
1. 确认不可撤单后真实竞价
2. 计算竞价强度评分
3. 筛选前 3 名候选股
4. 准备打板指令

【决策标准】
- 评分≥80 分：准备打板
- 评分 60-79 分：观察开盘
- 评分<60 分：放弃
```

### 9:25-9:30（准备阶段）

```
【任务】
1. 确认最终候选股（1-2 只）
2. 设置涨停价预警
3. 准备仓位（3 成上限）
4. 等待开盘

【准备清单】
□ 候选股 1-2 只
□ 涨停价确认
□ 仓位设置
□ 止损位设置
```

### 9:30-10:00（执行阶段）

```
【任务】
1. 观察开盘走势
2. 快速封板→打板
3. 开盘下杀→放弃
4. 震荡→观察后决策

【执行标准】
- 开盘后 5 分钟内封板：✅ 打板
- 开盘下杀>2%：❌ 放弃
- 震荡后上攻：⚠️ 观察封板
```

## 竞价陷阱识别

### ❌ 虚假竞价（避免）

| 陷阱 | 特征 | 识别方法 |
|------|------|----------|
| 大幅高开低走 | 9:20 高开>5%，9:25 降至<2% | 观察 9:20-9:25 变化 |
| 竞价无量 | 高开但竞价量比<1.5 | 检查量比 |
| 板块无助攻 | 个股强势但板块弱 | 查看板块涨停家数 |
| 昨日烂板 | 昨日炸板/烂板，今日高开 | 查看昨日走势 |

### ✅ 真实竞价（可参与）

| 特征 | 说明 |
|------|------|
| 9:20-9:25 稳定 | 不可撤单后价格稳定 |
| 竞价放量 | 量比>3，金额>1000 万 |
| 板块助攻 | 板块≥3 家涨停 |
| 昨日强势 | 昨日涨停/连板 |
