# 均线系统分析 v1.0

## 功能描述
利用 5/10/20/60 日均线系统分析趋势和买卖点。

---

## 均线参数

| 均线 | 周期 | 用途 |
|------|------|------|
| **MA5** | 5 日 | 超短线，攻击线 |
| **MA10** | 10 日 | 短线，操盘线 |
| **MA20** | 20 日 | 中线，生命线 |
| **MA60** | 60 日 | 长线，决策线 |

---

## 多头排列

### 形态特征
```
MA5 > MA10 > MA20 > MA60
所有均线向上发散
```

### 操作策略
| 位置 | 操作 | 说明 |
|------|------|------|
| **MA5 附近** | 买入 | 超短线买点 |
| **MA10 附近** | 买入 | 短线买点 |
| **MA20 附近** | 买入 | 中线买点 |
| **远离均线** | 持有 | 强势延续 |

### 胜率
| 情况 | 胜率 |
|------|------|
| **刚形成多头** | 80% |
| **多头中期** | 70% |
| **多头末期** | 50% |

## 实战案例

### 成功案例
```
【案例】多头排列 (2026-03-15)
MA5>MA10>MA20>MA60 向上发散
回踩 MA10 买入
上涨 15% 止盈
胜率：80%
```

### 失败案例
```
【案例】假金叉 (2026-03-10)
MA5 上穿 MA10
次日死叉
止损 -3%

【教训】金叉需配合成交量确认
```
