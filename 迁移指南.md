# 🦞 炒股龙虾系统 - 快速迁移指南

## 🎯 一句话总结

**在新电脑上执行以下命令，5 分钟内完成系统复刻：**

```bash
curl -o migrate.sh https://raw.githubusercontent.com/heqingyouxiang163/clawhub-stock-analysis/main/migrate.sh && chmod +x migrate.sh && ./migrate.sh
```

---

## 📋 完整步骤

### 步骤 1：检查环境要求

**必需**:
- ✅ Python 3.9+
- ✅ Git 2.0+
- ✅ 网络连接

**检查命令**:
```bash
python3 --version  # 应该显示 Python 3.9+
git --version      # 应该显示 git version 2.0+
```

### 步骤 2：运行迁移脚本

```bash
# 下载迁移脚本
curl -o migrate.sh https://raw.githubusercontent.com/heqingyouxiang163/clawhub-stock-analysis/main/migrate.sh

# 设置执行权限
chmod +x migrate.sh

# 运行迁移
./migrate.sh
```

### 步骤 3：验证安装

```bash
cd clawhub-stock-analysis

# 测试数据获取
python3 skills/tencent-stock-rank/tencent_stock_rank.py

# 测试推荐系统
python3 tools/高确定性推荐 - 定时任务.py
```

### 步骤 4：配置持仓（可选）

编辑 `tools/持仓配置.py`，填入你的持仓数据。

## 🔧 故障排除

### 问题 1: curl 命令失败

**解决**:
```bash
# 使用 wget 替代
wget https://raw.githubusercontent.com/heqingyouxiang163/clawhub-stock-analysis/main/migrate.sh
chmod +x migrate.sh
./migrate.sh
```

### 问题 2: Git 克隆失败

**解决**:
```bash
# 检查网络
ping github.com

# 使用代理
git config --global http.proxy http://your-proxy:port

# 或使用 SSH
git clone git@github.com:heqingyouxiang163/clawhub-stock-analysis.git
```

### 问题 3: Python 依赖安装失败

**解决**:
```bash
# 升级 pip
pip3 install --upgrade pip

# 逐个安装
pip3 install requests
pip3 install pandas
pip3 install numpy
```
