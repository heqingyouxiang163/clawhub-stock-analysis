## 📊 数据源

### 最优方案：腾讯财经 API

**接口**: `http://qt.gtimg.cn/q=sh600000,sz000001`

**优势**:
- ✅ 速度最快：72ms 响应
- ✅ 数据准确：实时行情，延迟<1 秒
- ✅ 批量支持：单次最多 150 只股票
- ✅ 字段丰富：开盘/最高/最低/当前/涨跌幅/成交量/成交额

**配置**:
```python
BATCH_SIZE = 200  # 每批 200 只
MAX_WORKERS = 10  # 10 线程并发
CACHE_TTL = 120  # 2 分钟缓存
```

**沪深主板代码范围**:
```
沪市：600000-600999, 601000-601999, 603000-603999, 605000-605399
深市：000000-000999, 001000-001999, 002000-002999, 003000-003399
总计：约 3467 只
```
