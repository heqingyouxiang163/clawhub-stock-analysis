# 🧠 强化学习系统使用说明

## 📋 功能概述

系统现已支持**实时强化学习**模式：

- ✅ 你回复"**对**"、"**不错**"、"**准**" → 强化当前判断逻辑
- ✅ 你回复"**错**"、"**不准**"、"**不行**" → 修正逻辑，记录教训
- ✅ 所有历史记录保存在 `memory/强化学习/` 目录
- ✅ 长期积累，持续优化

---

## 🎯 使用方法

### 场景 1：股票推荐后

**系统**：推荐 通鼎互联 (002491)，目标 +10%

**你回复**：
- ✅ "不错" → 系统强化这个推荐逻辑
- ❌ "不准" → 系统修正逻辑，记录教训

### 场景 2：走势预测后

**系统**：预测 国新能源 会反弹

**你回复**：
- ✅ "对" → 强化预测模型
- ❌ "错" → 修正预测逻辑

### 场景 3：操作建议后

**系统**：建议止损 -5%

**你回复**：
- ✅ "正确" → 强化止损策略
- ❌ "不行" → 修正止损逻辑

## 🤖 自动化

### 每日 23:00 自动生成学习报告

系统会在每日 23:00 自动：
1. 统计当日反馈
2. 生成学习报告
3. 保存到 `memory/强化学习/日期 - 学习报告.md`

### 监听器集成

可以将 `自动学习监听器.py` 集成到对话系统中：

```python
from tools.自动学习监听器 import FeedbackMonitor

monitor = FeedbackMonitor()

# 每次用户回复时调用
monitor.analyze_message(
    message="不错",
    context="推荐 通鼎互联 涨停"
)
```

## 🔧 技术细节

### 强化机制

1. **正面反馈**：
   - 增加该策略权重
   - 记录成功案例
   - 类似场景优先使用

2. **负面反馈**：
   - 降低该策略权重
   - 记录教训
   - 类似场景避免使用

3. **长期记忆**：
   - 所有反馈永久保存
   - 每日生成报告
   - 持续优化模型

---

## ⚠️ 注意事项

1. **数据隐私**：所有反馈仅保存在本地
2. **学习周期**：需要一定时间积累（建议至少 1 周）
3. **准确率波动**：初期准确率可能不稳定，属正常现象
